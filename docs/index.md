## 栏目

- **[每日存档](all_daily.md)** — 每天的 arXiv 速览，按 LLM 给出的相关性打分排序。
- **[精读存档](all_deep_reads.md)** — 当日 / 当期高相关性论文的深入解读，含问题动机、核心方法、关键假设与结果。
- **[期刊存档](all_journals.md)** — 季度抓取核心统计 / 计量 / ML 期刊的新发文章，按期刊分组。
