- **摘要**: 在天文与物理数据分析中，每个观测点常附带二元修正（如是否属于某子群或是否被污染），显式处理会引入 N 个额外参数导致 $2^N$ 种配置的指数爆炸。本文证明在一般条件下，对这类二元变量的精确边际化可转化为统计物理中 Ising 模型的数学形式，从而利用 Ising 模型的成熟工具实现快速且精确的似然计算。作者提出了计算代价极小的近似方案，并在 Ia 型超新星定标应用中验证其有效性，发现宿主星系质量分类的不确定性对推断的哈勃常数影响可忽略。对您而言，这篇论文作为天文统计 gateway reading 极佳：它清晰展示了如何用物理模型（Ising）解决统计计算中的组合爆炸问题，数据结构（二元潜变量+似然边际化）与模型假设阐述对 outsider 友好，且其潜变量边际化思路可迁移至因果推断中的混合/污染建模。
- **关键技术**: `Ising model marginalization`, `binary latent variable exact integration`, `exponential configuration reduction`, `likelihood approximation scheme`, `Type Ia supernova calibration`
- **为什么对您有用**: 作为 astrostats 的 gateway reading，对 outsider 极友好：清晰展示二元潜变量边际化的组合爆炸问题及 Ising 模型解法，数据与模型假设交代清楚；其潜变量精确/近似边际化思路可迁移至因果推断中 negative control / 混合群的处理。
