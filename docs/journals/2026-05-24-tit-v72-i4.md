- **摘要**: 本文研究量子信息中确定性端口 teleportation (dPBT) 与酉估计 的一一对应关系。设定为：用 N=n+1 个端口 teleport d 维态的最优保真度，等价于用 n 次调用估计 d 维酉操作的最优保真度。核心机制是利用 Schur-Weyl duality 与对称群表示论，在给定 dPBT 协议时显式构造出达到相同保真度的酉估计协议，反之亦然。利用此对应，作者将 dPBT 的渐近最优保真度界收紧为 1-O(d^4)N^{-2} ≤ F ≤ 1-Ω(d^4)N^{-2}（此前为 d^5 与 d^2 的 gap），并给出 n≤d-1 时酉估计最优保真度的精确公式 F=(n+1)/d^2，证明其与允许 indefinite causal order 的酉反转保真度相等。对您而言，本文的 Schur-Weyl duality 与对称群表示论技术在高维随机矩阵理论中有重要类比，收紧的渐近界展示了类似 minimax rate 的精细分析思路。
- **关键技术**: `Schur-Weyl duality`, `symmetric group representation theory`, `port-based teleportation`, `unitary estimation`, `asymptotic fidelity bounds`
- **为什么对您有用**: 虽然属量子信息领域，但 Schur-Weyl duality 与对称群表示论是高维随机矩阵理论 (RMT) 的核心工具，收紧渐近保真度界的方法论（从粗 gap 到精确 Θ(d^4 N^{-2})）对您在 RMT 中做 minimax rate 与精细界分析有类比借鉴价值。
