### 8. [10.1214/24-aop1696](https://doi.org/10.1214/24-aop1696) — On perfectly friendly bisections of random graphs

- **摘要**: 本文研究 Erdős–Rényi 随机图 G(n,1/2) 的友好二分问题，目标是找到顶点等分使得每个顶点在自身部分的邻居数多于另一部分。作者证明了存在临界常数 γ_crit≈0.17566：以高概率存在等分使每个顶点自身部分邻居数多出 (γ_crit−ε)n，且以高概率不存在多出 (γ_crit+ε)n 的等分。证明核心结合了布尔函数等周不等式、顶点传递集、switching 技巧、给定度序列图枚举及二阶矩方法。该结果实质性地解决了 Füredi (1988) 的猜想，确立了完全友好二分的存在性。虽然本文属概率组合学，但其对随机图极值结构的精确刻画与二阶矩方法，对您在 high-dimensional statistics / random matrix theory 中处理高维随机结构极值与概率界有方法论借鉴价值。
- **关键技术**: `second moment method`, `isoperimetric inequality`, `graph enumeration`, `switching technique`, `random graph theory`
- **为什么对您有用**: 本文属于概率组合学，与您 primary interest 中的 high-dimensional statistics / random matrix theory 相邻，其使用的二阶矩方法与等周不等式对处理高维随机结构极值问题有方法论借鉴意义。
