- **摘要**: 本文研究 batched network coding 中自适应重编码（adaptive recoding）的分布式优化问题，目标是在各批次随机丢包波动下确定下一节点最优重编码包数，关键 estimand 为下一节点批次秩分布的期望值（expected rank）。作者提出任意有限域下统一的自适应重编码框架，并证明当丢包模式为平稳随机过程（涵盖独立丢包与 Gilbert-Elliott 模型）时，expected rank 函数具有凹性。基于该凹性，理论证明存在一类最优解既能最小化重编码包数的随机性，又能容忍因测量误差或机器精度导致的秩分布误差，同时给出求解算法及将任意可行解调优至最优的 tuning 方案。对您可能有用：该工作展示了随机过程假设下优化目标凹性的严格证明与容忍数值误差的鲁棒优化设计，其算法与数值调优思路对统计计算中分布式/鲁棒数值方法设计有参考价值。
