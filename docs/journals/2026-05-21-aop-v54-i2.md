- **关键技术**: `nonasymptotic matrix concentration`, `Tracy-Widom fluctuations`, `spectral norm`, `superconcentration`, `extremum problem for variance patterns`, `Wishart matrices`
- **为什么对您有用**: 直接推进了您 primary interest 中的高维统计与随机矩阵理论（RMT），给出了非渐近谱范数界在 Tracy-Widom 尺度上的最优结果，对高维假设检验中极值统计量的尾部概率控制有直接理论价值。

### 2. [10.1214/25-aop1784](https://doi.org/10.1214/25-aop1784) — Kronecker-product random matrices and a matrix least squares problem

- **摘要**: 本文研究 Kronecker 积随机矩阵模型 $A \otimes I + I \otimes B + \Theta \otimes \Xi$ 的谱分布与预解式，其中 $A,B$ 为独立 Wigner 矩阵，$\Theta,\Xi$ 为确定性对角阵。作者证明了其 Stieltjes 变换可被自由算子的 Stieltjes 变换定量逼近，并给出了预解式的对角确定性等价逼近。进一步得到了 $n \times n$ 预解式分块在算子范数下的精确估计，揭示了非对角预解式元依据 Kronecker 结构位置分别落在 $n^{-1/2}$ 和 $n^{-1}$ 两个不同尺度上。此研究动机源于带线性约束的矩阵最小二乘问题，在 Wigner 输入下，上述分析给出了极小化子及其目标函数极小值的渐近刻画。对您有用：直接推进了高维统计中随机矩阵理论在 Kronecker 积结构与矩阵优化问题上的精确渐近分析，为相关矩阵计算与张量模型的谱推断提供了新工具。
- **关键技术**: `Kronecker-product random matrix`, `Stieltjes transform`, `free operator approximation`, `deterministic equivalent`, `Wigner matrix`, `matrix least squares`
- **为什么对您有用**: 直接契合您在“高维统计（随机矩阵理论）”与“统计计算（矩阵/张量）”上的核心兴趣，提供了 Kronecker 积结构下预解式与谱分布的精确渐近结果，可迁移至高维矩阵优化与张量主成分分析的推断问题。
