- **摘要**: 本文研究可识别加性误差结构方程模型（SEM）下因果序（causal ordering）的不确定性量化问题，目标是构建数据未能拒绝的因果序置信集。核心方法是将因果序的拟合优度检验通过残差 bootstrap 反转，形成置信集；具体地，对每个候选排序拟合加性模型并计算残差，通过 bootstrap 构造检验统计量的参考分布，再按 inversion 原则保留未被拒绝的排序。理论上证明了该置信集的渐近有效性（覆盖率渐近达到名义水平），并展示如何从排序置信集导出祖先关系的子集/超集以及融合模型不确定性的因果效应置信区间。该方法将因果发现从点估计推向有限样本推断，对您在因果推断中关注 identification 与假设检验的交叉研究有直接借鉴价值。
