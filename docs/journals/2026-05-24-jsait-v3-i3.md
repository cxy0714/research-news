### 2. [10.1109/jsait.2022.3219807](https://doi.org/10.1109/jsait.2022.3219807) — Sparsity-Free Compressed Sensing With Applications to Generative Priors
- **作者**: Alireza Naderi, Yaniv Plan
- **期刊/来源**: IEEE Journal on Selected Areas in Information Theory
- **分类**: vol 3 · issue 3 · pp 493-501
- 相关性 3/10 · novelty: `new_theory`
- **摘要**: 本文研究从低维噪声线性测量 y=BAx+e 中重建高维信号 x 的问题，设定中 B 为任意矩阵，A 为 sub-gaussian 随机矩阵，允许测量矩阵具有重尾、行列依赖及奇异值大动态范围。信号结构由非凸锥 T 或凸函数 f 的最小化诱导，从而摆脱了传统压缩感知的稀疏性假设。作者证明，在存在模型失配的情况下，近似经验风险最小化（ERM）仍能鲁棒恢复信号，其条件是有效测量数超过结构集的有效维度。核心发现是矩阵 B 的 stable rank 决定了有效测量数，理论证明依赖于 Jeong et al. (2020) 的随机矩阵理论（RMT）结果。将此框架应用于 ReLU 生成神经网络（GNN）先验时，得到了基于 stable rank 的精确重建保证；若 GNN 最后一层为随机权重，理论进一步允许部分 Fourier 测量矩阵，迈出了 GNN 压缩感知 MRI 理论分析的第一步。对您而言，本文将 RMT 工具（stable rank、sub-gaussian 矩阵奇异值性质）与高维重建的 minimax 率直接挂钩，是 RMT 在高维 inference 与非稀疏结构重建中应用的绝佳切入点。
- **关键技术**: `random matrix theory`, `stable rank`, `empirical risk minimization`, `compressed sensing`, `generative neural network priors`, `sub-gaussian matrix`
- **为什么对您有用**: 本文核心在于利用随机矩阵理论（stable rank）刻画高维非稀疏信号重建的充分条件，直接连接到您的高维统计与 RMT 兴趣；同时，对任意测量矩阵 B 的容忍度及模型失配的鲁棒性分析，为高维 inference 中放松经典独立同分布假设提供了可迁移的理论框架。
