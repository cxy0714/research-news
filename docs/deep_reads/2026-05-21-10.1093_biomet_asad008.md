# Ancestor regression in linear structural equation models

**作者**: C Schultheiss, P Bühlmann  
**来源**: Biometrika  
**主题**: 因果推断  
**相关性**: 9/10  
**链接**: <https://doi.org/10.1093/biomet/asad008>

---

### 一、核心问题与贡献
①研究了线性结构方程模型（linear SEM）中如何区分给定变量的祖先与非祖先并推断因果序的问题。②核心工具是基于线性回归系数统计检验的“祖先回归”方法。③主要贡献是在高斯误差设定下（传统方法因不可识别性而失效）仍提供渐近严格的 Type I error 控制，并给出了评估数据是否来自线性 SEM 的渐近有效拟合优度 p 值。

### 二、基础设定
- **核心概念与符号**：
  - $X_j = \sum_{k \in \text{PA}_j} \beta_{jk} X_k + \epsilon_j$：线性 SEM 设定，$\text{PA}_j$ 为直接父节点集，$\epsilon_j$ 为相互独立的误差项。
  - $\text{AN}_j$：变量 $X_j$ 的祖先集（包含 $X_j$ 自身）。
  - 祖先回归：将 $X_j$ 对 $X_k$ 及 $X_k$ 的祖先集 $\text{AN}_k$ 做线性回归，通过检验 $X_k$ 的回归系数是否为 0 来判断 $k$ 是否为 $j$ 的祖先。
- **关键假设**：
  - 线性与加性噪声：限制了模型的函数形式。
  - 误差项相互独立：SEM 的标准 Markov 假设。
  - 高斯分布（可选但最关键）：传统基于非高斯性（如 ICA）的方法在此假设下失效，因为模型不可识别。本文方法在此假设下依然保持 Type I error 控制，是相较于已有文献的核心突破。
  - 因果充分性：无隐藏混杂因子。与 PC 算法等经典方法假设一致，未做放宽。
- **问题背景**：
  - 传统因果发现方法（如 PC 算法、GES）基于条件独立性（CI）检验，在高斯设定下只能得到 Markov 等价类（CPDAG），无法判定因果方向。
  - 基于非高斯性的方法（如 Shimizu et al., 2006 的 LiNGAM）利用独立成分分析识别模型，但在高斯误差下完全失效。
  - 本文与最相关文献的区别：不依赖非高斯性提取方向信息，而是将因果图中的祖先关系转化为特定回归模型下的系数约束检验，以牺牲一定检验功效为代价，换取了高斯设定下的 Type I error 保证。

### 三、主要定理 / 核心结果
1. **原文陈述**：在 $H_0: k \notin \text{AN}_j$ 下，祖先回归中 $X_k$ 的回归系数渐近为 0；基于此构造的 t 检验在渐近下具有严格的 Type I error 控制，即使误差项服从高斯分布。
2. **直观解释**：如果 $k$ 不是 $j$ 的祖先，在总体水平上，将 $X_j$ 对 $X_k$ 及 $\text{AN}_k$ 回归时，$X_k$ 的偏回归系数必为 0；若 $k$ 是 $j$ 的祖先，该系数非零。这把图模型中的祖先关系映射为了回归系数的零与非零。
3. **解决了什么技术难点**：克服了高斯线性 SEM 因 Markov 等价导致的不可识别性。传统方法在此设定下无法区分 $X \to Y$ 与 $Y \to X$，本文通过引入特定的条件集（$\text{AN}_k$），在零假设下阻断了所有非因果的依赖路径，使得检验统计量在零假设下具有良定义的极限分布。
4. **适用条件与局限**：必须已知或能一致估计 $\text{AN}_k$（或其超集）。功效低于基于非高斯性的方法，因为高斯设定下缺乏非对称性信息，只能保守地检验祖先关系而非直接因果路径。拟合优度检验依赖于高维残差独立性的检验构造。

### 四、证明框架 / 方法设计
- **证明主干逻辑**：总体图模型性质推导 $\to$ 有限样本 OLS 估计量渐近分布 $\to$ 零假设下的检验水平控制。
- **拆解为 3-5 个关键逻辑步骤**：
  1. **总体水平图-回归等价性**：证明在总体水平上，$k \notin \text{AN}_j$ 当且仅当 $X_j$ 对 $X_k$ 控制 $\text{AN}_k$ 后的偏回归系数为 0。
  2. **条件集的估计与替代**：由于真实 $\text{AN}_k$ 未知，证明使用 $\text{AN}_k$ 的一致估计量（或特定的超集）作为条件集进行回归，不改变零假设下系数为 0 的性质。
  3. **渐近分布推导**：在零假设下，推导 OLS 估计量的渐近正态性，证明条件集的估计误差在 $H_0$ 下不影响检验统计量的极限分布。
  4. **拟合优度检验构造**：基于残差乘积的交叉矩条件构造检验统计量，推导其在模型设定正确时的极限分布，生成渐近有效的 p 值。
- **最关键的技巧性引理或"跳跃点"**：将因果发现中不可识别的方向推断问题，重构为在特定条件集下的回归系数检验问题。核心在于选择 $\text{AN}_k$ 作为条件集：它阻断了 $X_k$ 到 $X_j$ 的所有非因果关联（因为任何混杂因子的路径必经过 $\text{AN}_k$），从而在 $H_0$ 下“净化”了 $X_k$ 对 $X_j$ 的效应，使得 OLS 系数在 $H_0$ 下精确为 0。
- **数学工具评价**：经典 OLS 渐近理论与图模型拓扑性质的巧妙组合。未引入高深的新型概率工具，但在问题转化（因果图到回归方程的映射）上展现了极高的分析技巧。

### 五、与研究者兴趣的关联
- **连接到哪个子方向**：因果推断（因果发现/图模型下的假设检验）与数理统计（有限样本/渐近假设检验）的交叉。
- **可借鉴的核心思路或技术工具**：将因果结构的识别问题转化为带有严格 Type I error 控制的参数检验问题，这一思路可迁移至其他因果推断场景（如：检验某变量是否为工具变量、检验特定路径的中介效应是否为零）。构造特定条件集以“净化”偏回归系数的技术，在半参数推断中构造 Neyman 正交矩条件时具有启发意义。
- **值得精读的关键参考文献**：
  1. Shimizu et al. (2006) "A linear non-Gaussian acyclic model for causal discovery" (LiNGAM)：理解非高斯性如何打破 Markov 等价类，对比本文为何在高斯下牺牲功效换取 Type I error。
  2. Shah & Peters (2020) "The hardness of conditional independence testing and implications for causal discovery"：理解 CI 检验的困难性，体会本文绕过 CI 检验直接做回归系数检验的动机。

### 六、延伸思考与练习
- **假设扰动**：若放宽线性假设，假设 $X_j = f_j(\text{PA}_j) + \epsilon_j$（非线性加性噪声模型），祖先回归的 OLS 系数在 $H_0$ 下是否仍为 0？若否，需要引入何种非参数/半参数回归工具（如核机器或神经网络）来重构检验统计量，且如何保证 Type I error？
- **开放问题**：如何将祖先回归推广到高维设定（$p \gg n$）？在 $\text{AN}_k$ 的选择上能否结合 Debiased Lasso 或 Double Machine Learning 构造具有有效后选择推断的检验统计量？
- **理解检测题**：考虑一个简单的高斯线性 SEM：$X_1 \to X_2 \to X_3$。写出 $X_3$ 对 $X_1$ 的简单回归系数，以及 $X_3$ 对 $X_1$ 控制 $\text{AN}_1$（即仅 $\{X_1\}$）后的偏回归系数。解释为何简单回归无法用于检验 $X_1 \in \text{AN}_{X_3}$，而祖先回归可以。
