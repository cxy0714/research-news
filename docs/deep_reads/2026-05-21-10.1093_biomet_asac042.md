# Sample-constrained partial identification with application to selection bias

**作者**: Matthew J Tudball, Rachael A Hughes, Kate Tilling, Jack Bowden, Qingyuan Zhao  
**来源**: Biometrika  
**主题**: 因果推断  
**相关性**: 9/10  
**链接**: <https://doi.org/10.1093/biomet/asac042>

---

### 一、核心问题与贡献
①研究了目标函数与约束集均需由经验数据估计的一般性偏识别问题的统计推断。②核心方法是通过对估计的约束集进行适当的统计松弛，推导出最优值的渐近有效置信区间。③主要贡献是将该一般性理论应用于队列研究的选择偏倚敏感性分析，证明引入总体辅助信息可显著缩窄偏识别区间并克服现有方法的保守性。

### 二、基础设定
- **核心概念与符号**：
  - $\theta^* = \sup_{\gamma \in \Gamma^*} \psi(\gamma)$：目标参数（偏识别区间的上界），其中 $\psi$ 为目标函数，$\Gamma^*$ 为真实约束集。
  - $\hat{\theta}$：基于经验约束集 $\Gamma_n$ 求得的经验最优值。
  - $C_n$：为修正覆盖概率不足而对 $\Gamma_n$ 进行松弛后的统计置信集。
  - $d_H(\cdot, \cdot)$：Hausdorff 距离，用于度量约束集估计与真实集之间的偏差。
- **关键假设**：
  1. **参数空间紧致性与连续性**：$\Gamma^*$ 紧致，$\psi$ 连续。统计学含义为保证极值存在且连续映射定理可用。
  2. **经验过程收敛性**：$\sqrt{n}(\hat{\psi} - \psi)$ 依分布收敛于某高斯过程。统计学含义为目标函数估计具备 $\sqrt{n}$-速率且满足 Donsker 条件。
  3. **约束集估计收敛速率**：$d_H(\Gamma_n, \Gamma^*) = O_p(n^{-1/2})$。统计学含义为约束集边界估计足够精确，与目标函数同速收敛。
  4. **辅助信息约束**：总体层面已知的矩约束（如总体均值/方差已知）。统计学含义为通过额外信息截断可行域，缩窄识别界。
  - *假设对比*：相比传统偏识别推断（如 Manski 的线性规划框架或仅考虑目标函数估计误差的设定），本文放宽了对凸优化结构的依赖，允许非凸目标/约束，并显式处理了约束集本身的估计误差。
- **问题背景**：
  - 已有方法不足：现有偏识别推断多局限于凸优化问题，或假设约束集已知无误差；针对选择偏倚的敏感性分析（如 tipping point 方法）通常高度保守且难以融入外部先验信息。
  - 与最相关文献区别：相比 Romano & Shaikh (2010) 针对一般约束集的均匀推断，本文提供了更直接针对最优值（极值）的松弛推断框架；相比常规选择偏倚敏感性分析，本文将其重构为带辅助约束的偏识别问题，实现了从启发式保守分析到严格渐近推断的跨越。

### 三、主要定理 / 核心结果
1. **原文陈述**：在满足经验过程收敛与约束集 Hausdorff 距离 $O_p(n^{-1/2})$ 收敛的条件下，构造松弛集 $C_n = \{\gamma: \hat{g}(\gamma) \le \hat{q}_{1-\alpha}\}$，其中 $\hat{q}_{1-\alpha}$ 为约束函数估计的渐近分布分位数，则 $P(\theta^* \le \sup_{\gamma \in C_n} \hat{\psi}(\gamma)) \to 1-\alpha$。
2. **直观解释**：由于经验约束集 $\Gamma_n$ 存在采样误差，真实的极值点可能被错误地排除在 $\Gamma_n$ 之外，导致直接求上确界会系统性地低估 $\theta^*$。通过将约束边界向外“膨胀”一个统计松弛量 $q_{1-\alpha}$，补偿了估计误差，使得膨胀后的集合 $C_n$ 以 $1-\alpha$ 的概率包含真实的极值点，从而保证覆盖概率。
3. **解决了什么技术难点**：解决了非凸设定下目标函数估计误差与约束集估计误差的耦合问题。在非凸情况下，极值对边界扰动极度敏感，无法直接使用 Delta method 或常规的 M-估计理论。
4. **适用条件与局限**：必须满足约束集的 $O_p(n^{-1/2})$ 收敛，若约束集边界由高维非参估计构成导致收敛速率慢于 $\sqrt{n}$，则松弛量 $q_{1-\alpha}$ 将极大，置信区间可能退化为无信息区间；Donsker 条件限制了目标函数的复杂度。

### 四、证明框架 / 方法设计
- **证明主干逻辑**：经验过程理论 + 集合值分析 + 构造法。
- **拆解为 3-5 个关键逻辑步骤**：
  1. 从目标参数定义出发，将覆盖概率 $P(\theta^* \le \sup_{\gamma \in C_n} \hat{\psi}(\gamma))$ 转化为集合包含概率 $P(\Gamma^* \subset C_n)$ 及目标函数偏差的控制。
  2. 利用约束函数的渐近正态性/高斯过程逼近，构造 $C_n$ 的边界阈值 $\hat{q}_{1-\alpha}$，证明 $P(\Gamma^* \subset C_n) \ge 1-\alpha + o(1)$。
  3. 在 $C_n$ 上分析 $\sup \hat{\psi}(\gamma)$ 的渐近行为，利用 $\hat{\psi}$ 的 Donsker 性质与连续映射定理，处理目标函数的估计波动。
  4. 联立集合包含概率与函数波动，得出最终参数的渐近覆盖保证。
- **最关键的技巧性引理或"跳跃点"**：将集合的包含关系 $\Gamma^* \subset C_n$ 等价转化为逐点函数不等式 $\hat{g}(\gamma) \le \hat{q}_{1-\alpha}$ 的成立。这一跳跃将复杂的集合覆盖概率计算降维为对约束函数经验过程上确界分布的求解，从而可以利用经典的极值理论或 Bootstrap 获取分位数。
- **数学工具评价**：是经典半参数经验过程与集合值推断的巧妙组合。并未发明全新概率论工具，而是将 Hausdorff 距离与 Donsker 定理在偏识别极值问题中进行了严密的适配与拆解。

### 五、与研究者兴趣的关联
- **连接到哪个子方向**：Partial identification 的半参数推断 / Selection bias 的 sensitivity analysis。
- **可借鉴的核心思路或技术工具**：将启发式的敏感性分析严格转化为带约束的偏识别优化问题，并利用外部辅助数据（如总体矩约束）作为额外约束收紧识别域。这种“辅助信息约束收紧识别界”的思路可直接迁移到 **Proximal CI** 中的 negative control 约束设定，或 **高维 IV** 的矩约束推断中，将先验知识编码为约束集以缩窄置信区间。
- **值得精读的关键参考文献**：
  1. Romano & Shaikh (2010), *On the uniform convergence of constrained optimization estimators*：本文推断方法的理论基石，处理约束集估计与推断的早期严格框架。
  2. Zhao, Small, & Bhattacharya (2019), *Sensitivity analysis for inverse probability weighting*：针对选择偏倚敏感性分析的现代框架，对比本文如何将其纳入偏识别体系。
  3. Manski (2003), *Partial Identification of Probability Distributions*：偏识别领域的奠基之作，理解纯识别界与统计推断界差异的必读。

### 六、延伸思考与练习
- **假设扰动**：若约束集 $\Gamma^*$ 的边界由高维非参数估计得到（收敛速率为 $O_p(n^{-1/4})$），Hausdorff 距离收敛变慢。此时 $C_n$ 的松弛量 $c_n$ 需如何调整？结论可能从 $\sqrt{n}$-收敛退化为非参数速率，置信区间急剧变宽，技术上需要引入 Bootstrap 或高维极值理论来重新刻画 $\hat{g}(\gamma)$ 的极值分布。
- **开放问题**：如何在约束集本身是高维参数（如 $\gamma$ 维度随 $n$ 增长）时，构造有效的偏识别置信区间？现有的 Donsker 条件在高维下失效，可能需要结合高维统计中的 Sparsity 或 Regularization 技术重构约束集。
- **理解检测题**：假设你有一个二值处理的因果效应 $\tau = E[Y(1)-Y(0)]$，存在不可测混杂 $U$。你假设 $U$ 对结果的影响被限制在 $[-\delta, \delta]$ 内。请将此敏感性参数 $\delta$ 转化为本文偏识别框架中的约束集 $\Gamma^*$，并说明当引入总体约束 $E[Y]=\mu_0$（已知）时，$\Gamma^*$ 会如何被收紧，从而在数学上解释识别区间的缩窄。
