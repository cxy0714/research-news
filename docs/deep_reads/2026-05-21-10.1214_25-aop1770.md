# Large deviations of the largest eigenvalue of supercritical sparse Wigner matrices

**作者**: Fanny Augeri, Anirban Basak  
**来源**: Annals of Probability  
**主题**: 高维统计 / 随机矩阵  
**相关性**: 9/10  
**链接**: <https://doi.org/10.1214/25-aop1770>

---

### 一、核心问题与贡献
①研究了超临界稀疏 Wigner 矩阵（平均度至少为维度的对数级）最大特征值的大偏差性质。②核心工具是有限秩扰动分解与组合概率方法，将大偏差事件归约为局部子图（高权重顶点或团）的出现。③主要贡献是证明了该设定下大偏差率函数在典型值处不连续，揭示了有限秩扰动驱动大偏差的机制，补全了从常数平均度到对数平均度的理论空白。

### 二、基础设定
- **核心概念与符号**：
  - $n$：矩阵维度。
  - Sparse Wigner Matrix $M_n$：对角及上方元素 $M_{ij} = b_{ij} x_{ij}$，其中 $b_{ij} \sim \text{Bern}(p_n)$ 控制稀疏度，$x_{ij}$ 为次高斯随机变量控制权重。
  - $p_n$：边存在概率，超临界设定要求 $np_n \ge c \log n$。
  - $\lambda_1(M_n)$：矩阵最大特征值。
  - Semicircle law：稠密 Wigner 矩阵经验谱分布的极限。
  - Rate function $I(x)$：大偏差原理中的率函数，衡量 $\lambda_1$ 偏离典型值的指数衰减速度。
- **关键假设**：
  - $np_n \ge c \log n$（超临界稀疏性）：保证最大特征值"粘附"在半圆律支撑边缘，且图具有足够连通性，排除了极度稀疏下由孤立点主导的纯局部特征值行为。相比 Ganguly et al. 的 $np_n = O(1)$ 假设，此假设将稀疏度提升至半圆律依然适用的阈值之上。
  - $x_{ij}$ 次高斯：控制边权重的轻尾性质，排除重尾导致的单元素极值干扰。相比稠密RMT常使用的高斯假设，次高斯在稀疏网络中更具现实意义。
- **问题背景**：稠密 Wigner 矩阵的大偏差已有经典结论（率函数连续），极度稀疏（常数度）下大偏差由孤立星图决定。对数稀疏度区间（超临界但非稠密）的机制此前不明确，特别是局部结构如何通过有限秩扰动影响全局最大特征值。

### 三、主要定理 / 核心结果
1. **原文陈述**：设 $M_n$ 为超临界稀疏 Wigner 矩阵，$np_n \ge c \log n$。对于上尾事件 $\lambda_1(M_n) \ge x$（$x$ 大于半圆律支撑上界），其大偏差由两种机制之一主导：1) 存在一个顶点度数极大且其关联边权重之和极大；2) 存在一个团其边权重极大。所得率函数 $I(x)$ 在 $\lambda_1$ 的典型值处不连续。
2. **直观解释**：矩阵最大特征值的大幅偏离并非由所有元素的微小协同波动引起（稠密矩阵情况），而是由极少数行/列（有限秩）的剧烈异常引起。具体而言，要么某个节点连了特别多且权重大的边（秩1扰动），要么一小撮节点形成全连接且权重极大（秩 $k$ 扰动）。率函数不连续意味着：一旦偏离典型值，概率衰减的"速度"立即发生跳跃，不存在平滑过渡。
3. **解决了什么技术难点**：在 $p_n \to 0$ 的稀疏设定下，传统的矩量法或联合集中不等式失效，因为矩阵元素高度非均匀（大量为0）。需要精确计算局部子图出现的概率，并证明其他背景矩阵的谱半径不足以掩盖这些局部扰动。
4. **适用条件与局限**：必须满足 $np_n \ge c \log n$，若 $p_n$ 更小则进入常数度区间，机制完全不同。次高斯假设是必要的，若为重尾则大偏差机制变为单个元素极大。

### 四、证明框架 / 方法设计
- **证明主干逻辑**：构造法 + 有限秩扰动分解 + 极值理论。
- **拆解为 3-5 个关键逻辑步骤**：
  1. **矩阵分解与扰动隔离**：将矩阵 $M_n$ 分解为包含异常局部结构（高权重顶点/团）的有限秩矩阵 $A$ 与剩余背景矩阵 $B$，利用 Weyl 不等式将 $\lambda_1(M_n)$ 的下界归结为 $\lambda_1(A)$ 的下界与 $B$ 的谱范数控制。
  2. **背景矩阵谱范数控制**：证明在 $np_n \ge c \log n$ 下，背景矩阵 $B$ 的最大特征值依概率收敛于半圆律边界，不会干扰 $A$ 产生的大特征值。
  3. **局部结构概率计算**：构造特定的高权重顶点或团，计算其出现的概率，得到大偏差概率的下界。
  4. **大偏差上界匹配**：证明任何导致 $\lambda_1 \ge x$ 的事件，必然蕴含存在上述某种局部结构，利用组合计数与次高斯尾部条件将概率上界与下界匹配，得到精确的率函数。
- **最关键的技巧性引理或"跳跃点"**：证明"除了高权重顶点和团，没有其他结构能以更大概率产生大特征值"。这需要精细的图论与线性代数结合：将特征向量与矩阵乘积转化为图上的随机游走/路径计数，并证明在 $p_n \ge c \log n / n$ 时，长路径的贡献被指数压制，只有短路径（对应局部结构）起决定作用。
- **数学工具评价**：是随机矩阵谱理论、图极值组合与概率大偏差理论的精妙结合。特别是将谱问题转化为图结构存在性的对偶分析，是稀疏矩阵特有的非渐近方法。

### 五、与研究者兴趣的关联
- **连接到哪个子方向**：高维假设检验中的极值统计量与 RMT (Random Matrix Theory)。
- **可借鉴的核心思路或技术工具**："有限秩扰动导致率函数不连续"的范式。在基于最大特征值的假设检验中（如检测稀疏网络中的异常社区或高维协方差矩阵的尖刺模型），如果原假设是稠密/半圆律，备择假设是稀疏/局部异常，该结论直接给出了检验统计量 p 值计算的精确渐近公式。技术上的"矩阵分解+背景压制+局部结构主导"思路可迁移到高维因果推断中处理稀疏工具变量或稀疏网络干扰。
- **值得精读的关键参考文献**：
  1. Ganguly & Nam (2022, PTRF): 常数平均度下的大偏差，对比阅读可深刻理解稀疏度相变对谱性质的质变影响。
  2. Bordenave, Caputo, Chafaï, Piras (2024, AOP) 或类似关于稀疏矩阵谱半径的文献：理解 $np_n \approx \log n$ 这一相变临界点的数学机制。

### 六、延伸思考与练习
- **假设扰动**：若将 $x_{ij}$ 的次高斯假设放宽为重尾分布（如幂律分布），大偏差机制将如何变化？技术上需要引入什么新工具？（提示：大偏差将由单个最大矩阵元素决定，而非局部子图，需用正则变化理论替代次高斯尾部积分）。
- **开放问题**：当 $np_n$ 恰好处于 $\log n$ 的临界窗口（如 $np_n = \log n + c$）时，连通性的相变与最大特征值大偏差的相变如何交互？率函数在此临界区的连续性如何？
- **理解检测题**：假设稀疏 Wigner 矩阵的 $p_n = n^{-0.9}$（满足 $np_n \ge c \log n$），若我们要检验网络中是否存在一个由 $k$ 个节点组成的异常稠密子图，基于 $\lambda_1$ 的检验统计量在原假设下的尾部概率衰减速度是否与稠密高斯矩阵（$p_n=1$）相同？请用率函数的连续性/不连续性解释其本质区别及对检验功效的影响。
