# Estimation of treatment effect among treatment responders with a time‐to‐event endpoint

**作者**: Andreas Nordland, Torben Martinussen  
**来源**: Scandinavian Journal of Statistics  
**主题**: 因果推断  
**相关性**: 9/10  
**链接**: <https://doi.org/10.1111/sjos.12706>

---

### 一、核心问题与贡献
①研究了在主分层框架下，当结局为右删失生存数据时，由生物标志物定义的"处理响应者"群体的因果效应估计问题。②核心方法是基于排他性约束（非响应者无处理效应）实现参数识别，并推导出该主分层因果参数的 efficient influence function (EIF)。③据此构建了双重稳健且半参数有效的估计量，克服了生存数据右删失与主分层不可观测带来的估计偏差。

### 二、基础设定
- **核心概念与符号**：
  - $Z \in \{0,1\}$：处理分配（如安慰剂 vs 活性药）。
  - $R = R(1)$：潜在生物标志物响应（仅在处理组可观测），定义主分层。
  - $T(z)$：潜在生存时间，$C$：右删失时间，$X$：基线协变量。
  - 目标参数 $\theta$：主分层 $R(1)=1$ 上的处理效应（如 $E[T(1)-T(0) \mid R(1)=1]$ 或该层上的生存概率差/风险比）。
- **关键假设**：
  1. **Exclusion Restriction (ER) for non-responders**：$T(1) = T(0)$ 当 $R(1)=0$。统计学含义：处理对非响应者无因果效应，这是将主分层效应从不可识别转化为可识别的核心杠杆；相比一般主分层文献放宽了对联合潜在结果的参数化假设，但强化了生物学机制假设。
  2. **Monotonicity**：$R(1) \ge R(0)$（通常在安慰剂对照下假设 $R(0)=0$）。限制了主分层的数量。
  3. **Ignorability / Unconfoundedness**：$(T(z), R(z)) \perp Z \mid X$。保证处理分配的类随机性。
  4. **Independent censoring**：$T \perp C \mid Z, X$（或条件独立）。生存分析的标准前提。
- **问题背景**：传统主分层分析在生存数据下面临床床识别困难，且受限于右删失与不可观测分层，常需强参数化假设。与 Frangakis & Rubin (2002) 的奠基工作相比，本文通过 ER 假设实现了非参数识别；与常规生存分析相比，本文通过 EIF 处理了主分层带来的非规则参数问题与删失干扰。

### 三、主要定理 / 核心结果
1. **原文陈述**：在 ER、单调性与无混杂假设下，目标参数 $\theta$ 可由观测数据分布非参数识别；其 EIF 被显式推导出，形式包含处理机制、删失机制与生存分布的干扰参数泛函；基于 EIF 构造的 one-step 估计量 $\hat{\theta}_n$ 满足 $\sqrt{n}(\hat{\theta}_n - \theta) \leadsto N(0, V)$，且在干扰参数模型正确指定时达到半参数有效界，部分错误指定时仍一致（双重稳健性）。
2. **直观解释**：ER 假设相当于把"非响应者"的处理组与安慰剂组生存分布强行绑定，从而让"响应者"的潜在生存分布可以通过观测分布的减法运算被"抠"出来。EIF 则给出了调整干扰参数的最优方向，双重稳健性意味着即使对生存分布或删失机制的建模出现偏误，只要倾向得分正确，估计量仍收敛至真值。
3. **解决了什么技术难点**：解决了主分层潜在结果不可观测与生存数据右删失双重叠加带来的非标准缺失数据问题，特别是将无限维的累积风险函数纳入影响函数的推导。
4. **适用条件与局限**：ER 假设不可检验且极强（若非响应者存在安慰剂效应或副作用，识别即失效）；条件独立删失假设在存在时变混杂时可能违背；若协变量维度高，非参数干扰参数估计的收敛速率可能破坏 $\sqrt{n}$-渐近性。

### 四、证明框架 / 方法设计
- **证明主干逻辑**：构造法 + 矩量法（基于 EIF 的估计方程）。
- **拆解为关键逻辑步骤**：
  1. **识别性推导**：利用 ER 与单调性，将主分层上的反事实生存概率 $P(T(z) > t \mid R(1)=1)$ 表示为可观测条件概率的泛函（如 $P(T \mid Z=1, R=1)$ 与 $P(T \mid Z=0)$ 的加权组合）。
  2. **切线空间与 EIF 推导**：在无混杂与独立删失下，构造全数据切线空间，通过正交投影计算目标参数的 EIF。此过程需将右删失机制通过逆概率加权（IPCW）或增广项纳入影响函数。
  3. **估计量构造**：将 EIF 中的未知干扰函数（倾向得分 $\pi(X)$、条件生存函数 $S(t \mid X, Z)$、删失分布 $G(t \mid X, Z)$）替换为半参数/非参数估计量，构造 one-step 估计量。
  4. **渐近性质分析**：利用经验过程理论，证明在 Donsker 类条件下估计量的余项 $o_P(n^{-1/2})$，并验证双重稳健性。
- **最关键的技巧性引理或"跳跃点"**：将生存数据的计数过程与 EIF 推导结合的跳跃。具体而言，在推导 EIF 时，需对删失机制 $G$ 求路径导数，此时需利用 Doob-Meyer 分解将失效过程 $N(t)$ 与风险过程 $Y(t)$ 引入影响函数的增广项，这是将标准缺失数据 EIF 推广到右删失生存数据的核心技术。
- **数学工具评价**：是经典半参数理论在特定因果结构（主分层+ER）下的巧妙组合应用，特别是将 Robins 的 IPCW 与增广逆概率加权（AIPCW）框架无缝嵌入主分层的识别逻辑中。

### 五、与研究者兴趣的关联
- **连接到哪个子方向**：Principal stratification 下的 semiparametric efficiency 与 doubly robust estimation，特别是结合 survival data 的因果推断。
- **可借鉴的核心思路或技术工具**：
  1. 如何在强不可检验假设（ER）下，利用识别杠杆推导无限维参数的 EIF 并构建 DR 估计量；
  2. 将右删失机制（通过计数过程与鞅）整合进半参数影响函数的标准技术，可迁移到其他带有删失或截断的因果推断问题中。
- **值得精读的关键参考文献**：
  1. Robins, J. M., Rotnitzky, A., & Scharfstein, D. (1999). Sensitivity analysis for selection bias... (处理删失与缺失数据半参数理论的基石，理解 EIF 推导必读)。
  2. Frangakis, C. E., & Rubin, D. B. (2002). Principal stratification in causal inference. (主分层框架的源头，理解 ER 假设在识别中的根本作用)。

### 六、延伸思考与练习
- **假设扰动**：若放松排他性约束（即允许非响应者有处理效应 $T(1) \neq T(0) \mid R(1)=0$），目标参数将不可识别。此时需引入敏感性分析参数 $\delta(t) = P(T(1) > t \mid R(1)=0) - P(T(0) > t \mid R(1)=0)$。技术上，需在 EIF 中重新分解干扰参数，推导包含 $\delta$ 的偏置影响函数，并构造依赖于 $\delta$ 的 DR 估计量。
- **开放问题**：如何将此框架推广至存在时变混杂与依从性删失的纵向生存数据？是否可通过 g-estimation 或 sequential randomization 推广 EIF？
- **理解检测题**：假设删失机制不仅依赖于基线协变量 $X$，还依赖于随时间变化的协变量 $L(t)$，请定性地描述推导 EIF 时需要做何修改？提示：考虑条件独立删失下的风险集调整与逆概率权重的时变积分形式。
