# Least squares estimation of a quasiconvex regression function

**作者**: Somabha Mukherjee, Rohit K Patra, Andrew L Johnson, Hiroshi Morita  
**来源**: Journal of the Royal Statistical Society Series B  
**主题**: 非参数 / 半参数  
**相关性**: 8/10  
**链接**: <https://doi.org/10.1093/jrsssb/qkad133>

---

### 一、核心问题与贡献
①研究了多变量拟凸（及单调）回归函数在误差项仅存在二阶矩且可依赖协变量时的最小二乘估计问题。②核心工具是通过刻画函数空间的混合整数二次规划（MIQP）实现计算，并利用经验过程与截断技术推导 sharp oracle inequality。③主要贡献是证明了拟凸约束 LSE 的存在性，给出了仅依赖二阶矩的有限样本风险界，并提供了可计算的 MIQP 算法。

### 二、基础设定
- **核心概念与符号**：
  - $f_0$: 真实的回归函数，$f_0(x) = E(Y|X=x)$，满足拟凸性（及单调性）。
  - $\mathcal{F}$: 拟凸（及单调）函数类。
  - $\hat{f}_n$: 拟凸约束最小二乘估计量 (LSE)，$\hat{f}_n = \arg\min_{f \in \mathcal{F}} \sum_{i=1}^n (Y_i - f(X_i))^2$。
  - MIQP: 混合整数二次规划，用于将无限维的函数空间优化转化为有限维离散优化。
- **关键假设**：
  - 拟凸性/单调性假设：$f_0 \in \mathcal{F}$。统计学含义为利用经济公理的形状约束大幅缩减函数空间，规避非参数回归的维数灾难；相比无约束或纯凸约束，更贴合生产/成本函数的经济逻辑。
  - 误差二阶矩假设：$E(\epsilon_i^2|X_i=x) \le \sigma^2 < \infty$。允许误差异方差且仅要求二阶矩存在；相比传统形状约束文献常要求的亚高斯或指数矩假设，此条件大幅放宽。
  - 协变量紧支撑假设：$X_i$ 取值于紧集。这是保证 LSE 存在性的必要条件，若支撑无界，拟凸函数类非闭，LSE 可能不存在。
- **问题背景**：
  - 针对不足：多变量拟凸回归缺乏理论保障与计算方案。现有凸回归计算复杂且不满足经济公理，而已有拟凸估计（如基于下水平集的启发式算法）无法保证 LSE 存在性且缺乏有限样本理论。
  - 与最相关文献区别：相比 Seijo & Lim (2011) 的凸回归 LSE，本文处理的拟凸约束集非凸，存在性证明需全新拓扑构造；相比 Allon et al. (2007) 的拟凸启发式算法，本文严格定义了 LSE 并给出风险界。

### 三、主要定理 / 核心结果

1. **定理：LSE 的存在性**
   - 原文陈述：在紧致定义域上，拟凸（及单调）函数类上的 LSE 几乎必然存在。
   - 直观解释：虽然拟凸函数类在逐点收敛下不闭合（极限可能不再是拟凸函数），但通过构造极小化序列的下水平集，可证明目标函数的下确界在扩大的函数类中可达，且极小值点仍落在原拟凸类中。
   - 技术难点：克服约束集非凸带来的极值不可达问题。
   - 适用条件与局限：紧致定义域是必要条件；若定义域无界，结论可能失效。

2. **定理：Sharp Oracle Inequality / 有限样本风险界**
   - 原文陈述：存在绝对常数 $C>0$，使得 $E\left(\frac{1}{n}\sum_{i=1}^n (\hat{f}_n(X_i) - f_0(X_i))^2\right) \le \inf_{f \in \mathcal{F}} \frac{1}{n}\sum_{i=1}^n (f(X_i) - f_0(X_i))^2 + C \cdot R_n(\mathcal{F})$，其中 $R_n(\mathcal{F})$ 为与函数类复杂度相关的余项，常数可达1（sharp）。
   - 直观解释：LSE 的风险不仅被真实函数控制，还能逼近函数类中的任何误设函数，且余项反映了形状约束带来的复杂度惩罚，常数1意味着估计量没有无谓的方差放大。
   - 技术难点：在仅存在二阶矩且异方差条件下，控制经验过程的包络函数和截断误差。
   - 适用条件与局限：误差二阶矩条件不可去（否则期望可能不存在）；余项 $R_n(\mathcal{F})$ 的具体收敛速率受拟凸函数类度量熵的阶数限制，高维时可能较慢。

### 四、证明框架 / 方法设计

- **证明主干逻辑**：经验过程 + 截断法 + 离散化逼近。
- **拆解关键逻辑步骤**：
  1. **存在性构造**：从极小化序列出发，利用拟凸函数的下水平集表示，将无限维函数收敛转化为集合的包含关系，证明存在逐点收敛的子列且极限仍在 $\mathcal{F}$ 中。
  2. **MIQP 离散化**：将连续函数空间离散化，引入二值变量指示下水平集的包含关系，将拟凸约束转化为线性约束，目标函数为二次，形成 MIQP 问题，证明其解逼近连续 LSE。
  3. **截断处理矩条件**：对 $Y_i$ 进行截断，分离有界部分和重尾部分，利用 Markov 不等式控制重尾部分在风险中的贡献。
  4. **度量熵与经验过程**：计算拟凸函数类的括号熵，结合截断后的有界经验过程，应用 Syntychakov-Achieser 型引理控制经验极大值。
  5. **Oracle 不等式推导**：利用 LSE 的一阶条件（残差正交性），结合局部变分不等式，将估计误差与真实函数的偏差联系起来，得到 sharp 常数。
- **最关键的技巧性引理或"跳跃点"**：将拟凸约束转化为下水平集的嵌套结构，并利用二值变量进行 MIQP 刻画。这不仅在计算上使无限维问题可解，在理论上也为度量熵的计算提供了离散化逼近的桥梁。另外，仅用二阶矩控制异方差经验过程的截断技术也是核心难点。
- **数学工具评价**：经典经验过程与截断法的巧妙组合，结合了运筹学中的 MIQP 框架，属于跨领域工具的深度融合，而非全新分析框架。

### 五、与研究者兴趣的关联
- **连接到哪个子方向**：非参数理论中的形状约束推断 + 统计计算中的混合整数规划。
- **可借鉴的核心思路或技术工具**：
  1. **MIQP 建模思路**：将函数的形状约束（如拟凸、单调）转化为离散逻辑约束（下水平集的包含关系），这对处理复杂半参数/非参数约束的数值求解极具启发性，可迁移至其他复杂约束下的 M-估计计算。
  2. **弱矩条件下的经验过程控制**：仅依赖二阶矩的截断技术与 Oracle Inequality 推导，可迁移至高维或半参数模型中处理重尾异方差数据的效率界推导。
- **值得精读的关键参考文献**：
  1. Seijo & Lim (2011), "Nonparametric least squares estimation of a multivariate convex regression function": 凸回归 LSE 的存在性与渐近理论，对比阅读可深刻理解拟凸约束非凸性带来的拓扑困难。
  2. van de Geer (2000), "Empirical Processes in M-estimation": 弱矩条件下 Oracle inequality 的标准工具书，理解本文截断技术与经验过程控制的理论基础。

### 六、延伸思考与练习
- **假设扰动**：若将拟凸性假设放宽为 $\alpha$-拟凸（即 $f(\alpha x + (1-\alpha)y) \le \max\{f(x), f(y)\} + C\|x-y\|$，近似拟凸），结论会如何变化？技术上需要引入近似括号熵或局部 Rademacher 复杂度来控制近似误设带来的偏差项。
- **开放问题**：拟凸 LSE 在固定设计下的逐点收敛速率和渐近分布是什么？目前仅有有限样本风险界，渐近分布因形状约束的边界效应（如结点处的非标准分布）而极难推导。
- **理解检测题**：假设误差项 $\epsilon_i$ 服从 Cauchy 分布（一阶矩不存在），本文的 Oracle Inequality 是否仍然成立？如果不成立，需要对估计量（如从 LSE 换为 LAD）或证明框架做何种最小修改才能获得类似的风险界？
