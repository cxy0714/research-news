# Augmented doubly robust post-imputation inference for proteomic data

**作者**: Haeun Moon, Jin-Hong Du, Jing Lei, Kathryn Roeder  
**来源**: Annals of Applied Statistics  
**主题**: 效率理论 / Debiased ML  
**相关性**: 8/10  
**链接**: <https://doi.org/10.1214/25-aoas2012>

---

### 一、核心问题与贡献
①研究了质谱蛋白质组学数据中因大量缺失值导致插补后下游推断产生系统性偏差的问题。②核心方法是结合变分自编码器（VAE）高维插补与倾向得分参数估计，构建嵌入双重机器学习（DML）框架的增强双重稳健（DR）估计量。③在插补模型或缺失机制模型至少一个正确时，估计量具有 $n^{-1/2}$ -相合性与渐近正态性，并达到半参数有效界，在真实数据中实现了假阳性控制与统计功效的提升。

### 二、基础设定
- **核心概念与符号**：
  - $Y$：目标蛋白质丰度（部分观测）
  - $X$：高维肽段数据（辅助插补的协变量）
  - $R$：缺失指示变量（$R=1$ 观测，$R=0$ 缺失）
  - $\mu = E[Y]$：目标总体参数
  - $\pi(X) = P(R=1|X)$：倾向得分
  - $m(X) = E[Y|X]$：结果回归模型（由 VAE 插补值近似）
  - $\psi$：正交得分函数
- **关键假设**：
  - **强可忽略性 / MAR**：$R \perp Y | X$。统计学含义为给定高维肽段数据，蛋白质缺失与丰度无关。相比因果推断中的 unconfoundedness，这是标准的缺失数据假设，但在蛋白质组学中要求肽段信息能充分解释缺失机制（如检测极限导致的缺失）。
  - **重叠性**：$0 < P(R=1|X) < 1$ 几乎处处成立。避免倾向得分趋于 0 导致逆概率加权（IPW）方差爆炸。
  - **乘积误差速率条件**：$\|\hat{\pi} - \pi\|_2 \|\hat{m} - m\|_2 = o_p(n^{-1/2})$。相比传统 DR 估计要求单一模型达到 $o_p(n^{-1/4})$，该假设允许 VAE 插补模型收敛较慢，只要倾向得分模型足够精确即可。
- **问题背景**：现有蛋白质组学插补方法（如 KNN, Matrix Factorization）仅关注点估计恢复，忽略插补不确定性及偏差，导致下游假设检验失效（假阳性失控或功效低下）。与 Robins et al. (1994) 的经典 DR 估计和 Chernozhukov et al. (2018) 的 DML 相比，本文针对高维肽段-蛋白质层级结构，将 VAE 作为非参插补器引入 DR 框架，并严格证明了其 DML 框架下的有效性与推断性质。

### 三、主要定理 / 核心结果
1. **原文陈述**：在 DML 交叉拟合框架下，若乘积误差速率条件满足，则 $\sqrt{n}(\hat{\mu} - \mu) \rightsquigarrow N(0, V)$，其中 $V$ 为半参数有效界。
2. **直观解释**：只要倾向得分模型和插补模型中有一个是相合的，估计量就能消除插补带来的系统性偏差；若两者皆相合，则达到无模型误设风险的最优方差（半参数有效界）。
3. **解决了什么技术难点**：解决了高维 VAE 插补器收敛速率慢（通常达不到 $n^{-1/4}$）导致传统 DR 估计失效的问题，通过 DML 的交叉拟合与正交得分，允许机器学习方法的过拟合而不引入一阶偏差。
4. **适用条件与局限**：MAR 假设是核心局限，若缺失机制为 MNAR（非随机缺失，如丰度极低导致必然缺失），DR 框架识别性破坏；此外，倾向得分极小时，DR 估计方差仍会急剧增大。

### 四、证明框架 / 方法设计
- **识别策略与估计量设计**：
  - 识别策略：利用经典缺失数据 DR 公式 $\mu = E\left[ \frac{R}{\pi(X)} Y - \frac{R-\pi(X)}{\pi(X)} m(X) \right]$。由于 $Y$ 缺失时不可观测，用 VAE 插补值 $\hat{Y}$ 替代 $m(X)$ 的角色。
  - 估计量设计：构造正交得分 $\psi(O; \mu, \eta) = \frac{R}{\pi(X)} Y - \frac{R-\pi(X)}{\pi(X)} \hat{Y} - \mu$，其中 $\eta = (\pi, m)$ 为干扰参数。通过 DML 交叉拟合求解 $\sum \psi = 0$。
- **核心假设的可信度分析**：
  - MAR 假设：在质谱数据中，缺失通常由于肽段丰度过低导致检测不到，而肽段丰度由 $X$ 捕获，因此给定 $X$ 下 MAR 具有生物学合理性。但若存在未被 $X$ 捕获的系统性缺失（如特定实验批次效应），MAR 可能违背。
  - 验证方式：文中未显式验证 MAR（通常不可验证），但通过模拟中控制 FPR 的稳健性间接支持。
- **稳健性检验策略**：模拟中对比了不同缺失比例、不同插补模型误设下的表现，验证 DR 性质；真实数据中通过对比已知无差异蛋白质组的 FPR 来验证。
- **计算/实现细节**：VAE 用于高维肽段到蛋白质的降维与插补；倾向得分采用参数模型（如逻辑回归）；DML 交叉拟合将数据分 $K$ 折，交替训练干扰参数与计算正交得分，计算复杂度主要在 VAE 训练与折数 $K$ 的倍数。

### 五、与研究者兴趣的关联
- **连接到哪个子方向**：DML 框架下的半参数有效推断 与 缺失数据/因果推断中的双重稳健估计。
- **可借鉴的核心思路**：将高维非参/机器学习模型（如 VAE）作为结果回归（插补）器，结合低维参数倾向得分模型，利用 DML 正交得分消除 ML 插补偏差的范式。这一"ML预测 + 参数去偏 + DML正交化"的拆解思路，可直接迁移至高维因果推断中处理高维混淆因子或高维中介变量的场景。
- **值得精读的关键参考文献**：
  1. Chernozhukov et al. (2018) "Double/debiased machine learning for treatment and structural parameters"：DML 正交化与交叉拟合的理论基石，理解本文去偏机制的核心。
  2. Robins, Rotnitzky & Zhao (1994) "Estimation of regression coefficients when some regressors are not always observed"：经典 DR 估计与半参数效率界的奠基之作，本文是其在高维 ML 插补下的直接推广。

### 六、延伸思考与练习
- **假设扰动**：若缺失机制为 MNAR（如缺失概率同时依赖于未观测的 $Y$），DR 识别公式失效。技术上需要引入敏感性分析模型（如 pattern-mixture 或 selection model），或利用工具变量/阴性对照变量来识别因果效应/总体均值。
- **开放问题**：如何在 MNAR 机制下对高维蛋白质组学数据进行有效推断？能否将 Proximal Causal Inference 的阴性对照思想引入此框架，利用某些始终观测到的肽段作为阴性对照来识别 MNAR 下的参数？
- **理解检测题**：假设 VAE 插补模型完全误设（即 $E[\hat{Y}|X]$ 与 $E[Y|X]$ 差距很大，不满足相合性），但倾向得分模型 $\pi(X)$ 是正确且参数估计达到 $\sqrt{n}$-相合的。请推导此时 DML 估计量 $\hat{\mu}$ 的渐近展开，并说明其为何仍具有 $\sqrt{n}$-相合性，以及其渐近方差与半参数有效界有何区别。
