# Asymptotic properties of resampling‐based processes for the average treatment effect in observational studies with competing risks

**作者**: Jasmin Rühl, Sarah Friedrich  
**来源**: Scandinavian Journal of Statistics  
**主题**: 因果推断  
**相关性**: 8/10  
**链接**: <https://doi.org/10.1111/sjos.12714>

---

### 一、核心问题与贡献
①研究了观察性研究中存在竞争风险的时间-事件数据下，基于 g-formula 的平均处理效应（ATE）随机过程的推断问题。②核心工具是结合经验过程理论与计数过程鞅方法，严格证明三种重抽样方法（非参数 Bootstrap 及两种替代方案）的渐近有效性。③主要贡献在于解决了 ATE 随机过程渐近分布复杂无法直接用于构造同时置信带的难题，建立了重抽样过程与影响函数驱动过程的渐近等价性。

### 二、基础设定
- **核心概念与符号**：
  - $ATE(t)$：时间 $t$ 处的平均处理效应（基于潜在结果框架）。
  - Competing risks（竞争风险）：存在多种互斥事件，关注特定原因的发生概率（Cause-specific hazard 与 Cumulative incidence function, CIF）。
  - g-formula：在无未测量混杂下，对潜在 CIF 的标准化（或参数/半参数）估计公式。
  - $\hat{ATE}(t)$：基于 g-formula 的 ATE 估计量随机过程。
  - $\mathbb{G}_n$ 与 $\mathbb{G}_n^*$：原样本与重抽样经验过程算子。
- **关键假设**：
  - No unmeasured confounding (Ignorability)：$Y(t) \perp\!\!\!\perp A \mid L$，给定协变量下处理与潜在生存时间独立。与标准生存分析相同，但在竞争风险下需对每种原因别风险成立。
  - Positivity：$P(A=a \mid L) > \epsilon > 0$ a.s.，处理分配概率有下界。
  - Independent censoring (独立删失)：删失机制独立于事件过程（或给定协变量下条件独立），这是应用计数过程鞅理论的前提。
  - 相比已有文献：本文不依赖参数模型正确设定的强假设（允许半参数 g-formula），但在删失机制上仍依赖经典的鞅可料性假设。
- **问题背景**：
  - 已有方法不足：ATE 随机过程的渐近协方差结构极其复杂（涉及非参数协方差估计的逆问题与鞅积分泛函），难以解析求出并用于构造 time-simultaneous confidence bands。
  - 与最相关文献的区别：(1) 相比 Gill (1989) 等针对 Kaplan-Meier 的经典 Bootstrap 理论，本文处理了竞争风险与 g-formula 的复杂泛函结构；(2) 相比单纯依赖渐近正态的点估计，本文通过重抽样解决了整个随机过程的一致推断问题。

### 三、主要定理 / 核心结果
1. **原文陈述**：在正则条件下，基于重抽样的 ATE 过程 $\sqrt{n}(\hat{ATE}^*(t) - \hat{ATE}(t))$ 依概率条件弱收敛于与原过程 $\sqrt{n}(\hat{ATE}(t) - ATE(t))$ 相同的零均值高斯过程，两者具有一致的协方差结构。
2. **直观解释**：重抽样过程在原估计量附近的波动，完美模拟了原估计量在真实参数附近的波动。因此，重抽样分位数可以直接用来构造时间维度上的同时置信带，无需解析估计复杂的协方差泛函。
3. **解决了什么技术难点**：绕过了 g-formula 泛函在竞争风险下极其繁琐的渐近协方差矩阵解析求逆问题，通过条件弱收敛保证了置信带覆盖率的渐近有效性。
4. **适用条件与局限**：必须满足无未测量混杂和独立删失假设；若协变量维度较高，非参数 Bootstrap 的收敛速度可能受维数诅咒影响（本文未涉及高维设定）。

### 四、证明框架 / 方法设计
- **证明主干逻辑**：经验过程理论 + 计数过程鞅方法 + 函数 Delta 方法。
- **拆解为 4 个关键逻辑步骤**：
  1. **影响函数分解**：将 $\hat{ATE}(t)$ 通过函数 Delta 方法展开为其影响函数 $\phi$ 的经验均值，影响函数包含处理机制与原因别风险的鞅积分项。
  2. **鞅表示与经验过程映射**：证明影响函数驱动的经验过程 $\mathbb{G}_n \phi$ 弱收敛于高斯过程，利用鞅的可料变差与指数鞅不等式控制尾项。
  3. **重抽样条件分布的逼近**：证明在给定原样本下，重抽样经验过程 $\mathbb{G}_n^* \phi^*$ 依概率条件弱收敛于同一高斯过程。
  4. **余项控制**：利用 Donsker 定理或包络函数条件，证明非线性余项 $o_P(1)$ 在重抽样下依然为 $o_{P^*}(1)$。
- **最关键的技巧性引理或"跳跃点"**：如何在竞争风险的鞅结构下验证重抽样经验过程的随机等度连续性。需要构造特定的随机权重，并证明带权鞅积分过程的变差在重抽样下依然受控，这是连接鞅论与经验过程理论的桥梁。
- **数学工具评价**：是经典工具的精巧组合。将 Robins 的 g-formula 影响函数分解与 Andersen-Gill 的计数过程鞅理论结合，再套用 van der Vaart 经验过程的重抽样框架。

### 五、与研究者兴趣的关联
- **连接到哪个子方向**：纵向因果推断 与生存分析的结合，具体为竞争风险设定下的半参数推断与重抽样理论。
- **可借鉴的核心思路**：将 g-formula 估计量显式写为影响函数的无偏估计方程形式，并利用鞅结构处理时间维度的相依性，这一套"影响函数+鞅+经验过程"的组合拳可直接迁移到带有随时间变化混杂的纵向因果推断中，用于构造纵向 ATE 轨迹的同时置信带。
- **值得精读的关键参考文献**：
  1. Martinussen & Scheike (2006), *Dynamic Regression Models for Survival Data* - 提供了竞争风险与鞅方法的基础框架，理解鞅积分项方差的必读。
  2. van der Vaart & Wellner (1996), *Weak Convergence and Empirical Processes* - 重抽样渐近有效性的理论基石（特别是条件 Bootstrap 的经验过程理论）。
  3. Robins (1986) / Gill & Robins (2001) - g-formula 与因果推断影响函数的奠基性工作。

### 六、延伸思考与练习
- **假设扰动**：若放宽"无未测量混杂"假设，引入 sensitivity parameter $\Gamma$，ATE 过程将不再点识别，而是形成一个区间过程。技术上需要将鞅的零均值假设替换为受 $\Gamma$ 约束的漂移项，重抽样的渐近有效性需在漂移鞅框架下重新验证。
- **开放问题**：如何将此重抽样框架拓展到高维协变量设定（如 $p \gg n$），结合 debiased ML 或 double machine learning 估计 g-formula 中的子模型，并保证鞅积分项与交叉拟合过程的联合渐近性质？
- **理解检测题**：假设删失机制不仅依赖于基线协变量 $L$，还依赖于随时间变化的中间变量 $M(t)$，而 $M(t)$ 同时受处理 $A$ 和竞争事件影响。请分析此时 g-formula 估计量的影响函数结构会发生什么变化，并说明非参数 Bootstrap 在此情况下是否仍然渐近有效，为什么？
