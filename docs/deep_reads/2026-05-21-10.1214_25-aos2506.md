# Semiparametric modeling and analysis for longitudinal network data

**作者**: Yinqiu He, Jiajin Sun, Yuang Tian, Zhiliang Ying, Yang Feng  
**来源**: Annals of Statistics  
**主题**: 非参数 / 半参数  
**相关性**: 10/10  
**链接**: <https://doi.org/10.1214/25-aos2506>

---

### 一、核心问题与贡献
①研究了纵向网络数据中静态潜空间参数的半参数估计与推断问题。②通过构造调整了时间变化节点基线干扰的有效得分方程，结合商流形解决不可识别性，提出了一步更新估计量与惩罚最大似然估计。③证明了两种估计量均达到oracle误差界，实现了复杂网络依赖结构下的半参数有效估计。

### 二、基础设定
- **核心概念与符号**：
  - $A^{(t)}$：$t$ 时刻的邻接矩阵，$A_{ij}^{(t)}$ 为节点 $i$ 和 $j$ 在 $t$ 时的连边指示变量。
  - $Z_i \in \mathbb{R}^d$：节点 $i$ 的静态潜空间位置（感兴趣参数）。
  - $\mu_i^{(t)}$：节点 $i$ 在 $t$ 时刻的特异性基线干扰（无穷维干扰参数）。
  - 连边概率：$P(A_{ij}^{(t)}=1 \mid Z_i, Z_j, \mu_i^{(t)}, \mu_j^{(t)}) = \text{expit}(\mu_i^{(t)} + \mu_j^{(t)} - \|Z_i - Z_j\|)$。
  - 商流形 $\mathcal{M}$：处理 $Z$ 的正交群不变性导致的不可识别问题，参数空间视作 $\mathbb{R}^{n \times d} / O(d)$。
- **关键假设**：
  - **条件独立性**：给定潜空间位置 $Z$ 与基线干扰 $\mu^{(t)}$，网络连边条件独立。统计学含义是网络的所有依赖结构均被潜变量捕获，是潜空间模型的标准假设，但在纵向设定下跨时间依赖由 $\mu^{(t)}$ 刻画。
  - **半参数模型设定**：$\mu^{(t)}$ 为无穷维干扰参数，$Z$ 为有限维感兴趣参数。相比传统动态潜空间模型（如时间序列先验的 $Z^{(t)}$），该假设将时变效应完全剥离至 $\mu^{(t)}$，避免了潜空间参数的维度爆炸。
  - **商流形可识别性**：$Z$ 的正交变换不改变连边概率。相比传统的固定某节点坐标或加惩罚约束，该假设从微分几何本质出发，允许估计量在流形切空间上定义，不损失信息。
- **问题背景**：现有纵向网络模型多将潜空间设为时变，导致参数过多且难以分离稳定的网络拓扑结构；或使用参数化基线，缺乏灵活性。与最相关文献的区别：Hoff (2008) 的动态潜空间模型缺乏半参数效率理论且参数维度高；Bickel & Chen (2009) 等截面网络半参数文献未处理纵向时间异质性与商流形上的有效得分构造。

### 三、主要定理 / 核心结果

1. **定理：有效得分方程的推导**
   - **原文陈述**：潜空间参数 $Z$ 的有效得分 $\tilde{S}_Z$ 为原始得分 $S_Z$ 减去其在基线参数 $\mu$ 得分空间 $S_\mu$ 上的投影，即 $\tilde{S}_Z = S_Z - \Pi(S_Z \mid S_\mu)$。
   - **直观解释**：通过正交投影剔除了由时变节点活跃度（干扰参数）引起的连接概率变异，使得得分函数对 $\mu$ 的估计误差具有 Neyman 正交性，只保留纯粹的潜空间结构信息。
   - **技术难点**：纵向网络中 $\mu^{(t)}$ 的无穷维性质与网络复杂的条件依赖结构，使得信息算子的求逆与投影计算极其复杂。
   - **适用条件与局限**：需要网络稀疏度与节点数的特定阶数关系，以及 $\mu^{(t)}$ 的平滑/有界条件。局限在于条件独立性假设较强，若存在未捕获的同配性残差，正交性可能被破坏。

2. **定理：一步更新估计量与惩罚MLE的Oracle误差界**
   - **原文陈述**：在商流形切空间上，两种估计量 $\hat{Z}$ 均满足 $d(\hat{Z}, Z_0) = O_P(\sqrt{p/n})$（或相应的 oracle 收敛速率），其中 $p$ 为参数维度，$n$ 为节点数。
   - **直观解释**：即使存在无穷维的时变干扰参数 $\mu^{(t)}$，估计 $Z$ 的精度仍如同已知 $\mu^{(t)}$ 一样，达到了半参数有效界。
   - **技术难点**：证明经验过程的收敛性，特别是处理网络数据的非独立同分布结构（给定 $Z$ 下边的条件依赖）以及商流形上的距离度量。
   - **适用条件与局限**：网络不能太稠密也不能太稀疏，惩罚参数需恰当选择。一步更新需要相合的初始估计，惩罚MLE则需要合适的正则化项避免流形上的过拟合。

### 四、证明框架 / 方法设计

- **证明主干逻辑**：构造法 + 经验过程 + 矩方法。
- **拆解关键逻辑步骤**：
  1. **商流形几何结构建立**：定义参数空间的等价类，将 $Z$ 的不可识别性转化为流形上的切空间投影问题，证明得分函数在横截面上的良定义性。
  2. **有效得分与信息算子计算**：在潜变量模型下推导 $S_Z$ 与 $S_\mu$，利用条件期望性质计算信息算子 $I_{Z\mu}$ 与 $I_{\mu\mu}$，得到投影 $\Pi = I_{Z\mu}I_{\mu\mu}^{-1}$，构造正交得分 $\tilde{S}_Z$。
  3. **一步更新构造与展开**：基于初始估计 $\hat{Z}^{init}$，构造一步更新 $\hat{Z}^{os} = \hat{Z}^{init} + \hat{I}^{-1} \tilde{S}_Z(\hat{Z}^{init}, \hat{\mu})$，对更新项进行泰勒展开。
  4. **余项控制与Oracle界推导**：利用经验过程技术与网络相依数据的浓度不等式，证明二阶余项 $o_P(1/\sqrt{n})$，从而更新步骤不损失精度，达到有效界。
- **最关键的技巧性引理或"跳跃点"**：在商流形上定义有效得分方程。传统的半参数理论在欧式空间进行，但潜空间的旋转不变性使得 Fisher 信息矩阵奇异。作者将参数空间视作商流形，在切空间的横截面上进行推断，巧妙避开了奇异性问题，这是几何视角与统计推断的深刻结合。
- **数学工具评价**：是经典半参数理论（Begun et al., 1983; Bickel et al., 1993）与微分几何在网络数据非标准依赖结构下的深度结合，而非简单套用。

### 五、与研究者兴趣的关联
- **连接子方向**：半参数效率理论在非标准依赖数据（网络/图数据）中的应用；Neyman 正交性在复杂干扰参数下的构造。
- **可借鉴的核心思路**：1) 如何在具有复杂条件依赖的数据中推导和计算有效得分函数，特别是处理信息算子的求逆；2) 商流形视角处理不可识别参数，比传统的锚定节点法更具理论优雅性，可迁移到其他存在群不变性的潜变量模型（如某些因果潜变量模型、测量误差模型）。
- **值得精读的关键参考文献**：
  1. Bickel et al. (1993) *Efficient and Adaptive Estimation for Semiparametric Models*: 半参数有效得分方程与一步估计的经典理论基石，理解本文投影逻辑的必读。
  2. Ma & Zhu (2012) *A semiparametric approach to dimension reduction*: 同样涉及半参数有效得分与流形结构（降维中的仿射等变群），对理解本文几何视角极有帮助。
  3. Hoff (2008) *Modeling homophily and stochastic equivalence in symmetric relational data*: 潜空间模型的奠基作，理解本文模型出发点与局限。

### 六、延伸思考与练习
- **假设扰动**：若假设节点特异性基线 $\mu_i^{(t)}$ 不仅随时间变化，还依赖于其邻居节点的历史连边状态（即存在内生性/同群效应），结论如何变化？技术上需要引入工具变量或 proximal 变量来剥离内生性，原有的条件独立性假设被破坏，有效得分方程的投影空间需包含历史网络信息。
- **开放问题**：如何将此框架扩展到动态潜空间 $Z_i^{(t)}$（如带马尔可夫转移），同时保持对时变干扰参数的半参数有效性？这需要处理无穷维 $Z^{(t)}$ 与 $\mu^{(t)}$ 的双干扰结构。
- **理解检测题**：假设模型中不存在时变基线干扰 $\mu_i^{(t)}$（即 $\mu_i^{(t)} = \mu_i$ 为常数），请写出此时潜空间参数 $Z$ 的有效得分函数，并说明它与本文一般情况下的有效得分函数有何本质区别？（提示：考察 $S_\mu$ 张成的空间维数变化及投影的消失情况）。
