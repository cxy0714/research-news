# Variable Significance Testing for the Deep Cox Model

**作者**: Qixian Zhong, Jonas Mueller, Jane-Ling Wang  
**来源**: Journal of the American Statistical Association  
**主题**: 数理统计 / 假设检验  
**相关性**: 8/10  
**链接**: <https://doi.org/10.1080/01621459.2026.2615850>

---

### 一、核心问题与贡献
①研究了在非参数 Cox 模型下，如何对特定协变量与删失生存时间之间的相关性进行变量显著性检验。②核心方法是利用深度神经网络（DNN）建模非参数连接函数，并通过样本分割与交叉拟合构造检验统计量。③证明了 DNN 估计量可利用数据低维结构克服维数诅咒，且所提检验统计量在零假设下渐近正态、在对立假设下检验相合，实现了黑箱模型的严格统计推断。

### 二、基础设定
- **核心概念与符号**：
  - 非参数 Cox 模型：$\lambda(t|X) = \lambda_0(t) \exp\{f(X)\}$，其中 $f(\cdot)$ 为非参数连接函数，$X=(W,Z)$，$W$ 为待检验协变量，$Z$ 为控制变量。
  - 零假设 $H_0$：$W$ 与生存时间条件独立，即 $f(W,Z) = f(Z)$ 几乎处处成立。
  - $\hat{f}$：基于 DNN 的非参数连接函数估计量。
  - Cross-fitting：将样本分为 $K$ 折，轮流作为训练集估计 $\hat{f}$ 与作为推断集构造检验统计量，以消除过拟合带来的偏度。
- **关键假设**：
  - 条件独立删失：$C \perp T | X$（Cox 模型标准假设，保证似然结构的可识别性）。
  - 低维结构假设：$f(X)$ 具有内在低维结构（如加性结构或组合结构），这是 DNN 克服维数诅咒的核心前提，相比传统非参数光滑假设（需维度 $p$ 极小），大幅放宽了对全维度的严格限制。
  - DNN 复杂度约束：对网络深度、宽度及权重矩阵范数施加正则化限制，以平衡逼近误差与估计误差。
- **问题背景**：现有深度生存分析侧重预测而缺乏推断能力，黑箱性质导致无法进行假设检验。与 Farrell et al. (2021) 的 DNN 推断工作相比，本文处理了删失数据下的计过程鞅结构；与经典半参数 Cox 推断相比，本文避免了线性或比例风险结构的模型误设风险。

### 三、主要定理 / 核心结果
1. **DNN 估计量收敛速率（Theorem 1）**
   - **原文陈述**：在低维结构下，DNN 估计量 $\hat{f}$ 的 $L_2$ 收敛速率可达 $o_P(n^{-1/4})$。
   - **直观解释**：DNN 能够自适应数据的低维流形或加性结构，使得有效非参数维数降低，从而满足半参数推断所需的 $n^{-1/4}$ 关键速率门槛。
   - **技术难点**：在删失机制下建立 DNN 的逼近误差与估计误差界，需处理非凸经验风险最小化的局部极值问题。
   - **适用条件与局限**：依赖于低维结构的正确设定；若数据确为极高维无结构，速率仍会退化，无法进行后续推断。

2. **零假设下渐近正态性（Theorem 2）**
   - **原文陈述**：在 $H_0$ 下，基于 cross-fitting 构造的检验统计量 $T_n \xrightarrow{d} N(0, \sigma^2)$。
   - **直观解释**：样本分割切断了 nuisance parameter $\hat{f}$ 估计误差对检验统计量的高阶依赖，使得一阶线性近似（鞅积分）主导，从而获得中心极限定理。
   - **技术难点**：处理 $\hat{f}$ 估计误差在计过程鞅表示中的传播，证明交叉拟合后的二阶余项为 $o_P(1)$。
   - **适用条件与局限**：必须要求 $\hat{f}$ 收敛速率快于 $n^{-1/4}$；若不使用 cross-fitting，余项将包含复杂的经验过程项，难以控制。

3. **对立假设下检验相合性（Theorem 3）**
   - **原文陈述**：在固定对立假设 $H_1$ 下，$P(\text{Reject } H_0) \to 1$。
   - **直观解释**：只要 $f$ 依赖于 $W$ 的信号非零，检验统计量的均值将偏离 0 且方差可控，检验势趋于 1。

### 四、证明框架 / 方法设计
- **证明主干逻辑**：经验过程理论 + 计过程鞅论 + 样本分割解耦。
- **拆解关键逻辑步骤**：
  1. **Nuisance 估计与收敛分析**：利用 DNN 的逼近理论，证明在低维结构下 $\|\hat{f} - f\|_{L_2} = o_P(n^{-1/4})$。
  2. **鞅分解与线性化**：将基于 $\hat{f}$ 的对数偏似然或得分函数展开，利用计数过程鞅性质，将统计量分解为均值零的鞅项（主导项）和包含 $\hat{f}-f$ 的经验项（余项）。
  3. **样本分割与余项控制**：利用样本分割，证明余项（二阶项）在给定训练集下依概率收敛于 0，因为 $\hat{f}$ 与构造统计量的样本独立。
  4. **Cross-fitting 汇总**：将 $K$ 折的鞅项相加，利用独立增量性质与中心极限定理，得到整体的渐近正态性。
- **最关键的技巧性引理/跳跃点**：在删失数据下，将 DNN 估计误差 $\hat{f}-f$ 转化为鞅积分中的可控项。具体而言，需证明 $\int (\hat{f}-f) dM(t)$ 的条件方差可被 $\|\hat{f}-f\|^2$ 控制，这里样本分割是切断 $\hat{f}$ 与 $dM(t)$ 之间复杂依赖的关键跳跃点。
- **数学工具评价**：是 DNN 逼近论（Schmidt-Hieber 型结构）与半参数推断中 sample splitting/cross-fitting（Chernozhukov DML 范式）在生存分析鞅框架下的巧妙组合，非全新分析框架，但在删失数据下的结合极具技术含量。

### 五、与研究者兴趣的关联
- **连接子方向**：Debiased ML / semiparametric efficiency 在生存分析与因果推断中的应用。
- **可借鉴思路**：将 DML/Cross-fitting 的推断框架从均值回归/因果效应推广到计过程与鞅结构。这种"估计 nuisance parameter (DNN) -> 样本分割 -> 构造 score/martingale statistic -> 渐近正态"的路径，可直接迁移到带删失的因果推断（如生存时间的 IV 估计或 treatment effect 估计）中，解决黑箱 nuisance 下的有效推断问题。
- **值得精读的参考文献**：
  1. Chernozhukov et al. (2018) "Double/debiased machine learning for treatment and structural parameters": DML 与 cross-fitting 的基石文献，理解本文推断逻辑的必读。
  2. Farrell et al. (2021) "Deep neural networks for estimation and inference": DNN 推断的基础框架，本文将其拓展至删失数据鞅结构。
  3. Schmidt-Hieber (2020) "Nonparametric regression using deep neural networks with ReLU activation function": DNN 克服维数诅咒的低维结构理论依据。

### 六、延伸思考与练习
- **假设扰动**：若放宽 DNN 收敛速率至 $O_P(n^{-1/4})$（即恰好达到边界）或更慢，渐近正态性将失效，检验统计量存在不可忽略的偏度。技术上需要引入 higher-order influence function 或额外的 bias-correction 项来消除二阶余项的影响。
- **开放问题**：如何在非参数 Cox 模型下进行多重假设检验或变量选择后的有效推断，以克服 selection bias？能否将此框架推广至时依协变量或非比例风险模型（如加性风险模型）？
- **理解检测题**：假设不使用 sample splitting，而是用全样本同时估计 DNN 和构造检验统计量，在证明渐近正态性时，经验过程的 entropy bound 会遇到什么具体的数学障碍？DNN 的函数类复杂度在其中扮演了什么角色？
