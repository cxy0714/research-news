# Improving estimation efficiency for case-cohort studies with a cure fraction

**作者**: Qingning Zhou, Xu Cao  
**来源**: Biometrics  
**主题**: 其他  
**相关性**: 8/10  
**链接**: <https://doi.org/10.1093/biomtc/ujaf059>

---

### 一、核心问题与贡献
①研究了在广义 case-cohort 设计下，含治愈分数的生存数据中回归参数与治愈概率的半参数估计问题。②核心方法是基于 sieve maximum weighted likelihood 的两步估计法，并在第二步利用廉价协变量与结局的工作模型进行全样本信息更新。③证明了无论工作模型是否正确设定，更新后的估计量均具有一致性且渐近效率不低于仅用完全数据的估计量，实现了效率保证的 augmentation 策略。

### 二、基础设定
- **核心概念与符号**：
  - $T$: 事件时间，$C$: 删失时间，$Y=\min(T,C)$, $\Delta=I(T\le C)$。
  - Cure fraction: $P(T=\infty)>0$。Mixture cure model: $S(t|Z)=\pi(Z)+(1-\pi(Z))S_u(t|Z)$，其中 $\pi$ 为治愈概率，$S_u$ 为易感者生存函数。
  - Semiparametric transformation model: $g(S_u(t|Z))=\Lambda(t)+\beta^T Z$，$\Lambda(t)$ 为无限维基准累积危险率。
  - Case-cohort 设计：子队列 $\mathcal{V}$（简单随机抽样），病例 $\mathcal{C}$（发生事件的个体）。抽样指示变量 $\xi_i$，IPW 权重 $\omega_i$。
  - 协变量划分：昂贵协变量 $Z$（仅子队列与病例可测），廉价/辅助协变量 $W$（全样本可测）。
- **关键假设**：
  - (A1) 独立删失：$T \perp C \mid Z$。
  - (A2) 缺失机制：$\xi \perp (T,C,Z) \mid \Delta$（即 MAR，仅依赖于结局 $\Delta$）。
  - (A3) Sieve 空间逼近：$\Lambda(t)$ 属于某光滑函数空间，被有限维 B-spline 基底 $\Lambda_n(t)=\sum \gamma_k B_k(t)$ 逼近，且满足 bracketing number 的熵条件。
  - (A4) 工作模型设定：第二步的 augmentation 工作模型允许任意误设，无需正确设定条件期望。
  - *假设比较*：相比于传统 AIPW 方法要求工作模型正确设定才能保证效率提升，本文放宽了工作模型设定要求，仅要求其足够平滑；相比于标准 cure model 的 IPW 估计，本文通过 augmentation 突破了仅使用完全数据的效率下界。
- **问题背景**：Case-cohort 设计中昂贵协变量存在缺失，标准 IPW 估计效率低；现有 AIPW 方法在 mixture cure 模型下若工作模型误设会导致效率损失甚至不一致。与最相关文献的区别：相比 Bang & Tsiatis (2000) 等标准 AIPW，本文处理了 cure fraction 带来的非规则参数问题与 $\Lambda(t)$ 的无限维推断；相比 Lu & Ying (2004) 的 case-cohort cure model，本文利用了辅助变量进行效率提升。

### 三、主要定理 / 核心结果
1. **原文陈述**：Theorem 1 证明第一步 sieve weighted MLE $\hat{\theta}_1$ 具有相合性，且 $\sqrt{n}(\hat{\theta}_1-\theta_0) \to N(0, \Sigma_1)$；Theorem 2 证明第二步更新估计量 $\hat{\theta}_2$ 同样相合且 $\sqrt{n}(\hat{\theta}_2-\theta_0) \to N(0, \Sigma_2)$，且 $\Sigma_2 \le \Sigma_1$（Loewner 序），无论工作模型是否正确。
2. **直观解释**：第二步的 augmentation 项本质上是将 IPW 估计方程投影到由廉价协变量生成的 $\sigma$-代数空间上。即使投影空间（工作模型）选错，该投影项在真实模型下仍期望为0，相当于引入了一个均值为0的控制变量，从而在不引入偏差的前提下降低了方差。
3. **解决了什么技术难点**：在 mixture cure model 下，事件时间分布的 Fisher 信息矩阵奇异（cure 部分不提供 $\Lambda(t)$ 的信息），导致标准半参数 M-估计理论失效。本文通过构造特定的正则化条件与筛空间逼近，解决了无限维参数 $\Lambda(t)$ 与有限维参数 $(\beta, \alpha)$ 的联合估计与渐近分离。
4. **适用条件与局限**：必须满足 MAR 假设（缺失仅依赖结局）；工作模型虽可误设，但必须满足一定的平滑性以保证经验过程的收敛；若缺失机制不仅依赖结局还依赖未观测的协变量，则结论失效。

### 四、证明框架 / 方法设计
- **证明主干逻辑**：经验过程理论 + Sieve M-估计 + 投影增广方差缩减。
- **拆解关键逻辑步骤**：
  1. **Sieve 空间收敛**：证明加权对数似然泛函在筛空间上的极大值点收敛到真实参数，利用 argmax 定理与 bracketing number 控制经验过程的震荡。
  2. **渐近线性化**：对第一步得分函数 $\mathbb{P}_n \dot{l}(\hat{\theta}_1)=0$ 在 $\theta_0$ 处展开，分离有限维参数与无限维参数 $\Lambda_n$ 的影响，证明筛逼近误差 $o_p(n^{-1/2})$。
  3. **增广项的无偏性**：构造增广项 $A(W,\Delta,Y;\hat{\theta})$，证明在真实模型下 $E[A \mid Z,T]=0$，因此即使工作模型误设，增广估计方程仍无渐近偏差。
  4. **方差比较**：计算 $\hat{\theta}_2$ 的影响函数，证明 $\Sigma_1 - \Sigma_2$ 等于增广项在影响函数空间投影的方差，由于方差非负，得证 $\Sigma_2 \le \Sigma_1$。
- **最关键的技巧性引理或"跳跃点"**：增广项的构造。作者构造的增广项满足：当工作模型正确时，它逼近半参数有效影响函数的投影；当工作模型错误时，它退化为一个与原得分函数正交的噪声项。这种"正交性"保证了误设不破坏一致性，是效率保证的核心跳跃点。
- **数学工具评价**：是经典工具的巧妙组合。将 Robins & Rotnitzky (1994) 的 AIPW 效率提升思想移植到 Shen & Wong (1994) 的 Sieve M-估计框架中，克服了 cure model 非规则参数的推断难题。

### 五、与研究者兴趣的关联
- **连接子方向**：Semiparametric efficiency bounds / Augmented IPW / Missing data mechanisms / Semiparametric estimation。
- **可借鉴的核心思路**：**"工作模型误设亦不损失效率"的 augmentation 构造策略**。在部分识别或缺失数据问题中，若能构造出某种增广项使其在真实数据生成机制下与主得分函数正交（期望为0），则可大胆引入机器学习等非参工作模型进行效率提升，而无需承担误设导致不一致的风险。这与 DML 中的 Neyman orthogonality 有异曲同工之妙，但视角侧重于方差缩减而非偏差消除。
- **值得精读的关键参考文献**：
  1. Robins, Rotnitzky, and Zhao (1994) - AIPW 与缺失数据半参数效率界的奠基之作，理解 augmentation 为何能提升效率的必读。
  2. Shen and Wong (1994) - Sieve MLE 的渐近理论，处理无限维参数半参数模型的标准数学工具。

### 六、延伸思考与练习
- **假设扰动**：若将假设 (A2) 放宽，允许缺失机制 $\xi$ 依赖于未观测的昂贵协变量 $Z$（即 Missing Not At Random），结论如何变化？技术上需要引入 Proximal Causal Inference 中的 Negative Control 思想，利用代理变量识别缺失机制或构造无偏增广项。
- **开放问题**：第二步的工作模型能否使用高维机器学习算法（如 Random Forests 或 Lasso）拟合，并通过 cross-fitting 消除正则化偏差，从而在保证 $\sqrt{n}$-相合性的同时逼近半参数有效界？
- **理解检测题**：假设第二步的工作模型完全误设为一个常数函数（即未利用任何 $W$ 的信息），证明此时 $\hat{\theta}_2$ 的渐近方差 $\Sigma_2$ 严格等于第一步 IPW 估计量 $\hat{\theta}_1$ 的渐近方差 $\Sigma_1$，并解释这一现象在投影理论下的几何意义。
