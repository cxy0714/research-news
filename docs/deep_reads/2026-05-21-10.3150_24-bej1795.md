# Edgeworth expansion by Stein’s method

**作者**: Xiao Fang, Song-Hao Liu  
**来源**: Bernoulli  
**主题**: 数理统计 / 假设检验  
**相关性**: 8/10  
**链接**: <https://doi.org/10.3150/24-bej1795>

---

### 一、核心问题与贡献
①研究了如何在不依赖特征函数的纯概率框架下，利用 Stein 方法证明一般情形下的二阶 Edgeworth 展开。②核心工具是反复求解 Stein 方程、基于 Stein kernel 的 Stein 恒等式以及替换论证。③在随机变量仅含连续分量且测试函数为半直线示性函数的条件下，恢复了经典的 Edgeworth 展开结果，克服了已有 Stein 方法证明对测试函数光滑性或变量绝对连续性的依赖。

### 二、基础设定
- **核心概念与符号**：
  - $W = n^{-1/2}\sum_{i=1}^n X_i$：标准化部分和。
  - Edgeworth 展开：$P(W \le x) = \Phi(x) + n^{-1/2} p_1(x)\phi(x) + n^{-1} p_2(x)\phi(x) + o(n^{-1})$。
  - Stein 方程：$\mathcal{A}f(w) - w f(w) = h(w) - \Phi(h)$，其中 $\mathcal{A}$ 为生成元，$\Phi(h) = E[h(Z)]$（$Z\sim N(0,1)$）。
  - Stein kernel $\tau(x)$：满足 $E[\tau(X) f'(X)] = E[X f(X)]$ 的函数，用于替代密度的分部积分。
- **关键假设**：
  - **仅含连续分量**：随机变量的分布没有纯点（离散）部分，但允许奇异连续部分。相比文献中要求绝对连续（具有密度）的假设，此条件显著放宽。
  - **Cramér 条件**：$\limsup_{|t|\to\infty} |E[e^{itX}]| < 1$。这是控制特征函数尾部衰减、保证 Edgeworth 展开余项 $o(n^{-1})$ 一致性的常规必要条件。
  - **矩条件**：存在足够高阶的矩（通常需 $E|X|^4 < \infty$ 或更高），以保证展开系数 $p_1, p_2$ 有限。
- **问题背景**：经典 Edgeworth 展开高度依赖特征函数的 Fourier 反演，难以推广至复杂依赖结构（如高维/非线性统计量）。已有 Stein 方法证明（如 Barbour 1986, Chen & Shao 2001）要么要求测试函数 $h$ 光滑（排除了分布函数逼近所需的示性函数 $I_{(-\infty, x]}$），要么要求 $X_i$ 绝对连续。本文填补了 Stein 方法恢复经典 Edgeworth 展开的空白。

### 三、主要定理 / 核心结果
1. **原文陈述**：在 Cramér 条件、矩条件及 $X_i$ 仅含连续分量的假设下，对 $W = n^{-1/2}\sum_{i=1}^n X_i$，有 $\sup_{x \in \mathbb{R}} \left| P(W \le x) - \Phi(x) - n^{-1/2} p_1(x)\phi(x) - n^{-1} p_2(x)\phi(x) \right| = o(n^{-1})$。
2. **直观解释**：标准化和的分布函数与标准正态分布函数的偏差，可被偏度和峰度驱动的 $n^{-1/2}$ 和 $n^{-1}$ 阶局部形变精确修正，且该逼近对非光滑的示性函数依然成立。
3. **解决了什么技术难点**：解决了 Stein 方法中测试函数 $h(w) = I_{(-\infty, x]}(w)$ 不光滑导致 Stein 方程解 $f$ 的导数在 $x$ 处存在跳跃，从而无法直接对 $E[f'(W)]$ 进行 Taylor 展开的难点；同时绕开了 $W$ 非绝对连续导致密度不存在、无法使用分部积分的障碍。
4. **适用条件与局限**：Cramér 条件是必要的，否则 $o(n^{-1})$ 衰减可能失效；仅含连续分量假设排除了纯离散变量，对于格点分布，Edgeworth 展开需加入格点修正项，本文框架暂未覆盖；目前仅做到二阶展开，更高阶展开的 Stein 方法框架尚未建立。

### 四、证明框架 / 方法设计
- **证明主干逻辑**：构造法与替换论证。
- **拆解为 3-5 个关键逻辑步骤**：
  1. **Stein 方程的嵌套求解**：对目标 $h = I_{(-\infty, x]}$ 求解 Stein 方程得 $f_1$；因 $f_1'$ 不光滑，对 $f_1'$ 再次构造并求解 Stein 方程得 $f_2$，将逼近误差转化为对高阶光滑函数 $f_2$ 的期望。
  2. **Stein kernel 恒等式的应用**：利用 Stein kernel $\tau$ 将 $E[W f(W)]$ 转化为 $E[\tau(W) f'(W)]$，绕开对 $W$ 密度函数存在的需求。
  3. **替换论证**：将 $W$ 中的单个 $X_i$ 替换为独立正态变量 $Z_i$，构造中间插值变量 $W^{(i)}$，利用 Taylor 展开和 Stein 恒等式计算替换引起的差分。
  4. **残差控制**：利用 Cramér 条件和矩条件，结合特征函数衰减性质与分部积分，控制替换后累积的高阶余项，证明其具有 $o(n^{-1})$ 的阶。
- **最关键的技巧性引理或"跳跃点"**：**反复使用 Stein 方程吸收奇异性**。由于 $I_{(-\infty, x]}$ 导致解 $f_1$ 的二阶导数 $f_1''$ 在 $x$ 处有 delta 函数性质的奇异性，直接展开失效。作者通过对 $f_1'$ 再次施加 Stein 方程，利用正态分布的平滑效应将奇异性吸收到 $f_2$ 的更高阶导数中，这是突破光滑性限制的核心跳跃点。
- **数学工具评价**：是经典 Stein 方程与 Stein kernel 技术的深度组合与拓展。将 Stein 方法从 $O(n^{-1/2})$ Berry-Esseen 界的证明提升到了 $o(n^{-1})$ 的渐近展开精度，属于分析框架的升级。

### 五、与研究者兴趣的关联
- **连接到哪个子方向**：高阶 U 统计量的分布逼近与假设检验的高阶渐近理论。
- **可借鉴的核心思路或技术工具**："反复 Stein 方程 + Stein kernel" 的框架可迁移到高阶 U 统计量的 Edgeworth 展开。传统 U 统计量高阶展开依赖 Hoeffding 分解与特征函数，若采用此框架，可通过构造 U 统计量的 Stein kernel，在不依赖核光滑性假设下对非光滑检验统计量（如经验分布函数、秩统计量）进行高阶修正与临界值精确计算。
- **值得精读的关键参考文献**：
  1. Chen, L. H. Y., & Shao, Q. M. (2001). "Stein's method and Edgeworth expansion." *Bernoulli*. （对比本文，理解为何旧方法需要绝对连续假设，以及本文如何突破）。
  2. Barbour, A. D. (1986). "Asymptotic expansions based on the Poisson approximation." *Advances in Applied Probability*. （理解 Stein 方法做高阶展开的早期范式与光滑性限制的根源）。

### 六、延伸思考与练习
- **假设扰动**：若随机变量 $X_i$ 包含纯点（离散）分量，Edgeworth 展开的形式将发生何种变化？此时 Stein kernel 是否仍然存在？技术上需要引入何种测度分解工具来处理示性函数在跳跃点处的行为？
- **开放问题**：如何将此基于 Stein 方程的 Edgeworth 展开框架推广到高维随机向量（如多变量中心极限定理的 Edgeworth 展开），以解决高维假设检验中的 size 控制问题？
- **理解检测题**：假设 $W_n$ 是一阶 Hoeffding 分解后的标准化 U 统计量，已知其 Stein kernel $\tau_{W_n}(w)$ 渐近趋于 1。请简述如何利用本文的"反复 Stein 方程"思路，将 $P(W_n \le x)$ 的逼近误差从 $O(n^{-1/2})$ 提升至 $o(n^{-1})$，并指出在此过程中 $\tau_{W_n}$ 扮演了什么角色。
