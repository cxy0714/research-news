# The empirical copula process in high dimensions: Stute’s representation and applications

**作者**: Axel Bücher, Cambyse Pakzad  
**来源**: Annals of Statistics  
**主题**: 非参数 / 半参数  
**相关性**: 9/10  
**链接**: <https://doi.org/10.1214/25-aos2548>

### 一、核心问题与贡献
①研究了高维设定下（维度$d$相对样本量$n$指数级增长）经验copula过程的渐近表示问题。②核心方法是在底层copula满足弱光滑性条件下，利用标准经验过程的低维边际泛函去逼近经验copula过程的低维边际。③主要结论是证明了Stute表示在高维下依然有效（几乎必然误差关于边际一致有界），并将其应用于高维两两独立性检验，在比互不独立更广的模型类下实现了type-I error控制及Bootstrap FWER的强控制。

### 二、基础设定
- **核心概念与符号**：
  - $d$：数据维度，允许 $d = d(n) \to \infty$ 且 $\log d = o(n^\alpha)$（指数级高维）。
  - $C$：底层真实 copula 函数；$C_n$：经验 copula。
  - $\mathbb{C}_n(\boldsymbol{u}) = \sqrt{n}\{C_n(\boldsymbol{u}) - C(\boldsymbol{u})\}$：经验 copula 过程。
  - $\mathbb{G}_n$：标准经验过程。
  - $\tilde{\mathbb{C}}_n(\boldsymbol{u}) = \mathbb{G}_n(\boldsymbol{u}) - \sum_{j=1}^d \dot{C}_j(\boldsymbol{u}) \mathbb{G}_n(\boldsymbol{u}^{(j)})$：Stute 表示中的线性逼近项，其中 $\dot{C}_j$ 为偏导数，$\boldsymbol{u}^{(j)}$ 仅第 $j$ 维为 $u_j$ 其余为 $1$。
- **关键假设**：
  - **弱光滑性条件**：底层 copula $C$ 的一阶偏导数 $\dot{C}_j$ 存在且满足局部有界/变差条件。统计学含义：放松了经典文献中 copula 密度存在或偏导连续的强光滑性要求，使得结论对非光滑 copula 族（如 Marshall-Olkin copula）仍成立。
  - **边际一致逼近**：对任意固定维数 $k$ 的边际子集 $S$，误差项在 $\boldsymbol{u}_S$ 上几乎必然一致有界，且关于 $S$ 一致。统计学含义：保证了高维下多重检验的联合误差可控。
- **问题背景**：经典经验 copula 过程的 Stute 表示（Stute, 1984; Segers, 2012）仅适用于固定维数 $d$，无法适应现代高维数据。近期高维独立性检验文献（如 Han et al., 2017）多假设变量间互不独立。本文突破了固定维数限制，将 Stute 表示推广至指数级高维，并放宽了独立性检验中的互不独立假设。

### 三、主要定理 / 核心结果

1. **高维 Stute 表示定理**
   - **原文陈述**：在弱光滑性下，对任意固定维数 $k$，存在 $\sup_{S: |S|=k} \sup_{\boldsymbol{u}_S \in [0,1]^k} |\mathbb{C}_{n,S}(\boldsymbol{u}_S) - \tilde{\mathbb{C}}_{n,S}(\boldsymbol{u}_S)| = o(1)$ 几乎必然成立，且 $\log d = o(n^\alpha)$。
   - **直观解释**：高维经验 copula 过程的波动，可完全由经典经验过程的波动加上 copula 偏导数的调整项线性表出。由于只关心低维边际（如两两），高维带来的复杂性被一致误差吸收。
   - **解决的技术难点**：克服了 $d \to \infty$ 时经验过程收敛域随维度爆炸的问题，通过将高维问题投影到低维边际并控制边际间的一致误差，绕过了维数灾难。
   - **适用条件与局限**：必须要求 copula 偏导数存在；结论主要针对低维边际的泛函，对全局高维泛函（如全空间 $\mathbb{R}^d$ 上的 sup-norm）可能失效。

2. **高维两两独立性检验与 Bootstrap FWER 控制**
   - **原文陈述**：基于两两关联度量（如 Spearman's rho）的检验统计量，在互不独立之外的更广模型类下控制 type-I error；Bootstrap 临界值对一大类 DGP 实现 FWER 的强控制。
   - **直观解释**：利用 Stute 表示，检验统计量的渐近分布只依赖于低维边际的极限高斯过程，Bootstrap 能够有效逼近这些低维边际分布，从而在多重检验中精确控制总体犯第一类错误的概率。
   - **解决的技术难点**：高维多重检验中变量依赖结构未知导致 FWER 控制困难，通过 Bootstrap 和边际一致逼近解决。

### 四、证明框架 / 方法设计

- **证明主干逻辑**：构造法 + 经验过程逼近 + 联合概率界。
- **拆解关键逻辑步骤**：
  1. **误差分解**：将 $\mathbb{C}_n - \tilde{\mathbb{C}}_n$ 分解为经验分布函数逼近误差和偏导数积分逼近误差。
  2. **低维边际投影**：利用经验过程的连续映射和边际投影，将高维空间上的逼近转化为对固定维数 $k$ 的边际子集 $S$ 上的逼近。
  3. **一致误差控制**：利用弱光滑性假设和 Borel-Cantelli 引理，结合 Bernstein/Hoeffding 型不等式，控制所有 $\binom{d}{k}$ 个边际上的联合误差上界。
  4. **Bootstrap 有效性**：证明 Bootstrap 经验过程在低维边际上弱收敛到真实的极限高斯过程，利用 Stute 表示传递到 Bootstrap copula 过程。
- **最关键的技巧性引理或"跳跃点"**：如何在 $\log d = o(n^\alpha)$ 的指数级增长下，控制 $\binom{d}{k}$ 个边际上的联合误差。技巧在于利用边际的弱依赖结构，通过 Union bound 配合指数不等式，使得 $d$ 的增长被 $\log d$ 的惩罚吸收，而不破坏 almost sure 收敛。
- **数学工具评价**：经典经验过程工具（Donsker 定理、连续映射）与高维概率不等式的精巧结合。并非全新分析框架，但在高维 copula 表示的推导中展现了极强的技术控制力。

### 五、与研究者兴趣的关联

- **连接到哪个子方向**：高维假设检验、非参数经验过程逼近、半参数效率理论的高维推广。
- **可借鉴的核心思路或技术工具**："低维边际一致逼近"的思想可直接迁移至高维因果推断中处理大量工具变量（IV）或中介变量时的渐近理论；Bootstrap FWER 控制框架可直接用于高维因果发现或多重中介分析中的假设检验；弱光滑性下的线性泛函逼近技巧对高维半参数效率界的推导有参考价值。
- **值得精读的关键参考文献**：
  1. Segers (2012, *Asymptotics of empirical copula processes under non-restrictive smoothness assumptions*)：本文的基础，固定维数下 Stute 表示的现代严格证明，理解弱光滑性假设的来源必读。
  2. Chernozhukov, Chetverikov, Kato (2013, *Gaussian approximations and multiplier bootstrap for maxima of sums of random vectors*)：高维经验过程逼近和 Bootstrap 控制的标杆文献，本文的高维控制技术与之呼应。

### 六、延伸思考与练习

- **假设扰动**：若将弱光滑性假设（偏导数存在且局部有界）放松为仅存在局部Hölder连续或完全不可导（如非参数copula），Stute 表示将失效，此时需要何种泛函（如 Hadamard 导数）或新的逼近工具（如经验似然）来刻画极限分布？
- **开放问题**：如何将此高维 Stute 表示应用于半参数 copula 模型（如两阶段估计）的效率界计算？在 $d$ 相对 $n$ 多项式增长甚至 $d > n$ 时，能否通过正则化得到类似的边际逼近？
- **理解检测题**：假设你要构造一个基于所有 $d$ 个变量的联合经验copula的检验统计量（而非两两边际），本文的 Stute 表示定理是否仍然足以保证其渐近分布的收敛？为什么？（提示：考虑 sup-norm 下的收敛域与维数 $d$ 的关系）。
