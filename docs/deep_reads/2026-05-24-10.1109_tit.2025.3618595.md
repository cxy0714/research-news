# Pearson Chi-Squared Conditional Randomization Test

**作者**: Adel Javanmard, Mohammad Mehrabi  
**来源**: IEEE Transactions on Information Theory  
**主题**: 数理统计 / 假设检验  
**相关性**: 8/10  
**链接**: <https://doi.org/10.1109/tit.2025.3618595>

### 一、核心问题与贡献
①研究了高维混杂变量$Z$下响应变量$Y$与变量$X$的条件独立性（CI）检验问题。②核心工具是结合条件相对密度与条件随机化构造的Pearson Chi-squared Conditional Randomization (PCR) 检验。③主要贡献在于：仅需极少随机化次数即可获得有限样本精确的高分辨率p值，给出了基于条件相对密度的显式功效分析，并提出了无需调参的parameter-free PCR与抵抗$P_{X|Z}$轻微误设的robust PCR。

### 二、基础设定
- **核心概念与符号**：
  - $(X, Z, Y)$：目标变量、高维混杂变量、响应变量。
  - $P_{X|Z}$：$X$给定$Z$的条件分布（假设已知或可精确采样）。
  - $f_{Y|X,Z}(y|x,z) / f_{Y|Z}(y|z)$：**条件相对密度**（conditional relative density），衡量备择假设偏离原假设的距离。
  - $M$：条件随机化次数。
  - $K$：Pearson卡方统计量的分箱数（bin数）。
- **关键假设**：
  - **已知$P_{X|Z}$**：CI检验的根本困难在于$P_{X|Z}$不可识别，本文在已知或能精确采样$P_{X|Z}$的设定下工作（这是CRT框架的标准假设，但现实中常需估计，robust PCR部分放松了此假设）。
  - **i.i.d. 样本**：$(X_i, Z_i, Y_i)_{i=1}^n$独立同分布。
  - **分箱划分**：将$Y$的支撑集划分为$K$个互斥的箱体，以构造Pearson卡方统计量。
- **问题背景**：
  - CI检验是因果发现与因果推断的基石，但Shah & Peters (2020)指出在没有$P_{X|Z}$先验时CI检验无相合检验，因此必须引入分布假设。
  - 传统CRT（Candès et al., 2018）依赖$M$次随机化计算p值，分辨率受限于$1/M$，且缺乏显式功效分析。本文通过Pearson卡方统计量将p值计算从离散随机化逼近转化为连续的$\chi^2$分布近似，解决了分辨率与功效刻画的双重痛点。

### 三、主要定理 / 核心结果

1. **原文陈述（PCR检验的水平与p值分辨率）**：
   在$H_0: X \perp\!\!\!\perp Y \mid Z$下，通过1次观测样本与$M$次随机化样本构造的Pearson卡方统计量，其p值可由$\chi^2_{K-1}$分布精确逼近。即使$M$很小（如$M=1$），p值也具有连续分布，突破$1/M$的离散分辨率限制。
2. **直观解释**：
   传统CRT将观测统计量与$M$个随机化统计量排序，p值本质是离散的秩统计量。PCR将观测与随机化样本的频数差异转化为卡方统计量，利用i.i.d.性质与多项分布渐近理论，使得p值直接由$\chi^2$分布的分位数决定，从而获得连续高分辨率。
3. **原文陈述（功效分析）**：
   在备择假设下，PCR检验的渐近功效由非中心$\chi^2$分布决定，其非中心参数$\lambda$正比于样本量$n$、随机化次数$M$，以及条件相对密度在分箱上积分后的距离平方和。
4. **直观解释**：
   功效不仅取决于信号强度（条件相对密度偏离1的程度），还与分箱策略$K$和随机化次数$M$直接交互。这为检验的参数选择提供了显式的理论指导。
5. **原文陈述（Robust PCR的水平控制）**：
   当使用估计量$\hat{P}_{X|Z}$代替真实$P_{X|Z}$进行随机化时，若估计误差以某种$O(\delta)$速率受控，Robust PCR通过修正卡方统计量的拒绝域，保证第一类错误膨胀不超过$O(\delta)$。
