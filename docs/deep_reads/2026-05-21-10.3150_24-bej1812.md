# Inference for change-plane regression

**作者**: Chaeryon Kang, Hunyong Cho, Rui Song, Moulinath Banerjee, Eric B. Laber et al.  
**来源**: Bernoulli  
**主题**: 非参数 / 半参数  
**相关性**: 8/10  
**链接**: <https://doi.org/10.3150/24-bej1812>

### 一、核心问题与贡献
①本文研究了change-plane回归中目标函数存在多个极小值点（非唯一性）时的估计与推断问题。②核心方法是构造了两种处理极小值非唯一性的估计量，并基于其极限分布发展了参数Bootstrap推断程序。③主要贡献是建立了两种估计量的$n$-rate收敛速度与非标准极限分布，从理论上证明了参数Bootstrap的有效性，解决了非规则设定下change-plane参数无法进行常规推断的难题。

### 二、基础设定
- **核心概念与符号**：
  - **Change-plane regression**：模型形式通常为 $Y = X^\top\beta_1 I(X^\top\psi \le 0) + X^\top\beta_2 I(X^\top\psi > 0) + \epsilon$，其中 $\psi$ 为决定亚组划分的超平面参数。
  - **Non-uniqueness of minimizers**：目标函数（如最小二乘）在参数空间存在一个"岭"（ridge）或多个极小值点，导致 $\hat{\psi}$ 不唯一。
  - **$n$-rate convergence**：$\hat{\psi}_n - \psi_0 = O_p(n^{-1})$，即超参数估计量以 $1/n$ 的超相合速度收敛。
- **关键假设**：
  - **跳跃大小非零/固定**：超平面两侧的回归函数存在固定的跳跃。统计学含义：保证 $\psi$ 具有足够强的信号，从而实现 $n$-rate 收敛；若跳跃趋于零，收敛率将退化为 $\sqrt{n}$-rate。
  - **极小值非唯一性设定**：目标函数在真实参数 $\psi_0$ 附近存在多个极小值。统计学含义：打破了传统 M-估计量要求目标函数具有唯一极小值的正则条件，导致 Argmax Continuous Mapping Theorem 无法直接使用。
  - **误差与协变量的平滑/矩条件**：保证局部经验过程的弱收敛。
- **问题背景**：传统 change-plane 推断要求目标函数具有唯一极小值点（即严格可分），但在精准医学等实际场景中，亚组边界可能存在重叠或模糊，导致极小值非唯一，传统渐近理论失效。与最相关文献（如 Lee et al., 2018; Seo & Liao, 2019）相比，本文不再回避或假设非唯一性消失，而是直接在非唯一极小值的设定下建立严格的理论。

### 三、主要定理 / 核心结果
1. **原文陈述**：针对两种估计量 $\hat{\psi}_1$ 和 $\hat{\psi}_2$，分别证明了 $\|\hat{\psi}_j - \psi_0\| = O_p(n^{-1})$，且 $n(\hat{\psi}_j - \psi_0)$ 弱收敛到某个非标准的极限分布（如复合泊松过程的极值点泛函）。参数 Bootstrap 估计量的条件分布依概率收敛到该极限分布。
2. **直观解释**：由于超平面两侧存在固定跳跃，参数定位极准（$1/n$ 速率）。但由于极小值不唯一，估计量会在真实超平面附近的"岭"上随机游走，其分布由局部残差符号翻转驱动的随机过程决定，呈现非正态、非对称的极端非标准形态。参数 Bootstrap 通过拟合模型重抽样，重构了这一局部随机过程。
3. **解决了什么技术难点**：克服了目标函数非凸且极小值非唯一导致的 Argmax 定理失效问题；解决了 $n$-rate 估计量非标准极限分布导致解析推断不可行的问题。
4. **适用条件与局限**：必须依赖跳跃大小固定且非零的假设；参数 Bootstrap 严重依赖误差项分布与协变量分布的正确设定（参数化模型），对模型误设敏感。

### 四、证明框架 / 方法设计
- **证明主干逻辑**：局部经验过程构造 + 随机游走/复合泊松过程逼近 + 特定选择规则下的 Argmax 收敛。
- **拆解关键逻辑步骤**：
  1. **局部目标函数展开**：在 $n(\psi - \psi_0)$ 的局部尺度下，将目标函数差分展开为经验过程。
  2. **极限过程逼近**：证明该局部经验过程弱收敛到一个带有漂移项的复合泊松过程或两侧布朗运动，漂移项由跳跃大小与协变量分布决定。
  3. **处理非唯一性**：由于极限过程有多个极小值，通过构造特定的选择规则（如最小范数规则或投影规则），定义唯一的估计量映射，绕过拓扑障碍。
  4. **Bootstrap条件弱收敛**：证明 Bootstrap 局部经验过程在给定样本的条件下，弱收敛到同一个极限随机过程，从而保证 Bootstrap 分布的一致性。
- **最关键的技巧性引理或"跳跃点"**：在极小值非唯一的情况下，如何将估计量与极限过程的极小值联系起来。传统方法依赖目标函数在真实值处的唯一性来保证 Argmax 的连续性。本文可能通过分析局部经验过程的"方向导数"符号变化，或构造一个辅助的严格凸目标函数来选择唯一的代表点，从而在非唯一解的集合中锚定估计量的渐近行为。
- **数学工具评价**：是经典非规则渐近理论（如 Kim & Pollard, 1990 的 cube-root asymptotics 变体）与经验过程理论的深度组合。难点不在于全新框架，而在于将 Argmax 定理适配到非唯一极小值的非标准拓扑结构中，并对参数 Bootstrap 进行严密的条件渐近验证。

### 五、与研究者兴趣的关联
- **连接子方向**：非规则半参数推断中的假设检验与极限分布理论。
- **可借鉴的核心思路或技术工具**：
  1. **处理非唯一极小值的估计量构造策略**：在因果推断中，若工具变量仅部分识别或处理效应存在异质性模糊边界，目标函数常出现平坦区域或多个极小值。本文的"选择规则+局部过程逼近"思路可直接迁移至部分识别集合的边界推断。
  2. **参数 Bootstrap 在非标准收敛率下的验证框架**：对于 $n$-rate 或 cube-root rate 估计量，非参数 Bootstrap 通常不一致，而参数 Bootstrap 的条件弱收敛证明框架可为其他半参数/非参数模型（如分位数处理效应）的 Bootstrap 推断提供技术模板。
- **值得精读的关键参考文献**：
  1. Kim, J. & Pollard, D. (1990). Cube root asymptotics. *Annals of Statistics*. —— 理解非标准收敛率与 Argmax 定理的基石文献，本文局部经验过程逼近的源头。
  2. Seo, M. H. & Liao, Y. (2019). Change-point inference in threshold regression models. *Econometrica*. —— 前序 change-plane/threshold 推断工作，对比本文如何突破唯一极小值假设。

### 六、延伸思考与练习
- **假设扰动**：若将"跳跃大小固定"假设修改为"局部平滑"（即跳跃大小随 $n$ 衰减，如 $\delta_n = c n^{-\alpha}, \alpha \in (0, 1/2)$），收敛率和极限分布会如何变化？技术上需要引入局部渐近正态性（LAN）或漂移项的缩放变换，极限分布可能从复合泊松过程过渡到布朗运动泛函。
- **开放问题**：如何在高维设定（$p \gg n$）下进行 change-plane 推断？稀疏惩罚项（如 L1）如何改变局部极小值的非唯一性结构与极限分布？
- **理解检测题**：假设目标函数在真实参数 $\psi_0$ 处的极限过程是一个具有平坦底部（即在某个区间上取相同极小值）的布朗运动泛函。请设计一种估计量选择规则（不同于本文），使得该估计量仍具有 $O_p(n^{-1})$ 的收敛率，并说明你的规则下极限分布的直观形态与本文可能的不同之处。
