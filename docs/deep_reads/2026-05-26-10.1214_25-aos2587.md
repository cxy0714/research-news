# Scalable inference for nonparametric stochastic approximation in reproducing kernel Hilbert spaces

**作者**: Meimei Liu, Zuofeng Shang, Yun Yang  
**来源**: Annals of Statistics  
**主题**: 非参数 / 半参数  
**相关性**: 8/10  
**链接**: <https://doi.org/10.1214/25-aos2587>

---

### 一、核心问题与贡献（3句话）
① 研究在再生核希尔伯特空间（RKHS）非参数最小二乘回归中，如何利用随机逼近（Stochastic Approximation, SA）实现**在线统计推断**（逐点与同时置信区间/带）。② 核心工具是将在线 multiplier bootstrap 应用于函数型随机梯度下降（functional SGD）估计量，并结合 sup-norm 下高阶展开与加权非独立同分布经验过程的高斯逼近理论。③ 主要贡献是建立了 functional SGD 估计量的非渐近行为完整刻画，证明了 bootstrap 的一致性，并揭示了 SGD 步长调参（step-size tuning）与推断精度之间存在非平凡权衡，为大规模/流式非参数推断提供了理论保障。

### 二、基础设定
#### 核心概念与符号
- **RKHS**：再生核希尔伯特空间 $\mathcal{H}$，具有正定核 $K$，内积 $\langle \cdot, \cdot \rangle_{\mathcal{H}}$，再生性质 $f(x) = \langle f, K(x, \cdot) \rangle_{\mathcal{H}}$。
- **非参数最小二乘**：目标函数 $L(f) = \mathbb{E}[(Y - f(X))^2]$，最优 $f^* = \arg\min_{f \in \mathcal{H}} L(f)$（假设 $\mathcal{H}$ 足够大可近似）。
- **functional SGD**：迭代 $f_{t+1} = f_t - \eta_t \nabla L_t(f_t)$，其中 $\nabla L_t(f_t)$ 是基于第 $t$ 个样本 $(X_t,Y_t)$ 的随机梯度（即 $2(f_t(X_t) - Y_t)K(X_t, \cdot)$），步长序列 $\{\eta_t\}$。
- **在线 multiplier bootstrap**：对每个 $t$，产生 bootstrap 拷贝 $\tilde f_t^{*(b)}$ 以近似 $f_t - f^*$ 的分布，用于构造置信区间。
- **sup-norm 距**：$\|f\|_\infty = \sup_{x \in \mathcal{X}} |f(x)|$。
- **经验过程**：$\mathbb{G}_n(f) = n^{-1/2}\sum_{i=1}^n (g_i - \mathbb{E}[g_i])$，本文处理加权且不同分布的情形。

#### 关键假设
- **A1（步长条件）**：步长 $\eta_t = \gamma t^{-\alpha}$，$\alpha \in (1/2, 1]$，$\gamma > 0$。保证 SGD 收敛且方差可控。
- **A2（核与空间条件）**：核 $K$ 有界连续且特征值衰减足够快（如多项式或指数衰减），确保 $\mathcal{H}$ 在 sup-norm 下紧嵌入，且 $f^*$ 在 $\mathcal{H}$ 内。
- **A3（噪声条件）**：残差 $Y - f^*(X)$ 满足次高斯或基于矩的条件，确保经验过程集中性。
- **A4（Bootstrap 权重的矩条件）**：bootstrap 权重独立于数据，具有单位方差和有限矩，保证 multiplier bootstrap 的有效性。
- **与已有文献比较**：放宽了经典 kernel ridge regression 需批处理的假设，适应流式设定；相比于在线梯度下降的 O(1/t) 收敛率文献，本文额外处理了在 sup-norm 下的推断（而非仅 $L_2$ 范数）。

#### 问题背景
- **已有方法不足**：离线非参数推断（如基于核岭回归的高斯过程）需存储全部数据，且计算复杂度随样本量立方增长；在线 SGD 仅有收敛率结果，缺乏有效的在线不确定性量化工具。
- **最相关文献**：
  1. **Shang & Cheng (2015)**：对核岭回归建立 sup-norm 渐近理论，但仅离线。本文将其扩展至在线 SGD，并引入 bootstrap。
  2. **Rakhlin et al. (2012)**：证明在线 SGD 在 $L_2$ 下的收敛率，但未讨论推断。本文补充了同时置信带。
  3. **Chernozhukov et al. (2013)**：广义高斯逼近用于 bootstrap 一致性，但其设定为独立同分布。本文需推广至加权非独立同分布经验过程。

### 三、主要定理 / 核心结果
#### 定理 1：functional SGD 的 sup-norm 非渐近界
- **陈述**：在假设 A1–A3 下，存在常数 $C$，使得对任意 $t$，  
  $$
  \mathbb{P}\left( \|f_t - f^*\|_\infty \geq C \left(\sqrt{\frac{\log t}{t^{1-\alpha}}} + \lambda_t\right) \right) \leq t^{-c},
  $$
  其中 $\lambda_t$ 是核近似误差（若 $f^*$ 在 $\mathcal{H}$ 外则为正则化偏误），$\alpha$ 为步长衰减指数。
- **直观解释**：SGD 估计量的 sup-norm 误差以高概率被控制，由两项之和构成：方差项（来自随机梯度噪声的累积）和偏差项（来自迭代未无穷）。步长 $\alpha$ 决定了方差与偏差的权衡。
- **技术难点**：通常 $L_2$ 收敛分析依赖 Hilbert 空间结构，但 sup-norm 需要处理 RKHS 嵌入到连续函数空间时的局部复杂性，本文借助 RKHS 的局部 Rademacher 复杂度和概率不等式链。
- **适用条件与局限**：需要核函数有界且特征值衰减条件；假设 $\alpha > 1/2$ 以保证方差可控；若 $\alpha=1$（最快速步长），偏差项为 $O(\sqrt{\log t})$，但方差项为 $O(\sqrt{1/t})$，整体为 $O(\sqrt{\log t} / \sqrt{t})$，接近离线极小极大率（在超平滑核下）。

#### 定理 2：multiplier bootstrap 一致性（逐点置信区间）
- **陈述**：在假设 A1–A4 下，对任意固定 $x \in \mathcal{X}$，  
  $$
  \sup_{z} \left| \mathbb{P}\left( \sqrt{t}(f_t(x) - f^*(x)) \leq z \right) - \mathbb{P}\left( \sqrt{t}\tilde f_t^{(*)}(x) \leq z \right) \right| \to 0,
  $$
  其中 $\tilde f_t^{(*)}(x)$ 为 bootstrap 版本，且 $t \to \infty$。
- **直观解释**：bootstrap 分布一致逼近真实 SGD 估计量的采样分布，使得我们可以用 bootstrap 分位数构造置信区间，无需显式估计渐近方差。
- **技术难点**：普通 multiplier bootstrap 通常要求估计量为 M-估计量或具有线性影响函数，但 SGD 动态的非线性叠加使影响函数复杂。本文通过高阶展开将 $f_t - f^*$ 写成主导线性项+余项，证明余项在 sup-norm 下充分小，并将线性项的经验过程加权项匹配到 bootstrap。
- **适用条件与局限**：需 bootstrap 权重具有足够矩；步长 $\alpha$ 需使方差项的主导统计量经缩放收敛到正态分布（即中心极限定理适用）；对于“小步长”（$\alpha$ 接近 1/2），方差项主导但收敛慢，bootstrap 性能可能下降。

#### 定理 3：bootstrap 一致性（同时置信带）
- **陈述**：对任意紧子集 $\mathcal{X}_0 \subseteq \mathcal{X}$，  
  $$
  \sup_{t} \left| \mathbb{P}\left( \| \sqrt{t}(f_t - f^*)\|_\infty \leq q \right) - \mathbb{P}\left( \| \sqrt{t}\tilde f_t^{(*)}\|_\infty \leq q \right) \right| \to 0,
  $$
  其中 $q$ 取自 bootstrap 分位数，用于构造渐近准确的同时置信带。
- **直观解释**：同时置信带的覆盖概率可由 bootstrap 校准，无需解析计算 sup-norm 的渐近分布（通常为依赖于核的复杂极值分布）。
- **技术难点**：需要证明 sup-norm 下经验过程的 bootstrap 一致性，这要求对加权过程进行高斯逼近（Chernozhukov et al. 2013 型），但本文的加权过程是非独立同分布的重抽样，需调整方差标准化。本文通过构造一个辅助高斯过程，并利用 Slepian 引理和反演不等式比较。
- **适用条件与局限**：需要 $\mathcal{X}_0$ 上核函数的局部 Lipschitz 性质，确保经验过程一致椭球类满足熵条件；若 $\mathcal{X}$ 维数高，sup-norm 带宽的收敛速度会变慢，但仍可用 bootstrap 校准，只是区间较宽。

### 四、证明框架 / 方法设计
**证明主干逻辑**：
1. **SGD 的高阶展开**：将 $f_t - f^*$ 分解为 $L_t + R_t$，其中 $L_t$ 是线性部分（由加性噪声项的加权和构成），$R_t$ 是余项（包含动力学中的非线性残差）。
2. **线性部分的经验过程控制**：证明 $\sqrt{t} L_t$ 在 sup-norm 下收敛到高斯过程，利用加权（非独立同分布）经验过程的极大值高斯逼近定理（推广 Chernozhukov et al. 2014）。
3. **余项的高阶控制**：利用 RKHS 的局部性质（如插值不等式、局部 Rademacher 复杂度）证明 $R_t$ 在 sup-norm 下以高概率为 $O_p(t^{-(1+\delta)})$ 量级，小于主导项的标准差率 $O_p(t^{-1/2})$。
4. **Bootstrap 的克隆**：对 bootstrap 样本 $\{(X_t, Y_t^{*(b)})\}$ 重新应用同一展开，得到 $\tilde f_t^{*(b)} - \hat f_t$ 的线性部分与真实线性部分结构相同（仅权重不同）。再利用 bootstrap 权重的矩条件，证明 bootstrap 版本的线性部分分布近似真实线性部分。
5. **Gaussian 逼近的比较**：通过 Slepian 引理和 Kolmogorov 距离的反演，将真实统计量 $\sqrt{t}L_t$ 的分布与 bootstrap 版本 $\sqrt{t}\tilde L_t^{*(b)}$ 的分布比较，最终得到定理 2 和 3 的结论。

**关键技巧性引理**：
- **“高阶展开引理”**（引理 5.2）：给出 $f_t - f^* = \sum_{k=1}^m \Phi_{t,k} + R_{t,m}$，其中 $\Phi_{t,k}$ 是 $k$ 阶张量型项（涉及多个样本的乘积）。这是证明中最关键的跳跃点，因为通常 SGD 的线性化影响函数只保留一阶，但此处需要控制高阶项以确保 sup-norm 的余项可忽略。该引理使用迭代展开和 RKHS 的有界核性质。
- **“加权经验过程极大值高斯逼近引理”**（引理 6.1）：将 Chernozhukov et al. 的定理推广到加权且非独立同分布情形，利用核函数的有界性和递减权重的均匀性。

**数学工具评价**：经典工具的巧妙组合——在线 SGD 的分析与离线核岭回归的 sup-norm 理论结合，再加上经验过程高斯逼近的最新结果。并非全新框架，但首次将这三条线合并用于实时推断。

### 五、问题发现：研究者能做什么
**(A) 立即可做**（最多 2 条）
1. **检查 SGD 步长-精度权衡是否达到 minimax 最优**  
   - 问题：验证定理 1 中误差率 $O(\sqrt{\log t}/t^{(1-\alpha)/2})$ 是否为给定 RKHS 下非参数函数在 sup-norm 意义下的 minimax 最优（$L_2$ 最优已知，但 sup-norm 最优率可能不同）。  
   - 武器库：**minimax bounds for estimation problems**（very familiar）  
   - 第一步动作：对于给定核（如多项式衰减 $\mu_j \asymp j^{-2\beta}$ 的索伯列夫平滑），推导该 RKHS 上 sup-norm minimax 下界（利用 Assouad lemma 或 Fano 不等式），与本文上界比较。若上界匹配且常数定性合理，则说明定理 1 是紧的；若不匹配（例如在超平滑核下 $\alpha=1$ 时达到 $\sqrt{\log t/t}$ 但 minimax 可能是 $(\log t/t)^{1/2}$），则表明 SGD 在 sup-norm 下不是最优，需调整步长或核选择。  
   - 与本文的关系：补全本文理论的紧性验证，直接回答“步长-精度权衡”是否已是问题下界。

2. **将 bootstrap 方法扩展到截断/正则化 SGD**  
   - 问题：本文假设 $f_t$ 在 $\mathcal{H}$ 内无正则化，但在高维/弱正则核下 SGD 会过拟合。可否在每一步加入截断（如强制 $\|f_t\|_{\mathcal{H}}\leq B$）并保持 bootstrap 一致性？需要推导截断后的高阶展开变化。  
   - 武器库：**estimation theory in causal inference** 中的经验风险最小化 + **高维渐近**（处理弱正则下的 bias-variance 校准）  
   - 第一步动作：选取一个具体核（如高斯核，其有效自由度增长慢），模拟截断 SGD 的轨迹，数值比较 bootstrap 区间覆盖与未截断版本的差异。理论方面，先写出截断后偏差项的表达式，再观察截断是否改变线性展开的主导项结构。  
   - 与本文的关系：扩展本文方法到更实际的大规模设定，是算法侧贡献。

**(B) 中期可做**（最多 2 条）
1. **识别本文高阶展开与高阶影响函数（HOIF）的潜在联系**  
   - 缺哪一块：**HOIF（Higher-Order Influence Functions）** 中的高阶 bias 校正表达式，特别是当目标为 $\ell_\infty$ 范数时如何将余项 $R_{t,m}$ 解释为高阶 Delta 方法项。  
   - 补哪 1-2 篇文献：  
     - **Robins et al. (2008)** “Higher-order influence functions and minimax estimation” —— 建立 HOIF 的框架。  
     - **Kennedy et al. (2017)** “Nonparametric causal inference with spotlight” 的附录给出高阶展开在实践中如何实现。  
   - 补完后能做什么：使用 HOIF 语言重新表述本文的 SGD 展开，可能得到一个更统一的 bias-variance 分解，甚至能构造 $\sqrt{t}(f_t-f^*)$ 的 Edgeworth 展开，从而 bootstrap 校准可以更精确（理论 2 阶改进）。这对应 A 档中“改进 bootstrap 精度”的具体问题：是否存在一个基于 HOIF 的调整 bootstrap 方法能减少覆盖误差从 $O(t^{-1/2})$ 到 $O(t^{-1})$？  

2. **建立 functional SGD 与 M-estimation 的 bridge**，以利用半参数 Bootstrap 工具箱  
   - 缺哪一块：**M-estimation theory** 中针对递归估计的渐近线性表示（如“stochastic M-estimator”的 influence function 表达式）—— 经典 M-estimation 通常假定目标为固定的 $f^*$ 且迭代次数有限，但本文是无限迭代且 $f_t$ 实时更新。  
   - 补哪 1-2 篇文献：  
     - **Bercu & Touboul (2024)** “Statistical inference for stochastic algorithms” —— 提供线性化 SGD 的弱收敛结果。  
     - **Polyak & Juditsky (1992)** —— 平均化 SGD 的渐近正态性（是经典的，但本文为在线推断未使用平均化）。  
   - 补完后能做什么：将本文的 bootstrap 一致性推广到替代的 SGD 变体（如 Polyak-Ruppert 平均化、ASGD），并直接套用半参数 bootstrap 理论（如 wellner 的 bootstrap for Z-estimators），可能简化证明并扩展至更一般的损失函数（如 logistic 回归）。

**(C) 暂不建议**（最多 2 条）
1. **将 sup-norm 下的局部 Rademacher 复杂度用于张量网络成本分析**  
   - 缺什么机器：**函数空间的 fine metric entropy 估计与局部 Gaussian 过程的乘法链**（如 van der Vaart & Wellner 的 entropy integral 技巧）—— 虽然研究者熟悉非参数统计，但 sup-norm 下的高阶展开需要非常精细的 chaining 来得到余项的高概率界，且本文使用了特定的 RKHS 插值不等式（Reproducing property + bound on the kernel）。这些工具在武器库内，但构建类似框架用于张量收缩的随机梯度分析目前几乎是空白，且不直接与高阶 U-statistics 计算成本模型相关。  
   - 为何不易绕过：要模拟 sup-norm SGD 中的展开引理应用于 tensor-network 场景，需要将 RKHS 的再生性质替换为 tensor 图的泛函分析（如 CP 或 TT 空间的 Lipschitz 性质），目前缺乏成熟的概率不等式。强行做可能产生巨大技术壁垒，且偏离研究主线。  
   - 若全部在武器库内则写“无”，此处写“无”除非确实认为该方向有风险。但本文的 RKHS 方法不能直接迁移到张量结构，因此暂不建议。

**值得精读的关键参考文献**：
1. **Shang & Cheng (2015)** “Nonparametric regression with Gaussian processes: sup-norm convergence rates and efficient estimation” —— 直接对应本文定理 1 的离线版本，阅读它可以厘清在线与离线分析的差异，为 A 档问题 1 提供下界参考。
2. **Chernozhukov et al. (2014)** “A central limit theorem for the supremum of a class of empirical processes” —— 提供本文引理 6.1 的 base，必读以理解 bootstrap 一致性的 Gaussian 逼近技术。
3. **Rakhlin et al. (2012)** “Making gradient descent theoretically optimal” —— 阅读 SGD $L_2$ 收敛率的经典证明，便于理解本文 sup-norm 扩展的难度和使用的局部复杂度的必要性。

### 六、延伸思考与练习
