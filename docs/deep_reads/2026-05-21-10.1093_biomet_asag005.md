# On the consistency of bootstrap for matching estimators

**作者**: Ziming Lin, Fang Han  
**来源**: Biometrika  
**主题**: 因果推断  
**相关性**: 8/10  
**链接**: <https://doi.org/10.1093/biomet/asag005>

---

### 一、核心问题与贡献
①本文研究了最近邻匹配估计量在估计平均因果效应（ATE）时，naive bootstrap 一致性是否成立的问题。②核心方法是通过允许匹配数 $M$ 随样本量发散，并利用经验过程理论与渐近展开技术重新审视匹配估计量的分布逼近。③严格证明了当 $M \to \infty$ 时 naive bootstrap 具有一致性，从而将 Abadie & Imbens (2008) 发现的 bootstrap 不一致性归因为固定 $M$ 的局限，而非匹配估计量本身的固有缺陷。

### 二、基础设定
- **核心概念与符号**：
  - $\tau$：平均因果效应（ATE），$\tau = E[Y(1)-Y(0)]$。
  - $M$：最近邻匹配数（固定或发散）。
  - $\hat{\tau}_M$：基于 $M$ 个最近邻的匹配估计量。
  - $J_M(x)$：匹配指示权重函数，表示样本点 $x$ 被用作最近邻的次数。
  - $\mu_d(x), \sigma^2_d(x)$：潜在结果在协变量给定下的条件均值与条件方差（$d \in \{0,1\}$）。
- **关键假设**：
  - **Unconfoundedness & Overlap**：标准因果推断假设，保证可识别性与匹配的可行性。
  - **$M \to \infty$ 且 $M = o(f(n))$**：匹配数发散，但发散速率受协变量维度 $d$ 与样本量 $n$ 约束（如 $M = o(n^{2/(d+2)})$），这是本文推翻 AI08 结论的核心设定。
  - **光滑性条件**：$\mu_d(x)$ 与 $\sigma^2_d(x)$ 满足特定阶数的 Lipschitz/Hölder 连续性，用于控制匹配偏差。
  - **协变量分布支撑与密度下界**：保证最近邻距离以合适速率收敛。
  - *假设对比*：AI08 假设 $M$ 固定，导致条件方差估计呈现非平滑的阶梯函数性质；本文放宽至 $M \to \infty$，平滑了这种非平滑性，使标准经验过程理论重新适用。
- **问题背景**：AI08 证明了固定 $M$ 下 naive bootstrap 失效，导致后续研究只能转向 $m$-out-of-$n$ bootstrap 或 subsampling 等次优方法。本文指出，在 $M \to \infty$ 这一更符合实际大样本推断的设定下，naive bootstrap 天然有效，无需引入额外复杂的重抽样方案。最相关文献区别：Abadie & Imbens (2006) 建立了匹配估计量的渐近正态性；Abadie & Imbens (2008) 证明了固定 $M$ 下 bootstrap 失效；本文在发散 $M$ 下统一了渐近理论与 bootstrap 推断。

### 三、主要定理 / 核心结果
1. **原文陈述**：在正则条件与 $M \to \infty$ 满足特定速率的前提下，$\sup_{t} \left| P^*\left( \sqrt{n}(\hat{\tau}_M^* - \hat{\tau}_M) \le t \right) - P\left( \sqrt{n}(\hat{\tau}_M - \tau) \le t \right) \right| \xrightarrow{p} 0$，其中 $P^*$ 为给定原样本下的 bootstrap 条件概率。
2. **直观解释**：当 $M$ 固定时，匹配集合的指示函数 $J_M(x)$ 是高度非平滑的阶梯函数，bootstrap 重抽样造成的微小协变量扰动会导致匹配集合的“跳跃”，使得 bootstrap 分布无法捕捉真实的条件方差变异；当 $M \to \infty$ 时，匹配估计量在局部进行了大量平均，等价于一种非参数核回归，阶梯函数被平滑化，bootstrap 经验过程能够正确模仿原始样本的随机波动。
3. **解决了什么技术难点**：克服了匹配指示函数不连续导致的 bootstrap 经验过程不依概率弱收敛的问题，严格量化了 $M$ 发散如何消除条件方差估计中的非平滑项。
4. **适用条件与局限**：必须要求 $M \to \infty$；对协变量维度 $d$ 敏感（维数灾难），若 $d$ 较大，$M$ 的允许发散速率极慢，条件可能难以满足；要求条件方差函数连续，若方差函数本身存在跳跃，结论可能受影响。

### 四、证明框架 / 方法设计
- **证明主干逻辑**：渐近线性展开 + 经验过程控制。
- **拆解为关键逻辑步骤**：
  1. **Hájek投影与线性化展开**：将匹配估计量 $\sqrt{n}(\hat{\tau}_M - \tau)$ 分解为条件均值偏差项、条件方差项与高阶余项，证明在 $M \to \infty$ 下余项 $o_p(1)$。
  2. **条件方差项的平滑化分析**：证明当 $M \to \infty$ 时，匹配产生的条件方差估计 $\hat{\sigma}^2_d(X_i)$ 在 $L_2$ 范数下收敛于真实的 $\sigma^2_d(X_i)$，消除了固定 $M$ 下的阶梯跳跃变异。
  3. **Bootstrap经验过程逼近**：构造基于 bootstrap 权重的经验过程，证明其依概率弱收敛到与原样本经验过程相同的零均值高斯过程极限。
  4. **分布函数耦合**：利用三角不等式与连续映射定理，整合上述步骤，证明 bootstrap 分布函数与真实分布函数的极大偏差依概率趋于 0。
- **最关键的技巧性引理或"跳跃点"**：处理匹配权重 $J_{M,i}$ 在 bootstrap 样本中的波动。固定 $M$ 时，$J_{M,i}$ 的变异性是导致 bootstrap 失效的毒药；本文通过计算 $M \to \infty$ 时 $J_{M,i}$ 的二阶矩，证明其变异对总方差的贡献被 $1/M$ 速率稀释，从而将非平滑的离散匹配转化为平滑的连续权重，这是整个证明的“跳跃点”。
- **数学工具评价**：经典经验过程与渐近展开的精妙组合。并非全新分析框架，但对匹配估计量非平滑性的处理极具技术洞察力，将 $M \to \infty$ 的平滑效应严格量化为矩收敛条件。

### 五、与研究者兴趣的关联
- **连接到哪个子方向**：因果推断中的匹配与半参数估计渐近理论、Bootstrap推断。
- **可借鉴的核心思路或技术工具**："发散阶数平滑化非光滑估计量"的思想。在您研究的 higher-order U-statistics 或高维 debiased 估计中，固定阶数/步长往往导致分布逼近的非正态性或 bootstrap 失效，允许阶数随样本量适度发散可恢复标准渐近理论；此外，匹配估计量在 $M \to \infty$ 下等价于非参数回归的渐近等价性分析技巧，可直接迁移至您对 semiparametric efficiency bound 的推导中。
- **值得精读的关键参考文献**：
  1. Abadie & Imbens (2008, *Annals of Statistics*)：理解 bootstrap 不一致根源的必读文献，与本文形成直接镜像对比。
  2. Abadie & Imbens (2006, *Econometrica*)：匹配估计量渐近正态性与方差估计的奠基之作，其展开框架是本文的技术起点。

### 六、延伸思考与练习
- **假设扰动**：若协变量维度 $d$ 随样本量 $n$ 发散（高维设定），$M \to \infty$ 的条件将如何受影响？技术上可能需要引入稀疏性假设或高维最近邻收敛速率的新工具（如 RMT 中的谱方法或 sparse PCA），此时 bootstrap 方差矩阵的收敛可能需要借助近似消息传递（AMP）理论来刻画。
- **开放问题**：在 $M$ 固定但极小（如 $M=1$）的极端非参数设定下，是否存在某种修正的 prepivoting 或平滑 bootstrap 方法能恢复一致性？或者，如何将此发散 $M$ 的思想推广到带有倾向得分调整的增强匹配估计量？
- **理解检测题**：假设我们使用 $m$-out-of-$n$ bootstrap（子抽样），在 $M$ 固定的情况下，为什么它能够恢复一致性？请从经验过程的角度，对比 $m$-out-of-$n$ bootstrap 与 naive bootstrap 在处理匹配指示函数 $J_M(x)$ 不连续性时的差异机制。
