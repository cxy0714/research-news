# Causal mediation analysis: selection with asymptotically valid inference

**作者**: Jeremiah Jones, Ashkan Ertefaie, Robert L Strawderman  
**来源**: Journal of the Royal Statistical Society Series B  
**主题**: 因果推断  
**相关性**: 10/10  
**链接**: <https://doi.org/10.1093/jrsssb/qkae109>

---

### 一、核心问题与贡献
①研究了高维因果中介分析中，在混淆函数形式未知且存在中介选择时，如何对自然直接/间接效应（NDE/NIE）进行变量选择与有效推断的问题。②核心方法是将混淆函数视为 nuisance 参数并进行数据自适应估计，随后在基于有效影响函数的目标函数上施加新型正则化以选择重要中介，结合扰动重抽样解决选择后推断。③主要贡献是建立了估计量的 oracle 性质与局部渐近理论，并提供了高维中介设定下渐近有效的 post-selection 推断框架。

### 二、基础设定
- **核心概念与符号**：
  - $A$: 二值处理；$Y$: 结果；$M = (M_1, \dots, M_p)$: 高维中介向量；$W$: 基线混淆变量。
  - NDE (Natural Direct Effect): $\mathbb{E}[Y(1, M(0))] - \mathbb{E}[Y(0, M(0))]$。
  - NIE (Natural Indirect Effect): $\mathbb{E}[Y(1, M(1))] - \mathbb{E}[Y(1, M(0))]$。
  - 混淆函数: $Q_a(W) = \mathbb{E}[Y \mid A=a, W]$ 与 $G_a(W) = P(A=a \mid W)$ 等。
- **关键假设**：
  1. **Sequential Ignorability**: 给定 $W$，$(A, M)$ 与 $Y$ 的潜在结果独立，且 $A$ 与 $M$ 的潜在结果独立。*统计学含义*：NDE/NIE 的非参数识别条件。*对比*：本文维持了标准的识别假设，但大幅放宽了已有惩罚中介分析中对 $Q_a, G_a$ 的参数化/线性假设。
  2. **Sparsity of Mediation Effects**: 仅存在 $s \ll p$ 个非零的中介效应。*统计学含义*：高维参数空间的正则化可行性。*对比*：已有方法常假设整个回归模型稀疏，本文仅要求因果中介效应稀疏。
  3. **Nuisance 估计收敛速度**: $Q_a, G_a$ 的数据自适应估计需达到 $o(n^{-1/4})$ 收敛速度。*统计学含义*：保证 nuisance 估计误差不破坏目标参数的 $\sqrt{n}$ 渐近正态性。*对比*：比参数化假设弱，但比 DML 中的交叉拟合条件更强（未要求 sample splitting）。
- **问题背景**：现有惩罚中介分析方法（如 Zhao & Luo 2016）要么假设有限维线性模型足以消除混淆偏差，要么完全不控制混淆，这在实际中极易导致虚假发现与偏差。本文与最相关文献的区别：①相较于 Zhang et al. (2016) 的高维中介方法，本文不依赖结果的线性混淆模型；②相较于传统的 adaptive lasso，本文在局部渐近下证明了选择一致性，避免了传统方法在局部参数下的选择不一致。

### 三、主要定理 / 核心结果
1. **Oracle Property (定理主文)**：
   - *原文陈述*：在合适的正则化参数 $\lambda_n$ 下，惩罚估计量 $\hat{\beta}$ 依概率趋于 1 正确选择非零中介效应集合，且非零分量的渐近分布与已知真实子集的 oracle 估计量相同。
   - *直观解释*：估计量不仅能准确剔除无关中介（无假阳性），而且保留的中介效应估计不受选择过程的影响，具有半参数有效性。
   - *技术难点*：控制数据自适应 nuisance 估计的余项，证明其不破坏惩罚估计量的极值渐近行为。
   - *局限*：依赖 Donsker 类条件或 nuisance 的 $o(n^{-1/4})$ 速度，若混淆维度极高导致速度不达标，oracle 性质可能失效。
2. **Local Asymptotic Contrast (局部渐近对比定理)**：
   - *原文陈述*：在局部参数 $n^{-1/2}$ 邻域内，标准 adaptive lasso 无法保持选择一致性，而本文提出的基于影响函数的正则化方法可以。
   - *直观解释*：当真实中介效应极小（接近 $\sqrt{n}$ 级别）时，传统方法极易漏选或错选，本文方法通过直接对影响函数施加惩罚，在信号微弱处仍保持正确的收缩行为。
   - *技术难点*：在局部备择假设下推导惩罚 Z-估计量的联合极限分布。
   - *局限*：局部设定仅针对 $n^{-1/2}$ 速率，对更慢速率的微弱信号未做探讨。
3. **Perturbation Bootstrap Validity (扰动重抽样定理)**：
   - *原文陈述*：基于扰动权重 $\xi_i$ 的重抽样估计量 $\hat{\beta}^*$，在条件于数据下，依概率收敛于 oracle 估计量的真实渐近分布。
   - *直观解释*：通过给每个样本的影响函数乘上随机扰动权重，模拟样本变异性，从而绕过了解析推导 post-selection 分布的极高复杂度。
   - *技术难点*：证明重抽样过程中的选择事件与原样本选择事件以高概率一致。

### 四、证明框架 / 方法设计
- **证明主干逻辑**：基于 Z-估计量的经验过程理论与惩罚目标函数的凸分析。
- **拆解关键逻辑步骤**：
  1. **影响函数分解**：将目标参数的估计方程分解为真实影响函数、nuisance 估计误差项与二阶余项，证明二阶余项为 $o_p(n^{-1/2})$。
  2. **局部二次逼近**：在真实参数邻域内，将惩罚目标函数展开为二次项 + 线性项 + 惩罚项，利用经验过程理论控制一致收敛。
  3. **选择一致性**：利用反证法与 KKT 条件，证明在零假设（中介效应为 0）方向上，惩罚项的梯度足以将估计值收缩至 0。
  4. **渐近正态性**：在非零分量子空间上，惩罚项渐近消失，退化为标准的半参数 Z-估计量极限分布。
  5. **Bootstrap 条件分布**：将扰动权重视为条件独立的外生变量，证明条件 Z-估计量的次梯度分布逼近无条件极限分布。
- **最关键的技巧性引理/跳跃点**：**直接对有效影响函数构造惩罚估计方程**，而非对损失函数求导。传统 adaptive lasso 基于似然或平方损失，其梯度在局部参数下有偏；本文直接对无偏的影响函数施加 adaptive lasso 惩罚，使得在 $n^{-1/2}$ 局部域内，估计方程的线性项主导了惩罚项的偏差，这是实现局部选择一致性的核心跳跃。
- **数学工具评价**：是经典半参数理论、Z-估计量经验过程与 adaptive lasso 理论的精巧组合，并非全新分析框架，但在目标函数构造上具有范式创新。

### 五、与研究者兴趣的关联
- **连接子方向**：高维因果中介分析的半参数推断。
- **可借鉴的核心思路**：
  1. **基于 EIF 的正则化构造**：在处理高维因果参数时，不局限于传统的损失函数正则化，而是直接对有效影响函数/得分函数施加 $L_1$ 惩罚。这一思路可迁移至高维 IV 或 proximal CI 设定中，用于选择有效的工具变量或负控制变量。
  2. **Perturbation Bootstrap 替代 Selective Inference**：在 post-selection 推断中，避免了繁琐的截断正态分布计算，通过扰动重抽样直接逼近条件分布，对高维假设检验的统计计算具有极高实用价值。
- **值得精读的关键参考文献**：
  1. *Zhao & Luo (2016, Biometrics)*: 惩罚中介分析的早期代表作，对比阅读可深刻理解为何参数化混淆模型假设会限制推断的有效性。
  2. *Minnier, Tian & Cai (2011, JASA)*: Perturbation bootstrap 在惩罚估计方程中的理论基础，是本文推断部分的核心技术来源。
  3. *Zheng & van der Laan (2010, IJBI)*: 目标半参数最大似然估计（TMLE）框架，本文 nuisance 估计与影响函数分解的理论基石。

### 六、延伸思考与练习
- **假设扰动**：若放宽 nuisance 估计 $o(n^{-1/4})$ 的速度要求（例如极高维 $W$ 导致只能达到 $o(n^{-1/6})$），结论会如何变化？技术上需要引入 **Cross-fitting (DML)** 机制来消除 Donsker 类限制，并可能需要修改影响函数的构造以获得更高阶的鲁棒性。
- **开放问题**：如何将此框架扩展至纵向因果中介分析？此时时变混淆会引入多阶段的 nuisance 参数，其影响函数的嵌套结构将使正则化目标函数的构造与余项控制变得极为复杂。
- **理解检测题**：假设结果 $Y$ 是右删失生存数据，请基于本文思路， sketch 出如何将删失机制（如生存函数 $\hat{S}_C$）作为 nuisance 参数纳入影响函数，并构造针对高维中介效应的惩罚估计方程。提示：考虑如何将 IPCW 权重融入影响函数的线性项。
