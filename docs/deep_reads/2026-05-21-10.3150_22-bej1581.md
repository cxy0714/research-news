# Hypothesis testing for equality of latent positions in random graphs

**作者**: Xinjie Du, Minh Tang  
**来源**: Bernoulli  
**主题**: 数理统计 / 假设检验  
**相关性**: 8/10  
**链接**: <https://doi.org/10.3150/22-bej1581>

---

### 一、核心问题与贡献
①本文研究了广义随机点积图（GRDPG）中两个节点潜在位置是否相等（或仅差缩放因子）的假设检验问题。②核心工具是基于邻接矩阵或归一化拉普拉斯矩阵谱嵌入行向量的经验马氏距离。③证明了该检验统计量在原假设与局部备择假设下均渐近服从卡方分布，并给出了非中心化参数的显式表达式，进而解决了SBM与DCSBM、ER与SBM之间的模型选择问题。

### 二、基础设定
- **核心概念与符号**：
  - $A \in \{0,1\}^{n \times n}$：邻接矩阵；$P = \mathbb{E}[A]$：概率矩阵。
  - GRDPG($p,q$)：$P = U S U^\top$，潜在位置 $X_i \in \mathbb{R}^d$（$d=p+q$），满足 $P(A_{ij}=1|X_i,X_j) = X_i^\top I_{p,q} X_j$，其中 $I_{p,q} = \text{diag}(1,\dots,1,-1,\dots,-1)$。
  - ASE：邻接矩阵谱嵌入 $\hat{X} = \hat{U} \hat{S}^{1/2}$；LSE：归一化拉普拉斯谱嵌入 $\hat{Y}$。
  - $T_{ij}$：基于 $\hat{X}_i$ 与 $\hat{X}_j$ 构造的经验马氏距离统计量。
- **关键假设**：
  - **GRDPG 模型假设**：允许概率矩阵具有正负特征值，统一涵盖了 SBM、DCSBM 和 PABM。相比传统 RDPG 假设，放宽了对概率矩阵必须半正定的限制。
  - **潜在位置有界与矩条件**：控制谱嵌入的偏差项，保证逐行收敛。
  - **稀疏度条件 $n\rho_n \gg \log n$**：保证邻接矩阵的集中不等式成立，是 RMT 谱扰动界的基础。相比稠密图假设显著放宽，但排除了极稀疏情况（$n\rho_n \asymp \log n$ 或更低）。
- **问题背景**：
  - 已有方法不足：现有网络假设检验多针对全局结构（如社区数选择），缺乏对单个节点对潜在位置相等的局部检验；且多限于 SBM，未推广至带度校正或符号翻转的 GRDPG。
  - 与最相关文献的区别：1) Tang & Priebe (2018) 给出了 GRDPG 谱嵌入的极限分布，本文将其从点估计理论转化为具有显式非中心化参数的假设检验框架；2) Lei (2016) 针对 SBM 的社区检验，本文推广至 GRDPG 并引入缩放因子检验，解决了 DCSBM 模型选择问题。

### 三、主要定理 / 核心结果
1. **原文陈述**：
   - **Theorem 1 (Null distribution)**：在 $H_0: X_i = X_j$ 下，基于 ASE 的经验马氏距离 $T_{ij}^{(A)} \xrightarrow{d} \chi^2_d$。
   - **Theorem 2 (Local alternative)**：在 $H_{1,n}: X_i - X_j = \delta/\sqrt{n}$ 下，$T_{ij}^{(A)} \xrightarrow{d} \chi^2_d(\nu)$，非中心化参数 $\nu = \delta^\top \Sigma^{-1} \delta$（$\Sigma$ 为渐近协方差矩阵）。
   - **Theorem 3/4 (LSE counterparts)**：针对 LSE 的对应结论，非中心化参数形式不同。
2. **直观解释**：
   - 谱嵌入行向量 $\hat{X}_i$ 是真实潜在位置 $X_i$ 的渐近正态估计量，其协方差由 RMT 扰动结构决定。经验马氏距离将相关协方差“白化”，转化为标准的卡方分布。
   - 局部备择设定 $X_i - X_j = \delta/\sqrt{n}$ 使得信号强度恰好落在 $\sqrt{n}$ 收敛率的边界上，保证非中心化参数非零且有限，从而使得检验具有非平凡的功效。
3. **解决了什么技术难点**：
   - 将 RMT 中全局的谱扰动界（$\ell_2$ 范数）转化为逐行（$\ell_\infty$ 或特定行）的渐近正态性，并在局部备择下精确追踪均值向量的偏移与协方差矩阵的扰动，而非仅给出相合性。
4. **适用条件与局限**：
   - 必要假设：稀疏度 $n\rho_n \gg \log n$ 是保证逐行 CLT 成立的关键；局部备择的 $1/\sqrt{n}$ 收敛率是 ASE 的极小极大最优速率。
   - 可能放宽：对极稀疏网络（$n\rho_n \asymp \log n$），逐行正态性可能失效，结论可能不再成立。

### 四、证明框架 / 方法设计
- **证明主干逻辑**：随机矩阵扰动展开 + 中心极限定理 + Delta method。
- **拆解为关键逻辑步骤**：
  1. **谱分解扰动展开**：将 $\hat{X} - X W$ 展开为 $(A-P)$ 的多项式形式（留数展开），其中 $W$ 为正交/旋转矩阵，分离出线性主项与高阶余项。
  2. **逐行渐近正态性**：利用 Stein 方法或经典 CLT 证明 $(A-P)$ 的特定行向量组合渐近正态，从而得到 $\sqrt{n}(\hat{X}_i - X_i W)$ 的渐近正态性。
  3. **局部备择下的偏移追踪**：在 $X_i - X_j = \delta/\sqrt{n}$ 设定下，计算 $\hat{X}_i - \hat{X}_j$ 的均值与方差，证明方差在原假设与局部备择下同阶，均值产生 $\delta$ 量级的偏移。
  4. **经验马氏距离的极限**：用相合的协方差估计量构造马氏距离，利用 Slutsky 定理与连续映射定理得到非中心化卡方分布。
- **最关键的技巧性引理或"跳跃点"**：
  - 将全局谱扰动界转化为逐行展开的留数分析。具体而言，$\hat{X} - X W = (A-P) U S^{-1/2} + \text{remainder}$，如何在局部备择下控制 remainder 项的高阶渐近可忽略性，是证明中最具技巧性的步骤。这要求对特征向量投影算子的精细逼近。
- **数学工具评价**：经典工具的精巧组合。将 RMT 中的留数展开与经典数理统计中的局部渐近正态性（LAN）结合，把图模型的非参数/半参数结构转化为参数化的局部检验问题。

### 五、与研究者兴趣的关联
- **连接到哪个子方向**：高维统计中的随机矩阵理论（RMT）与数学统计中的假设检验（局部备择假设下的渐近功效分析）。
- **可借鉴的核心思路或技术工具**：将高维/复杂依赖结构（如网络邻接矩阵）的估计量（谱嵌入）通过扰动展开，剥离出独立同分布的噪声项，从而构造具有经典极限分布（如卡方分布）的检验统计量。这种“高维扰动 + 局部渐近正态”的范式可迁移至其他高维估计量的推断（如高维因子模型、隐变量模型的假设检验）。
- **值得精读的关键参考文献**：
  1. Tang & Priebe (2018) "Asymptotically efficient estimators for random graph inference"：提供了 GRDPG 谱嵌入的渐近正态性基础，是理解本文证明起点的必读文献。
  2. Cape, Tang & Priebe (2019) "The two-to-infinity norm and singular subspace geometry"：提供了将谱扰动从 $\ell_2$ 范数转化为逐行 $\ell_\infty$ 范数的核心技术，对高维推断极具参考价值。

### 六、延伸思考与练习
- **假设扰动**：若将稀疏度条件放宽至 $n\rho_n \asymp \log n$，谱嵌入的逐行正态性可能失效（出现泊松极限），此时卡方检验是否仍适用？技术上可能需要引入截断或新的集中不等式，或者退回考察 $\ell_2$ 范数检验统计量。
- **开放问题**：如何将此成对节点检验扩展到多个节点（如检验整个社区的均匀性），或在高维设定下（$d$ 随 $n$ 增长）构造类似的检验统计量？
- **理解检测题**：假设你有一个 GRDPG 模型，现需检验 $H_0: X_1 = c X_2$（$c$ 为未知缩放因子）。请基于本文思路，简述如何构造检验统计量，并指出与 $H_0: X_1 = X_2$ 的主要技术差异在哪里（提示：考虑参数 $c$ 的估计与投影空间的变化）。
