# Multivariate dynamic mediation analysis under a reinforcement learning framework

**作者**: Lan Luo, Chengchun Shi, Jitao Wang, Zhenke Wu, Lexin Li  
**来源**: Annals of Statistics  
**主题**: 因果推断  
**相关性**: 10/10  
**链接**: <https://doi.org/10.1214/24-aos2475>

---

### 一、核心问题与贡献
①研究了纵向设定下存在多重条件依赖中介变量的动态中介分析问题。②引入受强化学习启发的Markov中介过程与时变线性结构方程模型(SEM)，基于同时干预与干预演算定义个体中介效应。③推导出包含上游路径与跨期滞后效应的中介效应闭式表达，并提出具有渐近保证的迭代估计程序。

### 二、基础设定
- **核心概念与符号**：
  - **Markov中介过程 (MMP)**：受MDP启发，状态转移满足Markov性质的动态中介系统。
  - $S_t, A_t, M_t, Y_t$：时间$t$的状态、处理、中介向量、结果变量。
  - **时变线性SEM**：刻画$S_{t+1}, M_t, Y_t$的线性生成机制，系数矩阵随时间$t$变化。
  - **同时干预**：对多个/多期中介施加联合干预，以剥离特定路径的效应。
  - **干预演算**：Pearl的do-calculus代数化，用于定义和计算路径特定效应。
- **关键假设**：
  - **Markov假设**：$P(X_{t+1}|X_t, X_{t-1}, \dots) = P(X_{t+1}|X_t)$。统计学含义：截断历史依赖，使动态路径可解。相比一般纵向因果模型，这是极强的结构假设，但换取了闭式解的可能。
  - **顺序可忽略性**：给定历史观测，处理分配与潜在结果独立。与标准纵向因果推断一致，排除了未观测混杂。
  - **线性SEM假设**：变量间关系为线性时不变/时变。含义：允许矩阵分解与闭式路径求和。相比非参数模型，限制了非线性交互，但避免了维数灾难。
- **问题背景**：
  - 现有纵向中介分析多假设中介变量条件独立，或无法处理跨期滞后效应与上游中介路径的纠缠，导致效应分解不可识别或组合爆炸。
  - 与VanderWeele的纵向中介（多关注单中介或独立中介）相比，本文处理多重依赖中介；与Imai的动态中介相比，本文引入MDP结构给出闭式解而非仅依赖非参数识别。

### 三、主要定理 / 核心结果
1. **中介效应的闭式表达（定理/命题形式）**
   - **原文陈述**：在MMP与线性SEM下，个体中介效应可表示为系统矩阵乘积的闭式求和，具体为转移矩阵与直接效应矩阵的幂级数/连乘形式。
   - **直观解释**：将复杂的动态有向无环图(DAG)上的路径追踪，转化为矩阵代数中的Neumann级数。上游效应与滞后效应被矩阵的非对角线元素与次对角线块自动捕捉，路径组合爆炸问题被矩阵乘法降维。
   - **解决了什么技术难点**：解决了随时间$T$指数级增长的路径数量导致的识别与计算不可行问题。
   - **适用条件与局限**：强依赖Markov与线性假设。若非线性，矩阵乘积失效；若无Markov，状态空间需无限维，闭式解不再成立。

2. **迭代估计量的渐近性质**
   - **原文陈述**：基于逐时段回归得到的矩阵参数估计量，通过闭式表达式迭代计算的中介效应估计量$\hat{\theta}$具有$\sqrt{n}$一致性且渐近正态。
   - **直观解释**：尽管中介效应是参数的非线性矩阵函数，由于底层SEM参数的OLS/GLS估计是$\sqrt{n}$一致的，通过Delta method或鞅差分展开，最终效应估计仍保持最优速率。
   - **解决了什么技术难点**：处理了动态系统中参数估计误差在矩阵连乘与时间维度上的累积问题。
   - **适用条件与局限**：需标准的矩条件与特征值条件（保证矩阵级数收敛与OLS的良态性）。

### 四、证明框架 / 方法设计
- **证明主干逻辑**：构造法 + 代数递推 + 矩方法。
- **拆解关键逻辑步骤**：
  1. **路径代数化**：利用线性SEM，将DAG上的路径求和转化为矩阵多项式/级数的展开。
  2. **Markov递推**：利用Markov性质，将$t+1$期的效应分解为$t$期效应的线性变换（转移矩阵的作用），建立递推关系。
  3. **闭式求解**：求解该矩阵递推方程，得到几何级数形式的闭式解（如$(I-A)^{-1}$的展开）。
  4. **渐近分析**：对迭代估计量进行Taylor展开，利用鞅差分序列的中心极限定理，证明$\sqrt{n}$收敛性并计算渐近方差。
- **最关键的技巧性引理或"跳跃点"**：将因果图上的路径追踪（通常需要复杂的do-calculus推导）转化为**矩阵的Neumann级数/几何级数求和**。这是Markov+Linear带来的红利，使得组合爆炸的路径数被压缩为矩阵维度的多项式计算。
- **数学工具评价**：是经典工具（线性SEM、MDP转移矩阵、Delta method）的巧妙组合，并非全新分析框架，但在中介分析的识别问题上实现了从非参数/局部识别到全局闭式解的突破。

### 五、与研究者兴趣的关联
- **连接子方向**：纵向中介分析 与 动态因果推断。
- **可借鉴的核心思路**：
  - 将RL中的MDP结构引入因果图建模，用转移矩阵刻画时间依赖，这对处理longitudinal causal inference中的复杂滞后效应极具启发性。
  - 利用线性假设将路径识别转化为矩阵级数，为后续可能的debiasing或高维扩展提供了清晰的代数结构。若结合高维统计，可探索在$p \gg n$下对转移矩阵进行正则化估计及中介效应的debias推断。
- **值得精读的关键参考文献**：
  - Pearl (2009) *Causality*: 理解intervention calculus与反事实定义的基础。
  - Murphy (2003) *Optimal dynamic treatment regimes*: 理解MDP与顺序决策在因果推断中应用的开山之作，本文MDP设定的直接灵感来源。

### 六、延伸思考与练习
- **假设扰动**：若放宽线性SEM假设，改为部分线性或单指数模型，矩阵级数闭式解失效。技术上需要引入经验过程理论或局部似然方法来近似路径积分，或者退回使用半参数效率界的Neyman正交估计，计算复杂度将急剧上升。
- **开放问题**：
  - 如何在存在未观测混杂（违背Sequential Ignorability）时，在此MDP框架下进行Sensitivity Analysis？
  - 当中介维度$p$随时间或样本量发散时，如何进行高维动态中介推断？
- **理解检测题**：在本文的Markov中介过程与线性SEM设定下，假设存在一个中介变量$M_{t-1}^{(1)}$对$M_t^{(2)}$有直接影响，同时$M_{t-1}^{(1)}$也影响$S_t$。请利用文中推导的矩阵闭式解逻辑，说明$M_{t-1}^{(1)}$对$Y_{t+1}$的效应是如何被分解为"通过$M_t^{(2)}$的路径"与"通过$S_t$的路径"的叠加的（无需计算具体数值，写出矩阵映射逻辑即可）。
