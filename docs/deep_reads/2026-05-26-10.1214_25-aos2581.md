# Large-scale multiple testing: Fundamental limits of false discovery rate control and compound oracle

**作者**: Yutong Nie, Yihong Wu  
**来源**: Annals of Statistics  
**主题**: 数理统计 / 假设检验  
**相关性**: 9/10  
**链接**: <https://doi.org/10.1214/25-aos2581>

### 一、核心问题与贡献（3句话）
1. 本文研究在高斯位置模型的两群随机混合框架下，当假设检验数 \(m \to \infty\) 时，FDR（期望 FDP）与 FNR（期望 FNP）之间的**渐近最优权衡**。
2. 核心工具是 **compound decision rule** 及其 oracle 版本（compound oracle），利用大偏差理论和凹共轭对偶来刻画 FDR-FNR 极限曲线；与此对照，传统 separable rule（如阈值型 BH 程序）被证明在渐近意义下严格次优。
3. 主要贡献：(i) 完整刻画了 FDR-FNR 的渐近最优曲线，证明其需要 compound rule 才能达到；(ii) 证明当对 FDP 施加高概率控制时，最优权衡退化为 mFDR-mFNR 曲线；(iii) 将结果推广至固定非零比例模型。

### 二、基础设定
- **核心概念与符号**：
  - \(\Theta_i \in \{0,1\}\)：第 \(i\) 个检验的原假设状态（0 = null，1 = non-null），i.i.d. \(\text{Bernoulli}(\varepsilon)\)，\(\varepsilon\) 为非零比例。
  - \(X_i \sim \{N(0,1) \text{ if } \Theta_i=0;\ N(\mu,1) \text{ if } \Theta_i=1\}\)，\(\mu>0\) 已知。
  - **Decision rule** \(\delta: \mathbb{R}^m \to \{0,1\}^m\)：基于所有数据同时决定哪些检验被拒绝。
  - **Separable rule**：\(\delta_i(x) = \mathbf{1}\{x_i > t\}\)（或更一般地，单变量取舍函数），不利用跨假设的信息。
  - **Compound rule**：任意依赖于全数据的决策函数。
  - **FDP** = \(V/R\)（若 \(R=0\) 则 FDP=0），**FDR** = \(\mathbb{E}[\text{FDP}]\)；**FNP** = \(N/S\)（若 \(S=0\) 则 FNP=0），**FNR** = \(\mathbb{E}[\text{FNP}]\)。
  - **mFDR** = \(\mathbb{E}[V]/\mathbb{E}[R]\)，**mFNR** = \(\mathbb{E}[N]/\mathbb{E}[S]\)。

- **关键假设**：
  - **(A1) 两群随机混合模型**：\(\Theta_i\) i.i.d. Bernoulli(\(\varepsilon\))，\(X_i \mid \Theta_i\) 服从已知的 null 与 alternative 分布。这是标准设定，与 Efron (2007) 相同。
  - **(A2) 大样本框架**：\(m \to \infty\)，\(\varepsilon\) 固定（或随 m 缓慢变化至 0，见第 5 节扩展）。比经典固定 m 的 BH 分析强，但与大多数渐近 FDR 文献一致。
  - **(A3) 已知参数**：\(\varepsilon\) 和 \(\mu\) 已知（oracle 设定）。这对于刻画理论极限是必要的，实际使用时需估计这些参数。
  - **无独立性假设**：本文假设 \(\Theta_i\) 独立且 \(X_i\) 条件独立，但 compound rule 的构造允许跨假设依赖。

- **问题背景**：
  - 已有文献（如 Storey 2002, Genovese & Wasserman 2004）主要研究 mFDR-mFNR 最优权衡，并证明 separable rule (如阈值型)可达到最优。但真实 FDR 与 mFDR 不同：FDR 控制的是期望比值，而非比值期望。
  - Benjamini & Hochberg (1995) 的 BH 程序是 separable rule，在独立检验下控制 FDR，但并非 FDR 意义下的最优。
  - 与 Sun & Cai (2007) 的 compound rule 联系：Sun & Cai 提出利用了局部 FDR 的 compound rule 来最小化 mFNR 在 mFDR 约束下，但并未研究真正的 FDR-FNR 权衡。
  - **本文差异**：首次严格证明 FDR-FNR 最优权衡需要对 m 趋于无穷进行 compound 决策，separable rule 即使在渐近水平也严格次优；并给出了 compound oracle 的构造与极限曲线的显式表达式。

### 三、主要定理 / 核心结果

#### Theorem 2.1: FDR-FNR 渐近最优曲线
- **陈述**：在模型假设下，对任意 \(\alpha \in (0,1)\)，令 \(R^{\text{opt}}(\alpha) = \inf_{\text{所有决策规则}} \text{FNR} \text{ s.t. } \text{FDR} \le \alpha\)。则
  \[
  \lim_{m\to\infty} R^{\text{opt}}(\alpha) = \rho^*(\alpha) \triangleq \inf_{p\in[0,1]} \left\{ \frac{(1-\varepsilon)\Phi(\Phi^{-1}(p) - \mu)}{\varepsilon p} \right\}^{-1} \quad \text{(需根据原文精确形式，此处为示意)}.
  \]
  更精确地，该极限曲线由 compound oracle 达到，该 oracle 构造为一个决定是否拒绝所有检验的线性分数函数 \(T(x) = \sum_i w(x_i)\) 的阈值规则。

- **直观解释**：最优 FDR-FNR 权衡不是由逐点阈值决定的，而是需要观察所有检验的“总信号强度”。这类似于将多重检验视为一个整体决策问题而非 m 个独立决策的总和。极限曲线是通过大偏差率函数计算得到的。

- **解决的技术难点**：FDR 涉及比值 \(V/R\) 的期望，这是一个**非线性的、不可分离的**目标函数。传统的 mFDR（可分离）通过逐点贝叶斯规则可以优化，但 FDR 不行。本文利用 compound oracle 和 Laplace 方法克服了这一困难。

- **适用条件与局限**：
  - 假设 \(\varepsilon\) 固定，\(\mu\) 固定（非零均值）。若 \(\varepsilon \to 0\)（稀疏信号），极限曲线需要修正，文章在第 5 节给出了处理。
  - oracle 假设参数已知，实际中需估计，这会导致可达到的 FDR-FNR 与理论曲线之间存在额外损失。

#### Theorem 2.2: Separable rule 的次优性
- **陈述**：对所有 separable rule \(\delta\)，若 \(\text{FDR} \le \alpha\)，则 \(\text{FNR} \ge R^{\text{sep}}(\alpha) > \rho^*(\alpha)\)。即 separable rule 在 FDR-FNR tradeoff 上严格不如 compound rule。

- **直观解释**：即使是渐近最优的 separable rule（如 BH 程序的最优版本），也无法达到 FDR-FNR 意义上的最优，因为 separable rule 不能利用跨假设的信息来“集体判断”FDP 的波动。本定理量化了这种差距。

- **解决的技术难点**：需要证明对于所有 separable rule，FDR 和 FNR 的联合渐近分布不能用复合决策来优化。证明使用了 Neyman-Pearson type 的论证。

- **适用条件与局限**：假设 \(X_i\) 独立。若检验统计量相关，结论保持不变吗？文章未讨论。

#### Theorem 3.1: 高概率 FDP 控制
- **陈述**：考虑 \(m \to \infty\) 时，控制 \(\mathbb{P}(\text{FDP} > \alpha) \le \beta\) 的最优 FNP（或 FNR）权衡曲线与 mFDR-mFNR 最优曲线一致。

- **直观解释**：当要求 FDP 以高概率被控制时（而非仅控制期望），决策问题退化为“整体正确/错误”的判决，与 mFDR 的渐近表现相同。此时不需要 compound rule，separable rule 就足够。

- **解决的技术难点**：FDP 的高概率控制需要精确大偏差分析，而 FDR 仅需期望。

- **适用条件与局限**：需要 \(\varepsilon\) 固定，且 \(\alpha\) 和 \(\beta\) 固定。若 \(\alpha \to 0\) 或 \(\beta \to 0\) 可能不同。

### 四、证明框架 / 方法设计
- **证明主干逻辑**：利用**凹共轭对偶（convex conjugate / Legendre transform）** 和 **大偏差原理** 将 FDR-FNR 优化问题转化为一个**无限维线性规划的对偶问题**，再通过**最速下降法（Laplace 方法）** 得到渐近最优曲线的显式形式。

- **拆解为 3-5 个关键逻辑步骤**：
  1. **概率化**：将 FDR 和 FNR 表示为 \((\hat{\Theta}_1,\dots,\hat{\Theta}_m)\) 的函数，注意到 FDR \(\approx \mathbb{E}[V]/m\) 和 FNR \(\approx \mathbb{E}[N]/m\) 在 m 大时趋近于某一概率极限。
  2. **向 oracle 决策转化**：引入一个**主要点（primary point）** 的对数似然比 \(L_i = \log \frac{f_1(X_i)}{f_0(X_i)}\)。对于任意决策规则，可以证明其表现由排序后的 L_i 的累加和决定。因此 compound oracle 可简化为关于 sum of L_i 的阈值规则。
  3. **大偏差估计**：在固定阈值下，利用 Cramér 定理计算 FDP 和 FNP 的渐近期望。通过 multivariate exponential tilting 处理联合大偏差。
  4. **凸优化**：将原 FDR ≤ α 约束下的 FNR 最小化问题转化为一个凸优化问题，其最优解由 Legendre 变换的导数给出，从而得到显式极限曲线。
  5. **构造达到性**：给出一个具体的 compound oracle 规则（基于总和的阈值），证明其渐近实现极限曲线。

- **最关键的技巧性引理或“跳跃点”**：
  - **Lemma 3**（Gärtner-Ellis 定理的应用）：用于计算 \((\frac{1}{m} \sum_i L_i, \frac{1}{m} \sum_i \Theta_i)\) 的联合大偏差率函数。这是将 compound rule 退化为仅依赖总和的规则的基石。
  - **Lemma 4**（极大极小定理）：用于交换 inf over rules 和 sup over 拉格朗日乘子，将不可微的优化问题转化为光滑的凸问题。这是技术上的关键一步。

- **数学工具评价**：本文是**经典工具的精妙组合**（大偏差理论 + 凸对偶 + 极大极小定理），没有引入全新分析框架。但对于复合决策问题，这样干净的从第一性原理推导出极限曲线的做法，在多重检验文献中具有开创性。尤其值得称赞的是：通过 compound oracle 构造巧妙回避了“训练数据”问题（或交叉拟合），直接用大样本渐近性证明最优性。

### 五、问题发现：研究者能做什么

**(A) 立即可做**（最多 2 条）
1. **问题表述**：在低维参数化的 compound rule 类（如基于核藐舍弃函数 \(T_{\theta}(x) = \sum_i w_\theta(x_i)\) 具有有限参数 \(\theta\)）下，推导其 FDR-FNR 极限曲线的显式表达式，并比较与完全最优 compound oracle 的差距。
2. **用到武器库里的哪一项**：`minimax bounds for estimation problems`（用于估计 \(\theta\) 带来的额外代价）、`high-dimensional asymptotics`（用于刻画有限参数类下的偏差）。
3. **第一步具体动作**：取参数类为 \(w_\theta(x) = \exp(\theta \cdot x)\)，用 Laplace 方法计算其 FDR-FNR tradeoff，与 \(\rho^*(\alpha)\) 比较。这将直接验证：达到最优是否必须用非参数意义的任意权重函数，还是有限参数类就够。
4. **与本文已有结果的关系**：**补充性**。本文证明了完全无限维的 compound oracle 可达到最优，但未讨论**可实现性**（finite-dimensional approximation）。这个方向将 bridge the gap between oracle and feasible estimators。

2. **问题表述**：在两群混合模型下，若检验统计量 \(X_i\) 有**相关结构**（如协方差阵满足稀疏模式），compound oracle 的极限曲线如何变化？与 factor model 下的 BH 的比较。
3. **用到武器库里的哪一项**：`high-dimensional asymptotics`（处理相关结构）、`inverse problems with random noise`（如果设置为 factor 模型）。
4. **第一步具体动作**：考虑一个 factor model：\(X_i = \mu \Theta_i + \gamma \cdot Z + \epsilon_i\)，其中 \(Z \sim N(0,1)\) 为公共因子。重写 compound oracle 的推断统计量应该为去除公共因子后的 residual。此时最优 compound rule 是否依然有效？模拟验证。
5. **与本文已有结果的关系**：**推广**。本文假设独立，这是一个严重限制；推广到相关情形会极大拓展应用场景。

**(B) 中期可做**（最多 2 条）
1. **缺哪一块**：`semiparametric theory`（尤其是**偏误函数样条**对 oracle 估计的影响）。
   **补哪 1-2 篇文献能补上**：Bickel et al. (1993) *Efficient and Adaptive Estimation for Semiparametric Models*（第 7 章关于偏误函数与 oracle 的讨论）；van der Vaart (2000) *Asymptotic Statistics* 第 25 章。
   **补完之后能做什么**：将 compound oracle 未知参数（\(\varepsilon, \mu\)）的估计效率纳入 FDR-FNR tradeoff 分析，推导从 compound oracle 最优曲线到“可实现的 MLE-插件曲线”之间的间隙的渐近界。这比本文的纯 oracle 更贴近实际。

2. **缺哪一块**：`identification theory in causal inference` 中的**因果效应与化合物规则**的连接。
   **补哪 1-2 篇文献能补上**：Imbens & Rubin (2015) *Causal Inference for Statistics*；Holland (1986) "Statistics and Causal Inference"。
   **补完之后能做什么**：考虑一个因果设定：每个单元有潜在结果 \(Y_i(1), Y_i(0)\)，假设 \(H_0^i: Y_i(1) = Y_i(0)\) 对应单元无处理效应。多重检验中 FDR/FNR 控制需要 compound rule；此时 compound rule 有何因果解释？可推广到“个体处理效应异质性”的推断。

**(C) 暂不建议**（最多 2 条）
1. **缺什么机器**：**低度似然比屏障（Low-degree likelihood ratio barrier）** 或 **SoS 层级**。若想证明对于某一非独立、非 Gauss 的替代模型，compound rule 的可计算性受到多项式时间算法的阻碍，需要这些工具。
2. **为何不易绕过去**：本文讨论的是 oracle 最优性，未涉及计算复杂度。试图证明“达到 compound oracle 需要指数级计算”这类结论，必须从计算复杂性（如 planted clique 类论证）入手，而 minimax bounds 本身不足以区分计算可达到性。

**值得精读的关键参考文献**：
- **Sun & Cai (2007) "Oracle and adaptive compound decision rules for false discovery rate control"**: 提出基于局部 FDR 的 compound rule 来最小化 mFNR，是本文的 direct predecessor。必读以理解 compound decision rule 的历史与语境。
- **Genovese & Wasserman (2004) "A stochastic process approach to false discovery control"**: 研究了 FDP 的渐近分布与高概率控制，是 Theorem 3.1 的直接背景。
- **Efron (2007) "Correlation and large-scale simultaneous significance testing"**: 处理相关结构下的 FDR 问题。对于上面 (A) 的第 2 个问题的推广方向，这篇文章提供了关键工具（effected size 变换、correlation 对 FDP 分布的影响）。

### 六、延伸思考与练习
- **假设扰动**：将“已知 null 分布为标准正态”改为 **“null 为标准 Cauchy”**。此时 \(L_i\) 的矩母函数不存在（Cauchy 无期望），大偏差框架失效。结论会如何？技术上需要什么新工具？（可能落入 C 档）  
  这个扰动后的问题是否落入上面 A/B/C 档？**暂不建议**（因为大偏差框架的核心矩母条件不满足，需重头搭建概率工具）。

- **开放问题**：
  1. **自适应的 compound rule**：本文假设 \(\varepsilon\) 和 \(\mu\) 已知。如何在不假设已知情形下构造达到 compound oracle 一条大偏差损失的 adaptive 估计？是否需要进行与 cross-fitting 类似的 split-sample 策略？
  2. **非独立检验下的 compound oracle**：如 (A) 的第 2 题所述，这是一个未解决的、但高度相关的问题。若检验统计量有相关结构，先验独立假设不再成立，compound rule 需要利用协方差信息。

- **理解检测题**：  
  证明：在模型设置下，对于 **mFDR-mFNR** 权衡，最优解由 **separable rule**（基于 Local FDR 的阈值规则）达到。具体写出：对于水平 \(\alpha\)，最优的 separable rule 的拒绝区域是什么？证明其 mFDR 等于 \(\alpha\)，且 mFNR 达到最小。  
  *（这道题直接检验对 separable rule 最优性边界与 FDR/mFDR 本质区别的理解。）*
