# Optimal Estimation of Shared Singular Subspaces Across Multiple Noisy Matrices

**作者**: Zhengchi Ma, Rong Ma  
**来源**: IEEE Transactions on Information Theory  
**主题**: 高维统计 / 随机矩阵  
**相关性**: 8/10  
**链接**: <https://doi.org/10.1109/tit.2026.3667733>

---

### 一、核心问题与贡献
①研究了多个噪声矩阵间共享奇异子空间的估计问题，特别是在子空间部分共享（partial sharing）的复杂设定下。②核心工具是基于拼接矩阵的SVD（Stack-SVD）的渐近理论分析，以及针对部分共享结构提出的新估计量与算法。③主要贡献是严格刻画了Stack-SVD与Average-SVD的minimax最优性边界与失效条件，并在部分共享设定下提出了达到minimax rate-optimality的新估计量。

### 二、基础设定
- **核心概念与符号**：
  - $M^{(l)} = U^{(l)} \Sigma^{(l)} (V^{(l)})^\top$：第 $l$ 个噪声矩阵的SVD分解，$U^{(l)}$ 为左奇异向量矩阵。
  - Shared subspace / Unshared subspace：多个矩阵的奇异子空间中完全重合的部分与各自独有的部分。
  - Stack-SVD：将多个矩阵按行（或列）拼接后对整体矩阵做SVD提取共享子空间。
  - Average-SVD：分别对每个矩阵做SVD，将提取的奇异向量矩阵拼接/平均。
  - Signal strength (SNR)：奇异值大小与噪声水平的比值，决定估计一致性。
  - Angle deviation / Subspace distance：衡量不同矩阵间非共享子空间的偏离程度。
- **关键假设**：
  - 低秩假设：真实信号矩阵为低秩，这是高维随机矩阵降噪的基石。
  - 噪声假设：加性噪声，通常要求亚高斯或高斯分布，具有特定的方差结构。
  - 信号强度条件：奇异值需大于特定阈值（如 $\sqrt{n}$ 量级），以保证奇异向量的一致性。与单矩阵Davis-Kahan定理的扰动条件相比，多矩阵设定下的阈值受矩阵间角度偏离的交互影响。
  - 部分共享结构假设：假设子空间可分解为共享与独有部分，且独有部分与共享部分正交。相比已有文献通常要求的完全共享，该假设被显著放宽。
- **问题背景**：
  - 已有方法在多矩阵共享子空间估计中，常直接使用Stack-SVD或Average-SVD，但缺乏在部分共享与不同信噪比下的严格minimax理论保证。
  - 与最相关文献的区别：相比仅研究完全共享设定的多视角PCA或分布式PCA，本文深入刻画了部分共享的边界；相比仅提供算法收敛性的工作，本文给出了严格的minimax下界与速率最优性证明。

### 三、主要定理 / 核心结果

1. **完全共享设定下的Stack-SVD最优性与Average-SVD次优性定理**
   - 原文陈述：当所有矩阵的真实奇异子空间完全相同时，Stack-SVD的估计误差达到minimax下界；而Average-SVD的收敛速率可能慢于minimax速率。
   - 直观解释：Stack-SVD等效于增大了样本量（信号能量叠加），充分利用了多矩阵的共享信息；Average-SVD先对各自带噪的奇异向量做平均，受限于单矩阵估计的初始偏差，无法通过拼接消除噪声的各向异性。
   - 解决了什么技术难点：澄清了多视角文献中关于"先拼接还是先SVD"的长期争议，给出了严格的minimax速率对比。
   - 适用条件与局限：要求子空间完全相同，若存在微小偏离，Average-SVD的鲁棒性可能在此定理框架外。

2. **部分共享设定下Stack-SVD的相图定理**
   - 原文陈述：刻画了Stack-SVD保持一致、达到minimax最优、或估计不一致的相变条件，这些条件依赖于信号强度与子空间角度偏离的交互。
   - 直观解释：当非共享部分的信号强度较弱或角度偏离极小时，Stack-SVD将其视作"噪声"吸收，仍能提取共享部分；但当偏离较大或非共享信号较强时，Stack-SVD会发生"空间泄漏"，导致共享子空间估计不一致。
   - 解决了什么技术难点：将传统的Davis-Kahan扰动理论从单矩阵拓展至多矩阵拼接且存在异质信号干扰的复杂场景，分离了共享与独有信号的干扰机制。
   - 适用条件与局限：依赖于对角度偏离与信噪比精确的渐近阶数假设，常数项的精确相变点可能未完全刻画。
