# Estimating causal effects with hidden confounding using instrumental variables and environments

**作者**: James P. Long, Hongxu Zhu, Kim-Anh Do, Min Jin Ha  
**来源**: Electronic Journal of Statistics  
**主题**: 因果推断  
**相关性**: 8/10  
**链接**: <https://doi.org/10.1214/23-ejs2160>

---

### 一、核心问题与贡献
①研究了在存在隐藏混杂时，如何利用跨环境不变性（特别是连续环境变量）进行因果效应估计的问题。②将 Causal Dantzig (CD) 重新推导为广义矩估计（GMM），并据此提出适用于连续环境的 Generalized Causal Dantzig (GCD) 及其与 TSLS 结合的 Hybrid 估计量。③证明了 GCD 和 Hybrid 估计量的渐近正态性，克服了原 CD 无法处理连续环境的局限，且 Hybrid 估计量通过聚合不同来源的矩条件在多种设定下优于单独的 TSLS 或 CD。

### 二、基础设定
- **核心概念与符号**：
  - $Y$：结果变量；$X$：处理变量；$E$：环境变量；$U$：隐藏混杂。
  - $\beta_0$：目标因果参数（结构方程系数）。
  - CD (Causal Dantzig)：基于跨环境残差与环境的正交条件构造的估计量，原形式仅适用于离散环境。
  - GCD (Generalized Causal Dantzig)：将 CD 推广至连续环境的 GMM 估计量。
  - Hybrid：联合 GCD 矩条件与 TSLS 矩条件的联合 GMM 估计量。
- **关键假设**：
  - **Invariance assumption (跨环境不变性)**：给定 $X$ 和 $U$，$Y$ 的条件分布跨环境不变（即结构方程参数不变）。含义：环境 $E$ 对 $Y$ 无直接因果作用，仅通过 $X$ 起作用。相比传统 IV，这是从机制不变性角度出发的假设。
  - **Exogeneity of environment (环境外生性/独立性)**：$E \perp\!\!\!\perp U$。含义：环境变量与隐藏混杂独立，类似于经典 IV 的排他性限制。
  - **Relevance of environment (环境相关性)**：$E$ 与 $X$ 存在相关性。含义：环境变量能影响处理变量，类似于 IV 的相关性条件。
  - *假设放宽/强化*：相比 Rothenhäusler et al. (2018) 的原始 CD，本文显著放宽了环境变量必须为离散的分类变量的假设，允许 $E$ 为连续变量；相比传统 TSLS，本文通过不变性框架允许在更一般的误差分布下构造矩条件。
- **问题背景**：原始 Causal Dantzig 估计量仅通过环境分组的样本协方差差构造，缺乏统一的渐近理论框架，且无法利用连续环境变量的信息；传统 TSLS 在弱工具变量或模型轻微误设时表现不佳。与最相关文献的区别：相比 Rothenhäusler et al. (2018) 的 Causal Dantzig，本文将其纳入 GMM 框架并推广至连续环境；相比传统 IV 文献，本文将环境变量视为一种特殊的工具变量，并利用不变性构造更丰富的矩条件。

### 三、主要定理 / 核心结果
1. **原文陈述**：CD/GCD/Hybrid 估计量均可表示为 GMM 估计量，满足 $\sqrt{n}(\hat{\beta} - \beta_0) \xrightarrow{d} N(0, V)$，其中 $V = (G' \Omega^{-1} G)^{-1}$，$G = E[\partial g(W, \beta_0)/\partial \beta]$，$\Omega = E[g(W, \beta_0)g(W, \beta_0)']$。Hybrid 估计量的渐近方差矩阵在 Loewner 偏序下小于等于单独的 TSLS 或 GCD。
2. **直观解释**：将不变性条件转化为总体矩条件，使得经典 GMM 渐近理论直接可用。Hybrid 估计量由于同时利用了环境不变性矩条件和传统 IV 矩条件，聚合了更多的识别信息，因此具有更小的渐近方差。
3. **解决了什么技术难点**：解决了原 CD 估计量缺乏统一渐近分布推导的问题；克服了离散环境设定下无法处理连续变量 $E$ 的计算与理论障碍。
4. **适用条件与局限**：依赖于不变性假设与环境外生性假设的严格成立。若环境对 $Y$ 有直接作用（不变性违背），或环境与 $U$ 相关，GMM 矩条件不成立，估计量不一致。最优权重矩阵 $\Omega$ 的估计在样本量较小时可能不稳定。

### 四、证明框架 / 方法设计
- **证明主干逻辑**：构造法 + 经典 GMM 渐近理论。
- **拆解为 3-5 个关键逻辑步骤**：
  1. **矩条件构造**：从结构方程模型出发，利用不变性与外生性推导出残差与环境的正交条件，构造总体矩条件 $E[(Y-X\beta)h(E)]=0$（GCD）及 $E[(Y-X\beta)Z]=0$（TSLS）。
  2. **等价性验证**：证明在离散环境下，特定的 $h(E)$ 选择使得 GMM 目标函数与原 CD 目标函数等价，建立与已有文献的连接。
  3. **两步 GMM 实施**：第一步使用单位权重矩阵得到初始一致估计，第二步利用初始估计计算最优权重矩阵 $\hat{\Omega}$，得到渐近有效的 GMM 估计量。
  4. **渐近性质推导**：应用标准 GMM 定理，计算雅可比矩阵 $G$ 和矩方差 $\Omega$，得出联合估计量的渐近正态性及方差公式，并通过矩阵分块求逆证明 Hybrid 估计量的方差占优。
- **最关键的技巧性引理或"跳跃点"**：将 Causal Dantzig 的损失函数（涉及跨环境协方差矩阵的差）重写为样本矩条件的二次型。这一步是桥梁，使得原本看似非标准的优化问题落入 GMM 的标准范式，直接解锁了渐近理论与连续变量推广的路径。
- **数学工具评价**：经典工具（GMM 框架）的巧妙组合与重构。没有发明新数学，但通过视角转换（不变性 -> 矩条件）解决了原方法的理论与计算瓶颈。

### 五、与研究者兴趣的关联
- **连接到哪个子方向**：因果推断中的 IV 估计与不变性因果推断。
- **可借鉴的核心思路或技术工具**：将非标准因果估计量（如基于不变性、分布鲁棒性的估计量）转化为 GMM 框架，是获取渐近理论、进行假设检验和构造置信区间的通用范式。Hybrid 估计量的思路（结合不同来源的矩条件/识别策略）可直接迁移到 Proximal CI 中，例如结合 negative control 矩条件与传统 IV 矩条件，或结合不同代理变量构造联合 GMM 以提升估计效率。
- **值得精读的关键参考文献**：
  1. Rothenhäusler et al. (2018) "Causal Dantzig"：原始 CD 方法的来源，理解本文动机与等价性推导必读。
  2. Hansen (1982) "Large Sample Properties of GMM Estimators"：GMM 渐近理论的基石，本文理论推导的直接依据。
  3. Peters et al. (2016) "Causal inference using invariant prediction"：跨环境不变性因果推断的另一核心文献，提供了不同的不变性视角与算法。

### 六、延伸思考与练习
- **假设扰动**：若"跨环境不变性"假设部分违背（例如环境 $E$ 对 $Y$ 有微弱的直接因果效应，即排他性假设近似失效），GCD 和 Hybrid 估计量将不再一致。技术上需要引入敏感性分析框架，量化偏差随违背程度的变化，或将严格的矩条件 $E[g]=0$ 放宽为区间约束 $||E[g]|| \leq \delta$，走向部分识别。
- **开放问题**：如何在高维设定下（$X$ 维度远大于样本量）进行 GCD 或 Hybrid 估计？需要结合正则化或 Debiased ML 技术处理高维矩条件或高维协变量。
- **理解检测题**：假设环境变量 $E$ 是连续的，且你选择基函数 $h(E) = (1, E, E^2)'$ 来构造 GCD 的矩条件。请写出此时的总体矩条件形式，并说明相比于仅使用 $h(E) = (1, E)'$，增加 $E^2$ 项对估计量的渐近方差和过度识别检验有何影响？
