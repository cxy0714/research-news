# Simultaneous semiparametric inference for single-index models

**作者**: Jiajun Tang, Holger Dette  
**来源**: Bernoulli  
**主题**: 非参数 / 半参数  
**相关性**: 9/10  
**链接**: <https://doi.org/10.3150/24-bej1834>

---

### 一、核心问题与贡献
①本文研究了部分线性单指标模型中参数分量（指标系数）与非参数分量（联系函数）的联合半参数推断问题。②核心工具是基于再生核希尔伯特空间（RKHS）与经验过程理论建立平滑样条估计量的 Bahadur 表达。③主要贡献是证明了联系函数估计量与指标系数估计量渐近独立，并据此构造了联系函数的同时置信带及参数-非参数联合假设检验的 multiplier bootstrap 方法。

### 二、基础设定
- **核心概念与符号**：
  - 模型：$Y = \theta^T Z + g(X^T \beta) + \epsilon$，其中 $\theta$ 为线性参数，$\beta$ 为单指标系数，$g(\cdot)$ 为未知联系函数。
  - $\hat{g}_n$ 与 $(\hat{\theta}_n, \hat{\beta}_n)$：基于惩罚最小二乘的平滑样条估计量。
  - RKHS：$\mathcal{H}_K$，用于刻画 $g$ 的函数空间与惩罚项的几何结构。
  - Bahadur 表达：将半参数估计量分解为 i.i.d. 随机项之和与高阶余项的线性化展开。
- **关键假设**：
  - 光滑性假设：$g$ 属于 Sobolev 空间 $\mathcal{W}^{p,2}$，阶数 $p>1$。含义：保证非参数估计的极小极大最优速率 $n^{-2p/(2p+1)}$ 可达；相比低阶光滑假设，此条件支撑了 Bahadur 余项的收敛速度。
  - 设计变量分布：$X, Z$ 具有紧支撑且密度函数有界远离0。含义：避免边界效应与设计矩阵的奇异问题，确保 RKHS 中经验算子向总体算子的收敛。
  - 误差项：$\epsilon$ i.i.d. 且具有亚高尾。含义：支撑经验过程中的指数不等式与极大值的极值理论分析。
- **问题背景**：
  - 已有方法的不足：现有单指标模型的推断多集中于参数分量的 $\sqrt{n}$-渐近正态性，或非参数分量的逐点推断，缺乏对两者收敛速率不同（$\sqrt{n}$ vs $n^{p/(2p+1)}$）的联合弱收敛分析，无法进行包含非参数分量的联合假设检验。
  - 与最相关文献的区别：相比 M-估计或核估计方法，平滑样条在 RKHS 框架下具有天然的投影正交性；相比纯参数或部分线性模型（如 Belloni et al., 2015），单指标模型中 $g(X^T\beta)$ 的非线性耦合导致参数估计误差对非参数估计的干扰更难剥离。

### 三、主要定理 / 核心结果
1. **Bahadur 表达定理**
   - 原文陈述：平滑样条估计量 $(\hat{\theta}_n, \hat{\beta}_n, \hat{g}_n)$ 可分解为独立同分布和的线性泛函加上 $o_P(n^{-1/2})$（参数部分）与 $o_P(n^{-p/(2p+1)})$（非参数部分）的余项。
   - 直观解释：将复杂的非线性半参数估计量“一阶泰勒展开”为简单算术平均，剥离出决定极限分布的随机主导项。
   - 解决的技术难点：克服了 $g$ 在估计量 $\hat{\beta}$ 处取值带来的非线性耦合，将函数空间的误差与参数空间的误差解耦。
   - 适用条件与局限：要求 $p>1$ 以保证余项足够小；若光滑度不足，余项将主导极限分布，Bahadur 展开失效。

2. **渐近独立性定理**
   - 原文陈述：在给定 $x^T\beta$ 处，$\hat{g}_n(x^T\hat{\beta}_n)$ 与 $(\hat{\theta}_n, \hat{\beta}_n)$ 渐近独立。
   - 直观解释：参数分量的估计误差主要投影在参数空间的切空间上，而非参数分量的估计误差投影在正交的函数空间（RKHS的特定子空间）上，平滑样条的惩罚结构使得两者的随机波动互不干扰。
   - 解决的技术难点：打破了“非参数估计在参数组合点取值必然引入参数估计误差”的直觉，证明了交叉协方差项在特定速率下消失。
   - 适用条件与局限：高度依赖平滑样条在 RKHS 中的全局正交投影性质；若替换为局部多项式回归，该独立性通常不成立。

3. **极小极大最优速率**
   - 原文陈述：$\hat{g}_n$ 在 $L^2$ 风险下达到极小极大最优速率 $n^{-2p/(2p+1)}$。
   - 直观解释：平滑样条在 Sobolev 空间上没有浪费统计信息，是最优自适应的。

### 四、证明框架 / 方法设计
- **证明主干逻辑**：构造法 + RKHS 算子谱分析 + 经验过程。
- **拆解为 3-5 个关键逻辑步骤**：
  1. **算子方程转化**：将惩罚最小二乘的一阶条件转化为 RKHS 中的算子方程，把 $\hat{g}_n - g_0$ 表示为经验项与参数误差 $(\hat{\beta}_n-\beta_0)$ 的泛函。
  2. **经验算子收敛**：利用经验过程理论，证明样本协方差算子向总体算子的依概率收敛，控制 RKHS 范数下的余项。
  3. **Bahadur 余项控制**：通过 chaining 技术与矩不等式，证明非线性余项在特定模下为 $o_P(n^{-1/2})$ 或 $o_P(n^{-p/(2p+1)})$，完成线性化。
  4. **协方差结构解析**：计算 Bahadur 主导项的协方差，利用 RKHS 中再生核与参数梯度的正交性，证明交叉协方差为 0，得到渐近独立。
  5. **Bootstrap 逼近**：构造 multiplier bootstrap 统计量，证明其条件分布能逼近原统计量的极限分布（基于 Donsker 定理与连续映射定理）。
- **最关键的技巧性引理或"跳跃点"**：
  - **RKHS 中参数误差与函数误差的解耦**：展开 $g(X^T\hat{\beta}) - g(X^T\beta)$ 时会产生 $g'(X^T\beta)X^T(\hat{\beta}-\beta)$ 项。跳跃点在于，平滑样条的惩罚结构使得 $g'$ 与 $\hat{g} - g$ 在 RKHS 的特定内积下正交，从而在计算联合极限分布的协方差时，该交叉项的高阶余项奇迹般地消失，这是渐近独立性的根源。
- **数学工具评价**：是经典工具（RKHS、经验过程、Bahadur 表达）的深度组合与精巧计算，特别是将 RKHS 的几何正交性转化为概率论中的渐近独立性，展现了半参数理论中几何与概率的深刻联系。

### 五、与研究者兴趣的关联
- **连接到哪个子方向**：半参数理论中的联合推断与假设检验。
- **可借鉴的核心思路或技术工具**：
  - 1. **Bahadur 表达在异速分量联合推断中的应用**：在处理收敛速率不同的参数与非参数分量（如因果推断中的连续处理效应函数与混杂参数）时，Bahadur 展开是获取联合极限分布的唯一路径，可迁移到 Proximal CI 中对 bridge function 的联合推断。
  - 2. **Multiplier Bootstrap 规避复杂渐近量**：在半参数模型中，有效影响函数的方差或非参数分量的长程协方差极难解析估计，本文基于 Bahadur 展开构造 multiplier bootstrap 的策略，可直接用于构造因果效应函数的同时置信带或检验处理效应是否超过某阈值。
- **值得精读的关键参考文献**：
  - 1. Mammen & van de Geer (1997) "Penalized quasi-likelihood and the computation of smoothing splines"：RKHS 在半参数推断中的奠基性工作，理解本文算子方程与投影的基础。
  - 2. Belloni, Chernozhukov & Kato (2015) "Some inference on partially linear models"：对比部分线性模型下的联合推断，理解单指标模型非线性耦合带来的额外技术难度与独立性差异。

### 六、延伸思考与练习
- **假设扰动**：若将平滑样条替换为局部多项式回归，渐近独立性是否依然成立？直觉上，核估计缺乏全局的正交投影结构，交叉项 $g'(\cdot)(\hat{\beta}-\beta_0)$ 可能无法消除。若要挽救，是否需要引入 Neyman orthogonality 构造正交矩条件来消除参数误差对非参数估计的一阶影响？
- **开放问题**：如何将此联合推断框架推广到高维单指标模型（$p \gg n$）？在 Lasso 等变量选择后，如何构造参数分量与非参数分量的同时置信带（Debiased SCB）？
- **理解检测题**：在部分线性单指标模型 $Y = \theta Z + g(X^T\beta) + \epsilon$ 中，若我们要检验 $H_0: g \equiv 0$（即模型退化为纯线性模型），请基于本文的渐近独立性结果与 Bahadur 表达，构造一个检验统计量，并说明其零分布如何通过 multiplier bootstrap 获得。
