# Environment invariant linear least squares

**作者**: Jianqing Fan, Cong Fang, Yihong Gu, Tong Zhang  
**来源**: Annals of Statistics  
**主题**: 因果推断  
**相关性**: 9/10  
**链接**: <https://doi.org/10.1214/24-aos2435>

---

### 一、核心问题与贡献
①研究了多环境线性回归中存在内生性/伪变量时，如何仅依赖条件期望不变性与环境异质性识别真实因果参数与重要变量集的问题。②核心方法是构建融合预测误差与跨环境不变性惩罚的EILLS目标函数，及其在高维设定下的$\ell_0$惩罚扩展。③主要贡献是在近最小识别条件下证明了EILLS估计量的非渐近$\ell_2$误差界，并证明了高维$\ell_0$-EILLS的变量选择一致性，首次在一般线性模型中实现了无需先验结构知识或工具变量的统计有效invariance learning。

### 二、基础设定
- **核心概念与符号**：
  - $E$：环境集合，$e \in E$，多环境数据 $\{(y^e, X^e)\}$。
  - $S$：重要变量集（直接原因），$S^c$：伪变量集。
  - $\beta^*$：真实参数，支撑集为 $S$。
  - Invariance：$E^e[y|X_S] = X_S\beta^*$ 对所有 $e$ 成立。
  - EILLS目标函数：$L(\beta) = \sum_e w_e E^e[(y^e - X^e\beta)^2] + \lambda \sum_e w_e \|E^e[X^e(y^e - X^e\beta)]\|_2^2$（预测损失 + 不变性惩罚）。
- **关键假设**：
  1. **条件期望不变性**：$E^e[y|X_S] = X_S\beta^*$。统计学含义：因果机制的稳定性，允许噪声分布与协变量分布随环境变化。相比已有文献（如ICP要求全条件分布不变性），此假设大幅放宽。
  2. **近最小识别条件**：环境间的异质性足以区分不变变量与伪变量。具体而言，伪变量与残差在不同环境中的协方差结构存在足够变异。统计学含义：这是可识别性的边界条件，若伪变量与残差的相关性在所有环境中完全相同，则等价于单环境内生性，不可识别。
  3. **线性内生性**：$E^e[\epsilon^e|X^e_{S^c}] \neq 0$。允许伪变量与误差项相关，打破了传统OLS的外生性假设。
- **问题背景**：
  传统因果推断处理内生性依赖工具变量（IV），但有效IV极难获取；现有invariance learning（如ICP）依赖强分布不变性且缺乏高维下的有限样本保证，常退化为保守的集合估计。本文与最相关文献的区别：相比 Peters et al. (2016) 的ICP，本文放宽了不变性假设并提供了严苛的非渐近误差界；相比 Rothenhäusler et al. (2021) 的Causal Dantzig，本文不要求特定的干预结构知识，且建立了高维变量选择一致性。

### 三、主要定理 / 核心结果
1. **定理：EILLS估计量的非渐近$\ell_2$误差界**
   - **原文陈述**：在近最小识别条件下，EILLS估计量 $\hat{\beta}$ 满足 $\|\hat{\beta} - \beta^*\|_2 \le O_P\left(\sqrt{\frac{s \log p}{n}}\right)$（$s=|S|$），且伪变量对应的系数估计误差被严格控制。
   - **直观解释**：即使存在内生性，只要环境间异质性足够，EILLS能将因果参数的估计误差收敛到仅依赖重要变量维数的Oracle速率，伪变量的内生干扰被不变性惩罚项彻底滤除。
   - **技术难点**：在内生性下，OLS的Hessian矩阵非正定，传统M估计理论失效。本文通过不变性惩罚重构了目标函数的曲率，证明了在伪变量空间上的强凸性。
   - **适用条件与局限**：必须满足近最小识别条件；若环境异质性极弱（接近单环境），惩罚项无法提供足够曲率，误差界会退化。目前局限于线性模型。

2. **定理：高维$\ell_0$-EILLS的变量选择一致性**
   - **原文陈述**：$\ell_0$惩罚的EILLS估计量在高维设定下能以高概率精确恢复重要变量集 $S$。
   - **直观解释**：通过$\ell_0$惩罚强制稀疏，结合EILLS的不变性约束，算法不仅能估计参数，还能准确挑选出因果变量，剔除伪变量。
   - **技术难点**：$\ell_0$惩罚导致非凸优化，需证明局部极小值不干扰变量选择，且需处理离散支撑集上的浓度不等式。
   - **适用条件与局限**：需要最小信号强度条件（$\min_{j\in S}|\beta^*_j|$不能太小）及一定的环境异质性强度。

### 四、证明框架 / 方法设计
- **证明主干逻辑**：构造法 + 矩方法 + 局部曲率分析。
- **关键逻辑步骤**：
  1. **目标函数拆解**：将EILLS目标函数在真实参数 $\beta^*$ 处泰勒展开，分离出重要变量空间的预测误差项与伪变量空间的不变性惩罚项。
  2. **异质性矩阵的正定性证明**：利用近最小识别条件，证明由环境异质性构成的二阶导数矩阵（修正的Hessian矩阵）在伪变量子空间上是正定的，这是克服内生性导致曲率缺失的核心。
  3. **一阶项的浓度不等式**：利用经验过程的矩不等式，控制随机误差与协变量乘积在跨环境下的波动。
  4. **变量选择一致性推导**：对 $\ell_0$ 惩罚目标，证明真实支撑集对应的子空间上目标函数值严格小于任何包含伪变量的支撑集，利用反证法排除过拟合与欠拟合。
- **最关键的技巧性引理/跳跃点**：**异质性矩阵在伪变量空间上的正定性分析**。在内生性下，$X_{S^c}$ 与 $\epsilon$ 相关，传统Hessian矩阵 $E[X_{S^c}^T X_{S^c}]$ 无法隔离内生干扰。本文通过引入跨环境的协方差差异，构造了类似 Causal Dantzig 的曲率矩阵，证明了该矩阵在 $S^c$ 子空间上的限制严格大于0，从而将因果识别问题转化为目标函数的几何曲率问题。
- **数学工具评价**：是经典的高维M估计浓度不等式与因果推断invariance思想的巧妙组合，将因果识别的代数条件无缝嵌入到非渐近统计理论中。

### 五、与研究者兴趣的关联
- **连接子方向**：因果推断中的内生性处理（无需IV的识别路径）与高维变量选择。
- **可借鉴思路**：将因果不变性假设转化为目标函数中的惩罚项（moment condition penalty），从而将因果识别与高维M估计理论统一。这种"将因果结构假设写成惩罚项"的思路可直接迁移到 **proximal CI** 设定中：若将 negative control 的矩条件（如 $E[Y|W, X]$ 的独立性）写成类似 EILLS 的跨环境/跨工具变量惩罚项，可能构造出无需参数设定的 proximal 最小最大估计量。
- **值得精读的参考文献**：
  1. Peters, Janzing, Schölkopf (2016) - *Causal inference using invariant prediction*。ICP的奠基作，理解invariance principle如何从分布不变性退化为条件期望不变性的起点。
  2. Rothenhäusler et al. (2021) - *Causal Dantzig*。对比其干预假设与本文的异质性假设，理解多环境回归中不同曲率构造的等价性与差异。

### 六、延伸思考与练习
- **假设扰动**：若将"条件期望不变性"放宽为"条件分位数不变性"（如中位数不变），EILLS的惩罚项应如何修改？此时二阶导数矩阵的曲率分析将面临什么挑战？（提示：需要从平滑的二次矩转向非平滑的检查函数，技术上需引入经验过程的 bracketing entropy）。
- **开放问题**：如何在半参数模型（如部分线性模型 $y = X_S\beta^* + g(Z) + \epsilon$）中构造类似的 EILLS 估计量，并在不指定 $g(\cdot)$ 形式的情况下建立非渐近误差界？
- **理解检测题**：假设存在两个环境，伪变量 $X_{S^c}$ 在两个环境中的分布完全相同，且与误差项 $\epsilon$ 的相关性也完全相同（即 $E^1[X_{S^c}\epsilon] = E^2[X_{S^c}\epsilon]$）。请从 EILLS 目标函数的二阶导数矩阵（曲率）角度，证明此时 EILLS 无法识别真实参数 $\beta^*$。
