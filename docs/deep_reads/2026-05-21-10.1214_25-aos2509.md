# Semiparametric adaptive estimation under informative sampling

**作者**: Kosuke Morikawa, Yoshikazu Terada, Jae Kwang Kim  
**来源**: Annals of Statistics  
**主题**: 效率理论 / Debiased ML  
**相关性**: 10/10  
**链接**: <https://doi.org/10.1214/25-aos2509>

---

### 一、核心问题与贡献
①研究了在 informative sampling（信息性抽样）导致样本选择偏差的设定下，如何对总体目标参数进行半参数最优估计的问题。②核心方法是将抽样权重视为随机变量推导半参数有效界，并基于抽样权重的 working model 构造自适应估计量。③证明了所提估计量在正则渐近线性（RAL）估计类中具有相合性、渐近正态性，且在 working model 正确时达到半参数有效界，弥补了传统 Horvitz-Thompson 估计量非有效的缺陷。

### 二、基础设定
- **核心概念与符号**：
  - $Y$：目标变量（未入样时不可见）
  - $X$：协变量（始终可见）
  - $\delta$：选择指标（$\delta=1$ 表示入样）
  - $\pi(Y,X)$：入样概率（倾向得分），$W = 1/\pi$ 为抽样权重
  - $O = (\delta, \delta Y, X, W)$：观测数据分布
  - $\theta$：目标参数（如 $E(Y)$ 或回归系数）
  - $S_{\text{eff}}$：有效得分函数

- **关键假设**：
  - **Informative sampling（非忽略抽样）**：$\pi$ 依赖于 $Y$，即 $P(\delta=1|Y,X) \neq P(\delta=1|X)$。统计学含义是打破了缺失数据中的 MAR 假设，导致基于完全数据的似然函数失效，是产生选择偏差的根源。与标准缺失数据文献相比，本文明确将 $W$ 纳入观测数据分布而非视为固定常数。
  - **Positivity**：$\pi > 0$ a.s.。统计学含义是保证权重有界，避免方差爆炸，是 RAL 估计量存在的必要条件。
  - **Working model on weights**：对 $W$ 的条件分布（如 $E(W|Y,X)$）建立参数模型。与已有文献相比，本文不要求此模型正确指定，模型正确时达到有效界，错误时退化为稳健但非有效的 RAL 估计量，实现了自适应。

- **问题背景**：传统 Horvitz-Thompson 估计量虽无偏相合，但仅使用权重而未利用 $X$ 与 $Y$ 的联合信息，渐近方差非最优。与 Pfeffermann & Sverchkov (1999) 等基于模型的方法相比，本文基于半参数理论提供了更严谨的效率界与最优性保证；与 Rotnitzky & Robins (1997) 在缺失数据下的半参数界推导相比，本文聚焦于 $W$ 的随机性及其 working model 的自适应构造。

### 三、主要定理 / 核心结果

1. **定理1：半参数有效界推导**
   - **原文陈述**：在 informative sampling 下，目标参数 $\theta$ 的半参数有效界为 $V^* = \{E(S_{\text{eff}} S_{\text{eff}}^T)\}^{-1}$，其中 $S_{\text{eff}}$ 是 $\theta$ 的得分函数在观测数据 $O$ 的无参数切空间上的投影。
   - **直观解释**：由于 informative sampling 导致 $Y$ 和 $\delta$ 不独立，有效得分必须扣除由抽样机制引入的冗余信息。有效界量化了在只观测到 $(\delta, \delta Y, X, W)$ 且不假设完整分布时，估计 $\theta$ 所能到达的最小渐近方差。
   - **解决了什么技术难点**：解决了在非忽略抽样下，如何将 $W$ 的随机性纳入切空间计算，并分离出目标参数的得分与冗余参数得分的正交分量。
   - **适用条件与局限**：需要 $\pi$ 有界远离0；若 working model 错误，有效界可能无法达到，但估计量仍相合。

2. **定理2：自适应估计量的渐近性质**
   - **原文陈述**：基于 working model 构造的估计量 $\hat{\theta}$ 满足 $\sqrt{n}(\hat{\theta} - \theta) \to N(0, V^*)$，当且仅当 working model 正确时达到有效界；若 working model 错误，渐近方差为 $V_{\text{wrong}} \ge V^*$。
   - **直观解释**：通过估计 $W$ 的条件期望（working model），构造类似于 AIPW 的项，消除一阶偏差。当 working model 正确时，二阶项消失，达到最小方差；错误时退化为稳健但非有效的估计量。
   - **解决了什么技术难点**：在非参数/半参数空间中，如何利用 working model 构造得分函数使其在模型正确时切于有效切空间，错误时仍在 RAL 类中。
   - **适用条件与局限**：依赖于 nuisance 参数的 $\sqrt{n}$ 相合估计或使用交叉拟合以避免过拟合偏差。

### 四、证明框架 / 方法设计

- **证明主干逻辑**：构造法 + 切空间投影法 + 渐近线性展开。
- **拆解为 3-5 个关键逻辑步骤**：
  1. **切空间分解**：定义观测数据 $O$ 的无参数切空间 $\mathbb{T}$，将其分解为目标参数 $\theta$ 的得分空间、抽样机制（$\delta, W$）的得分空间与条件期望的得分空间。
  2. **有效得分计算**：将 $\theta$ 的得分函数向 $\mathbb{T}$ 中冗余方向的正交补投影，得到有效得分 $S_{\text{eff}}$。关键在于利用 $W$ 的条件期望性质（如 $E(W-1|Y,X)=0$）简化投影。
  3. **构造影响函数**：基于 $S_{\text{eff}}$ 和 working model 构造估计方程，使得影响函数在 working model 正确时等于有效影响函数。
  4. **渐近性质验证**：验证所提估计量满足 RAL 条件，通过 Taylor 展开和经验过程理论证明渐近正态性，并计算渐近方差。
- **最关键的技巧性引理或"跳跃点"**：利用 $E(W|Y,X)$ 的性质将包含 $W$ 的项转化为可估的 nuisance 参数。在 informative sampling 下，$E(W|Y,X)$ 的引入类似于 AIPW 中的结果模型和倾向得分模型，这里的"双重稳健"结构是跳跃点：即使对 $W$ 的 working model 错误，只要 $\theta$ 的模型正确，估计量仍相合。
- **数学工具评价**：是经典的半参数效率理论（Bickel et al., 1993; Tsiatis, 2006）在 informative sampling 特定缺失结构下的巧妙应用与扩展，核心创新在于将 $W$ 的随机性纳入切空间并实现自适应。

### 五、与研究者兴趣的关联
- **连接到哪个子方向**：半参数有效界 与缺失数据/选择偏差下的自适应估计。
- **可借鉴的核心思路或技术工具**：将 informative sampling 视为一种非随机缺失，推导有效得分时处理 $W$ 的随机性的投影技巧；构造基于 working model 的自适应估计量，实现"模型正确时有效，错误时稳健"的双重性质。这可以直接迁移到 proximal causal inference 中处理 unmeasured confounding 时的效率界推导与估计，特别是当 negative control 变量的分布类似于抽样权重时的处理。
- **值得精读的关键参考文献**：
  1. Rotnitzky & Robins (1997) Semiparametric regression estimation in the presence of dependent censoring.（缺失数据下半参数界的奠基之作，对比理解 informative sampling 与 dependent censoring 的异同）
  2. Robins, Rotnitzky & Zhao (1994) Estimation of regression coefficients when some regressors are not always observed.（AIPW 与双重稳健的起源，理解 working model 错误时的稳健性保障）
  3. Bickel et al. (1993) Efficient and adaptive estimation for semiparametric models.（半参数有效界与切空间投影的标准教材）

### 六、延伸思考与练习
- **假设扰动**：若放宽 Positivity 假设，即 $\pi$ 可以趋近于 0（极端 informative sampling），结论会如何变化？技术上需要引入正则化或高维倾向得分模型，渐近分布可能不再正态，需要极值理论或偏差-方差权衡的新工具。
- **开放问题**：当协变量 $X$ 维度较高时，如何结合 DML 或 debiased 方法对 $E(W|Y,X)$ 等 nuisance 参数进行估计，以避免过拟合导致的效率损失？
- **理解检测题**：在本文的 informative sampling 设定下，假设我们错误指定了 $W$ 的 working model，但正确指定了 $Y$ 的条件期望模型。请说明为什么此时估计量仍然相合，并指出其渐近方差与有效界 $V^*$ 之间的定量关系（是变大还是相等？为什么？）。
