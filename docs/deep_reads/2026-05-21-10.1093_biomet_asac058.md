# On the implied weights of linear regression for causal inference

**作者**: Ambarish Chattopadhyay, José R Zubizarreta  
**来源**: Biometrika  
**主题**: 因果推断  
**相关性**: 8/10  
**链接**: <https://doi.org/10.1093/biomet/asac058>

---

### 一、核心问题与贡献
①研究了观察性研究中线性回归调整如何通过隐含个体权重近似随机化实验的协变量平衡与目标总体特征。②推导了多种线性回归隐含权重的闭式表达，并证明其等价于求解带平衡约束的凸优化问题。③在有限样本下刻画了回归隐含的目标总体，大样本下从权重视角证明了回归估计量的多重稳健性，并基于凸优化等价性提出了设计阶段的回归诊断方法。

### 二、基础设定
- **核心概念与符号**：
  - $Y_i, A_i, X_i$：个体的结局、处理分配与协变量向量。
  - $\hat{\tau}$：回归估计量（如 OLS, WLS, 交互项回归等）。
  - $w_i$：隐含权重，即回归估计量可表为 $\hat{\tau} = \sum_{i=1}^n w_i Y_i$。
  - Implied target population：隐含目标总体，由权重的协变量矩 $\sum w_i X_i$ 刻画。
- **关键假设**：
  - **Unconfoundedness ($Y(a) \perp A | X$)**：无混淆性，因果推断的基础识别假设，本文在此框架下拆解回归权重。
  - **Linearity / Constant treatment effect**：部分有限样本权重闭式解依赖线性条件期望或恒定处理效应假设，大样本多重稳健性分析中放宽了此假设。
  - **Overlap/Positivity ($0 < P(A=1|X) < 1)$**：倾向得分有界，保证权重的非极端性。
  - *假设对比*：传统回归诊断关注残差与拟合优度（假设模型正确），本文关注协变量平衡与目标总体代表性（允许模型错判）；相比设计阶段方法（如匹配），本文将回归从"黑盒估计"还原为"带约束加权"，强化了其对模型错判的鲁棒性假设。
- **问题背景**：传统线性回归在因果推断中常被视为参数模型，其如何实现协变量平衡、隐含推断何种总体往往不透明。与 Angrist & Pischke 或 Lin (2013) 关注回归系数的渐进性质不同，本文对标 Rosenbaum, Rubin 等人的设计学派思想，将回归还原为加权平衡问题，弥补了回归建模与设计推断之间的理论鸿沟。

### 三、主要定理 / 核心结果
1. **有限样本下的隐含权重与目标总体刻画**
   - **原文陈述**：对 OLS、WLS 及含交互项的回归，推导出 $w_i$ 的闭式表达，并证明 $\sum_{i:A_i=1} w_i X_i = \sum_{i:A_i=0} w_i X_i$，该公共矩刻画了回归隐含的目标总体。
   - **直观解释**：回归并非简单拟合条件期望，而是隐式构造了一个加权的伪总体，强制处理组和对照组在该伪总体下协变量分布平衡。不同回归规格（是否带交互项）对应不同的目标总体（如带交互项的 OLS 目标总体近似全样本）。
   - **解决的技术难点**：打破了回归是"模型依赖"的刻板印象，给出了有限样本下权重的精确代数结构。
   - **适用条件与局限**：闭式解依赖线性设定；若真实模型非线性，有限样本下的平衡可能偏离真实的因果目标总体。

2. **大样本下的多重稳健性**
   - **原文陈述**：在特定回归设定下（如带交互项的 OLS），隐含权重的极限形式使得估计量在以下模型至少两个正确时具有一致性：(1)结果模型，(2)处理模型（倾向得分），(3)协变量平衡约束。
   - **直观解释**：从权重视角看，回归估计量之所以稳健，是因为其隐含权重同时试图拟合结果与平衡协变量。即使结果模型设定错误，权重的平衡机制仍能保证协变量分布相似，从而消除混淆偏倚。
   - **解决的技术难点**：从构造机理（而非单纯的渐近展开）解释了回归估计量的多重稳健性。
   - **适用条件与局限**：需要重叠性假设；多重稳健性依赖于特定的回归规格（如 Lin 估计量），简单 OLS 不具备此性质。

3. **隐含权重与凸优化的等价性**
   - **原文陈述**：一般回归方法的隐含权重等价于求解如下凸优化问题：$\min_w \sum w_i^2 / p_i$（或类似离散度指标），约束为精确的协变量平衡 $\sum_{i:A_i=1} w_i X_i = \sum_{i:A_i=0} w_i X_i$ 及归一化约束。
   - **直观解释**：OLS 本质上是在寻找一组方差最小（最平滑）的权重，同时强制实现协变量在一阶矩上的精确平衡。这打通了回归（估计）与设计（平衡约束）的壁垒。
   - **解决的技术难点**：建立了回归与最优平衡权重（如 Zubizarreta 2015 的设计方法）的严格数学等价。
   - **适用条件与局限**：等价性依赖于线性矩约束；若引入非线性平衡约束，则超出 OLS 框架。

### 四、证明框架 / 方法设计
- **证明主干逻辑**：构造法 + 矩条件 + 凸优化对偶性。
- **拆解为 3-5 个关键逻辑步骤**：
  1. **权重代数提取**：将回归估计量 $\hat{\tau}$ 重写为 $\sum w_i Y_i$，利用投影矩阵（Hat matrix）与残差的正交性，解出 $w_i$ 关于 $X_i, A_i$ 的显式表达式。
  2. **目标总体代数验证**：将闭式权重代入 $\sum w_i X_i$，利用分块矩阵求逆或 Frisch-Waugh-Lovell 定理，证明处理组与对照组的加权协变量矩相等，识别出目标总体。
  3. **大样本极限展开**：将有限样本权重进行泰勒展开或求概率极限，分离出倾向得分与结果模型的渐近贡献，证明当部分模型错判时平衡条件依然渐进成立。
  4. **凸优化对偶构造**：构造带线性等式约束的二次规划问题，写出拉格朗日函数，利用 KKT 条件证明其最优解的代数形式与 OLS 隐含权重完全一致。
- **最关键的技巧性引理或"跳跃点"**：利用 KKT 条件证明凸优化与 OLS 权重的等价性。这一跳跃将一个看似纯粹的统计估计问题，重构为一个带约束的工程/运筹优化问题，是全文理论桥梁的基石。
- **数学工具评价**：经典线性代数（投影矩阵与 FWL 定理）与凸优化（KKT 条件）的巧妙组合。并非全新分析框架，但视角转换极具启发性，将统计估计还原为优化与平衡。

### 五、与研究者兴趣的关联
- **连接到哪个子方向**：因果推断中的协变量平衡与目标总体推断，特别是回归估计与设计方法的统一。
- **可借鉴的核心思路或技术工具**：
  - 将估计量写为线性组合 $\sum w_i Y_i$ 并分析其隐含权重的思路，可直接迁移至高维推断（如 Debiased ML 中的隐含权重分析）或半参数估计（如 AIPW 的隐含权重结构），以审视其平衡与稳健机制。
  - 凸优化等价性思路可用于构造新估计量：在因果推断中，可自定义目标函数（如加入稳健性惩罚）与平衡约束，通过求解优化问题直接设计具有特定性质的估计量，而非依赖传统回归规格。
- **值得精读的关键参考文献**：
  1. Zubizarreta, J. R. (2015). "Stable weights that balance covariates for estimation with incomplete outcome data." *JASA*. 本文凸优化视角的直接来源，了解设计阶段平衡权重的经典文献。
  2. Lin, W. (2013). "Agnostic notes on regression adjustments to experimental data: Revisiting Freedman's critique." *Annals of Applied Statistics*. 回归调整在实验中的隐含加权与稳健性分析，是本文理论的重要对照与基础。

### 六、延伸思考与练习
- **假设扰动**：若将线性回归替换为非参数回归（如核回归或 Series estimation），隐含权重的闭式解不再存在。如何近似刻画其隐含目标总体？技术上可能需要引入经验过程理论或局部多项式权重的渐近分析，平衡约束将从一阶矩扩展到函数空间。
- **开放问题**：如何将隐含权重的凸优化等价性推广到带有工具变量（IV）的设定中？2SLS 的隐含权重是否等价于某种带工具变量平衡约束（如协变量与残差正交）的优化问题？
- **理解检测题**：假设在 OLS 回归中加入了处理与协变量的交互项（Lin 估计量），请推导其隐含权重 $w_i$ 的形式，并说明相比于无交互项的 OLS，其隐含目标总体发生了什么变化，为什么这有助于提高对异质性处理效应的稳健性？
