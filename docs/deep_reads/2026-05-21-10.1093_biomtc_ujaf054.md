# Double robust variance estimation with parametric working models

**作者**: Bonnie E Shook-Sa, Paul N Zivich, Chanhwa Lee, Keyi Xue, Rachael K Ross et al.  
**来源**: Biometrics  
**主题**: 因果推断  
**相关性**: 9/10  
**链接**: <https://doi.org/10.1093/biomtc/ujaf054>

---

### 一、核心问题与贡献
①研究了非随机化暴露下双重稳健（DR）平均因果效应估计量的方差估计稳健性问题。②核心工具是基于M估计理论的经验三明治方差估计量与非参数Bootstrap。③证明了在参数工作模型下，经验三明治方差与Bootstrap方差具有双重稳健性（仅需一个模型正确即可提供一致方差估计与名义覆盖率），而传统基于影响函数的方差估计因忽略干扰参数估计的变异性而在单一模型正确时失效。

### 二、基础设定
- **核心概念与符号**：
  - $O = (W, A, Y)$：观测数据（协变量、暴露、结果）。
  - $\psi$：目标因果参数（ATE）。
  - $\pi(W; \beta)$：倾向得分工作模型（暴露机制），参数 $\beta$。
  - $\mu(W; \alpha)$：结果条件期望工作模型，参数 $\alpha$。
  - $\phi(O; \psi, \alpha, \beta)$：DR估计方程（如AIPW的估计函数）。
  - $\Sigma_{IF}$：基于影响函数的方差估计量，仅计算 $\phi$ 的经验方差。
  - $\Sigma_{Sandwich}$：经验三明治方差估计量，基于联合估计方程的M估计理论。
- **关键假设**：
  - *Ignorability / Exchangeability*：$Y^a \perp\!\!\!\perp A \mid W$，无未测量混杂。
  - *Positivity*：$0 < P(A=1 \mid W) < 1$ 几乎必然成立。
  - *Parametric Working Models*：$\pi$ 和 $\mu$ 属于参数模型族，允许误判。
  - *Doubly Robust Consistency*：若 $\pi$ 或 $\mu$ 至少一个正确设定，$\hat{\psi}$ 依概率收敛于真值 $\psi_0$。
  - *假设对比*：传统文献认为DR估计量的有效推断需双模型正确（保证IF为真实IC），本文放宽了方差一致性的条件，证明单模型正确下三明治方差仍一致。
- **问题背景**：DR点估计具有稳健性，但传统IF方差估计破坏了这种稳健性（需双模型正确才一致）。与最相关文献（如Rotnitzky & Robins 2001对误判模型下渐近分布的讨论）的区别在于：本文明确将DR估计方程嵌入M估计框架，系统推导并证明了经验三明治与Bootstrap在*方差层面*的双重稳健性，澄清了干扰参数估计变异性对推断的决定性影响。

### 三、主要定理 / 核心结果
1. **原文陈述**：设 $\hat{\theta} = (\hat{\psi}, \hat{\alpha}, \hat{\beta})$ 为联合M估计量，若 $\pi(W;\beta)$ 或 $\mu(W;\alpha)$ 至少一个正确设定，则经验三明治方差估计量 $\hat{\Sigma}_{Sandwich} = \hat{A}^{-1}\hat{B}\hat{A}^{-\top}$ 依概率收敛于 $\sqrt{n}(\hat{\psi}-\psi_0)$ 的真实渐近方差；而 $\hat{\Sigma}_{IF}$ 在仅有一个模型正确时收敛于错误值。
2. **直观解释**：当模型误判时，估计方程的期望不为零（非中心化），且干扰参数 $(\hat{\alpha}, \hat{\beta})$ 的抽样波动会传导给 $\hat{\psi}$。IF方差仅计算了估计方程的边际方差，忽略了干扰参数变异性带来的额外修正项。三明治方差通过 $A^{-1}$ 矩阵（Sensitivity matrix）自动捕捉了干扰参数估计波动对 $\hat{\psi}$ 的放大/衰减效应，从而还原了真实的渐近方差。
3. **解决了什么技术难点**：澄清了DR估计量在模型误判下的渐近分布结构，特别是量化了干扰参数估计对极限分布的贡献，打破了"DR推断必须依赖双模型正确"的定势。
4. **适用条件与局限**：必须假设参数工作模型。若使用半参数/非参数工作模型（如DML中的机器学习），三明治方差中 $A$ 矩阵的估计及理论性质需要重新审视（收敛速度和交叉拟合项的消除）。Bootstrap在高维/非参数设定下的理论保证可能失效。

### 四、证明框架 / 方法设计
- **证明主干逻辑**：M估计/估计方程的渐近理论。
- **拆解为 3-5 个关键逻辑步骤**：
  1. **构造联合估计方程**：将目标参数 $\psi$ 与干扰参数 $\alpha, \beta$ 的估计方程堆叠，形成联合M估计方程 $U_n(\theta) = \sum \phi(O_i; \theta) / n = 0$。
  2. **一阶泰勒展开**：在真实极限值 $\theta^* = (\psi_0, \alpha^*, \beta^*)$ 处对 $U_n(\hat{\theta})$ 展开，得到 $\sqrt{n}(\hat{\theta} - \theta^*) \approx - \dot{U}(\theta^*)^{-1} \sqrt{n} U_n(\theta^*)$。
  3. **计算渐近方差**：基于展开式，$\hat{\theta}$ 的渐近方差为三明治形式 $A^{-1} B \{A^{-1}\}^\top$，其中 $A = E[\dot{\phi}(\theta^*)]$, $B = Var(\phi(\theta^*))$。
  4. **DR性质代入**：利用DR性质证明，当 $\alpha$ 或 $\beta$ 之一正确时，$\hat{\psi}$ 的渐近方差表达式退化为仅依赖于正确设定模型部分的变异性，且经验三明治估计量 $\hat{A}^{-1}\hat{B}\hat{A}^{-\top}$ 依概率收敛于此真实方差。
  5. **IF方差对比**：证明 IF 方差估计量等价于假设 $\alpha, \beta$ 已知（即 $A$ 矩阵为单位阵分块）时的方差，在模型误判时缺失了 $A^{-1}$ 带来的修正项。
- **最关键的技巧性引理或"跳跃点"**：将DR估计方程视为包含 nuisance parameters 的 M-estimation 问题。关键在于求导矩阵 $A$ 的计算。当模型误判时，$E[\partial \phi / \partial \alpha] \neq 0$ 或 $E[\partial \phi / \partial \beta] \neq 0$，这些偏导数项在 IF 方差中被忽略，但在三明治方差中被保留并恰好抵消了非中心项带来的偏差。
- **数学工具评价**：经典M估计理论的巧妙应用。将因果推断中的DR估计量无缝嵌入经典统计推断框架，揭示了经典三明治方差在因果参数推断中的隐藏稳健性。

### 五、与研究者兴趣的关联
- **连接到哪个子方向**：Doubly robust inference / semiparametric efficiency / variance estimation under misspecification (TMLE/AIPW 的方差推断)。
- **可借鉴的核心思路或技术工具**：在构造 DR 估计量（如 TMLE 或 AIPW）的置信区间时，若使用参数工作模型，应摒弃传统的 IF variance，转而使用 empirical sandwich variance 或 Bootstrap。在 DML 框架下，若使用半参数/机器学习工作模型，交叉拟合虽然消除了部分 nuisance 估计的偏差，但若模型收敛速度不够快，三明治方差的有限样本表现仍值得深究。这为研究高维/非参数 DR 估计的稳健推断提供了基准思路。
- **值得精读的关键参考文献**：
  1. Rotnitzky & Robins (2001) "Inference for semiparametric models: double robustness" - 理解模型误判下渐近分布非中心化的经典源头。
  2. Stefanski & Boos (2002) "The Calculus of M-estimation" - 理解三明治方差计算与求导矩阵提取的标准操作手册。

### 六、延伸思考与练习
- **假设扰动**：若将参数工作模型替换为半参数/非参数模型（如DML中的机器学习算法，收敛速度 $o(n^{-1/4})$），经验三明治方差的理论性质如何变化？此时求导矩阵 $A$ 的估计是否仍然有效？需要什么新工具（如交叉拟合下的经验过程界）？
- **开放问题**：在高维稀疏设定下（如Debiased Lasso构造的DR估计量），如何构造具有双重稳健性的方差估计？Bootstrap在此设定下是否仍然有效？
- **理解检测题**：假设结果模型 $\mu(W;\alpha)$ 误判，暴露模型 $\pi(W;\beta)$ 正确。请写出AIPW估计方程对 $\alpha$ 的偏导数 $\partial \phi / \partial \alpha$，并解释为什么该偏导数的期望不为零是导致 IF 方差失效、而三明治方差有效的根本数学原因。
