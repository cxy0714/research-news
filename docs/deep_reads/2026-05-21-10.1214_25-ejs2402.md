# Hoeffding-type decomposition for U-statistics on bipartite networks

**作者**: Tâm Le Minh, Sophie Donnet, François Massol, Stéphane Robin  
**来源**: Electronic Journal of Statistics  
**主题**: 其他  
**相关性**: 9/10  
**链接**: <https://doi.org/10.1214/25-ejs2402>

### 一、核心问题与贡献
①研究了行-列可交换二分网络邻接矩阵上的 U-统计量的分解与渐近推断问题。②核心工具是基于 Aldous-Hoover-Kallenberg 表示定理构造的新型 Hoeffding 型分解，将统计量正交投影到行效应、列效应及行列交互效应子空间。③主要贡献是刻画了此类结构下非退化 U-统计量的渐近正态性，并给出了自然且易于实现的渐近方差估计量。

### 二、基础设定
- **核心概念与符号**：
  - $X = (X_{ij})$：$m \times n$ 邻接矩阵，行代表一类节点，列代表另一类节点。
  - Row-column exchangeability（行-列可交换性）：矩阵分布在对行指标和列指标分别进行置换下保持不变。
  - AHK representation：$X_{ij} = f(\xi_i, \eta_j, \lambda_{ij})$，其中 $\xi_i$（行潜变量）、$\eta_j$（列潜变量）、$\lambda_{ij}$（独立噪声）相互独立。
  - $U$-statistic：$U = \binom{m}{k}^{-1}\binom{n}{l}^{-1} \sum U_{I,J}$，核函数阶数为 $(k,l)$。
  - Hoeffding 分解：$U = \mu + U_R + U_C + U_{RC}$，分别代表总体均值、行主效应、列主效应与交互效应。
- **关键假设**：
  - **Row-column exchangeability**：比传统 i.i.d. 假设弱，允许行列内部存在依赖，但要求置换不变性。相比一般图上的 vertex exchangeability，此假设利用了二分图的双模态结构。
  - **AHK representation 存在性**：由 Kallenberg 表示定理保证，是本文分解的基石，将网络依赖解耦为潜变量的独立结构。
  - **非退化性条件**：$\text{Var}(U_R) = o(1/m)$ 且 $\text{Var}(U_C) = o(1/n)$，交互项方差占主导。
- **问题背景**：传统 Hoeffding 分解针对 i.i.d. 样本，网络数据存在内在依赖导致经典投影失效。已有网络 U-统计量研究多针对单部图，二分图的双模态依赖结构导致传统投影空间不适用。与最相关文献（如单部图可交换矩阵的 U-统计量研究）的区别在于：本文利用二分图的 AHK 表示，给出更精细的行-列-交互三向正交分解，解决了方差估计的不可识别问题。

### 三、主要定理 / 核心结果
1. **Hoeffding 型分解定理**
   - **原文陈述**：任意行-列可交换矩阵上的 U-统计量 $U$ 可唯一分解为 $U = \mu + U_R + U_C + U_{RC}$，其中 $U_R$ 仅依赖行潜变量 $\xi$，$U_C$ 仅依赖列潜变量 $\eta$，$U_{RC}$ 为交互项，且三者两两正交。
   - **直观解释**：将 U-统计量的变差分解为行异质性贡献、列异质性贡献和纯交互（残差）贡献。几何上，这是向 $\sigma(\xi), \sigma(\eta), \sigma(\xi,\eta)$ 生成的 $L^2$ 子空间的正交投影。
   - **解决了什么技术难点**：在非 i.i.d. 且具有双模态潜变量结构下定义了保持无偏性与正交性的投影算子。
   - **适用条件与局限**：依赖 AHK 表示定理，仅适用于行-列可交换矩阵。若网络存在度相关性（如 preferential attachment），此假设失效。

2. **渐近正态性与方差估计定理**
   - **原文陈述**：若 $U$ 是非退化的，则 $\sqrt{mn}(U - \mu) \xrightarrow{d} \mathcal{N}(0, \sigma^2_{RC})$。且基于样本行/列均值的投影可构造出 $\sigma^2_{RC}$ 的一致估计量 $\hat{\sigma}^2_{RC}$。
   - **直观解释**：当行列边际效应的方差随 $m,n \to \infty$ 衰减足够快时，U-统计量的波动主要由交互项驱动，中心极限定理成立。
   - **解决了什么技术难点**：解决了网络依赖数据下 U-统计量渐近方差的估计难题，传统 jackknife/bootstrap 在此结构下失效或计算不可行。
   - **适用条件与局限**：非退化条件是核心，若不满足则渐近分布非正态（可能是混合正态），此时方差估计量不再收敛于目标方差。

### 四、证明框架 / 方法设计
- **证明主干逻辑**：构造法 + 条件期望投影 + 矩量法。
- **拆解为 4 个关键逻辑步骤**：
  1. **AHK 代换**：将邻接矩阵元素 $X_{ij}$ 替换为 $f(\xi_i, \eta_j, \lambda_{ij})$，将网络依赖转化为潜变量的独立结构。
  2. **正交投影构造**：定义条件期望算子 $P_R = \mathbb{E}[\cdot | \xi]$，$P_C = \mathbb{E}[\cdot | \eta]$，通过类似方差分析的加减法构造 $U_R, U_C, U_{RC}$，验证其正交性。
  3. **方差阶的刻画**：计算 $\text{Var}(U_R), \text{Var}(U_C), \text{Var}(U_{RC})$ 的阶，证明在非退化条件下，行/列项的方差阶低于交互项。
  4. **交互项的 CLT**：利用 $\lambda_{ij}$ 的条件独立性，对 $U_{RC}$ 应用经典 U-统计量的 Hoeffding 投影法证明渐近正态性。
- **最关键的技巧性引理或"跳跃点"**：利用 AHK 表示将网络 U-统计量转化为关于独立潜变量的泛函。这一步将复杂的图依赖结构"解耦"，使得传统的基于独立性的投影理论得以应用。
- **数学工具评价**：是经典 Hoeffding 分解思想与 AHK 表示定理的巧妙组合。并非全新分析框架，但在二分图特定结构下对投影空间的划分（行、列、交互）极具针对性，解决了方差估计不可识别的问题。

### 五、与研究者兴趣的关联
- **连接到哪个子方向**：higher-order U-statistics 的分解与渐近理论，特别是非 i.i.d. 依赖结构（网络/阵列数据）下的 U-统计量推断。
- **可借鉴的核心思路或技术工具**：将 AHK 表示定理作为"解耦"工具，把复杂依赖数据转化为潜变量空间上的独立结构，进而定义正交投影和 Hoeffding 分解。这一思路可迁移至其他具有潜变量可交换结构的复杂数据（如多路阵列、带协变量的网络）的 U-统计量高阶推断中。此外，其方差估计量的构造方法（基于样本行/列均值的投影）可直接用于构造网络参数的置信区间。
- **值得精读的关键参考文献**：
  1. Kallenberg (2005) *Probabilistic Symmetries and Invariance Principles*：理解 AHK 表示定理的严格数学基础，是本文分解的基石。
  2. Lee (1990) *U-Statistics*：经典 U-统计量分解与渐近理论的基准，对比理解本文如何推广投影空间。
  3. Lundquist et al. (2017) *U-statistics for partially observed networks*：了解单部图/一般图上的 U-统计量处理方式，对比二分图结构的特殊性。

### 六、延伸思考与练习
- **假设扰动**：若放宽"行-列可交换性"假设，例如允许行节点具有异质性（不再同分布），AHK 表示不再成立，此时分解会如何变化？技术上可能需要引入条件独立图（CI graph）或更一般的鞅差分序列工具来刻画投影，方差估计可能不再具有显式解。
- **开放问题**：如何将此分解推广到动态二分网络（时间序列网络）或多层二分网络？此时 AHK 表示需引入时间/层维度的潜变量，交互项的渐近性质可能依赖于时间序列的混合速度。
- **理解检测题**：假设核函数 $h$ 退化为仅依赖单行（即 $U = \frac{1}{m}\sum_{i=1}^m g(\xi_i, \eta_{1:n})$），请根据本文的 Hoeffding 分解推导其渐近分布，并说明此时 U-统计量是否退化？如果不退化，渐近方差的结构是什么？
