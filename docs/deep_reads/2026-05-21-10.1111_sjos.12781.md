# Ratio‐consistency of some invariant <i>U</i>‐statistic‐based estimators with an application to high‐dimensional data ranking

**作者**: Jia Guo, Bu Zhou  
**来源**: Scandinavian Journal of Statistics  
**主题**: 其他  
**相关性**: 9/10  
**链接**: <https://doi.org/10.1111/sjos.12781>

---

### 一、核心问题与贡献
①研究了高维设定下协方差矩阵函数的旋转-平移不变无偏估计及其比率一致性问题。②核心工具是基于U-统计量线性组合构造不变估计器，并利用Hoeffding分解与U-统计量方差公式推导比率一致性的充分条件。③主要贡献是提出了一般性的不变估计器构造程序与比率一致性判定引理，避免了直接处理高维特征结构的复杂依赖，并据此提出了一种新的高维数据排序方法。

### 二、基础设定
- **核心概念与符号**：
  - $\Sigma$: $p \times p$ 总体协方差矩阵。
  - $f(\Sigma)$: 协方差矩阵的函数（如迹、行列式等），为待估参数。
  - Ratio-consistency: 估计量 $\hat{\theta}_n$ 满足 $\hat{\theta}_n / \theta \xrightarrow{p} 1$，提供相对误差控制，是高维推断中保证后续步骤（如排序、检验）有效性的关键。
  - Rotation-translation-invariant: 估计量在数据经过正交变换和平移后保持不变，符合协方差结构估计的天然要求。
- **关键假设**：
  - 弱矩假设：仅需特定阶数的矩存在（远低于亚高斯/亚指数假设要求）。
  - 维度-样本量关系：允许 $p/n \to \infty$（超高维设定）。
  - 信号阶数条件：目标参数 $f(\Sigma)$ 的平方阶数需主导估计量方差的阶数。
  - *对比与放宽*：相比高维推断文献常要求的亚高斯噪声或协方差矩阵特征根间距/极值条件，本文仅依赖矩条件与代数结构，假设大幅放宽。
- **问题背景**：
  - 已有方法的不足：高维协方差函数估计常依赖特征结构分析或强分布假设，且估计量往往不具备旋转平移不变性或有偏，导致比率一致性难以建立。
  - 最相关文献区别：相比 Chen & Qin (2010) 等基于迹的特定检验，本文提供更一般的协方差函数估计框架；相比传统高维U统计量文献，本文聚焦于"不变性"与"比率一致性"的代数与概率结构映射。

### 三、主要定理 / 核心结果
1. **原文陈述**：若基于U-统计量线性组合的估计量 $\hat{\theta}$ 满足 $\text{Var}(\hat{\theta}) = o(\theta^2)$，则 $\hat{\theta}$ 具有比率一致性。在弱矩条件下，所构造的 $f(\Sigma)$ 的无偏不变估计 $\hat{\theta}$ 满足 $\text{Var}(\hat{\theta}) = O(n^{-1} p^{c_1})$ 且 $\theta^2 = \Omega(p^{c_2})$，当 $c_2 > c_1 + 1$ 时具备比率一致性。
2. **直观解释**：估计量的方差相对于其平方均值的阶数为无穷小，保证了相对误差依概率趋于0。只要目标参数 $f(\Sigma)$ 随维度增长足够快（信号强于噪声累积），估计量即使在 $p \gg n$ 时也能保持相对精度。
3. **解决了什么技术难点**：将比率一致性的判定转化为纯粹的方差阶数分析，避开了高维经验过程紧致性论证或随机矩阵局部特征值定律的复杂依赖。
4. **适用条件与局限**：需要精确计算或放缩U-统计量的方差；对于非退化U-统计量，方差计算本身可能繁琐。若 $f(\Sigma)$ 增长过慢（如 $c_2$ 太小），比率一致性可能失效。

### 四、证明框架 / 方法设计
- **证明主干逻辑**：构造法 + Hoeffding分解 + 矩量法。
- **拆解为 3-5 个关键逻辑步骤**：
  1. **不变无偏估计器构造**：利用对称多项式与U-统计量的对应关系，将 $f(\Sigma)$ 表示为样本矩的线性组合，证明其旋转-平移不变性与无偏性。
  2. **Hoeffding分解**：将构造的U-统计量分解为投影项与退化核项，分离出主要方差来源。
  3. **方差阶数精确放缩**：利用U-统计量方差公式（基于核函数的L2范数），将 $\text{Var}(\hat{\theta})$ 表示为关于 $p$ 和 $n$ 的多项式阶数。
  4. **比率一致性判定**：比较方差阶数与 $\theta^2$ 的阶数，在 $p, n \to \infty$ 的渐近框架下验证 $\text{Var}(\hat{\theta}) / \theta^2 \to 0$。
- **最关键的技巧性引理或"跳跃点"**：将高维协方差矩阵函数的代数恒等式与U-统计量的Hoeffding分解结合。利用旋转不变性，将看似复杂的交叉项方差计算转化为关于迹算子 $\text{tr}(\Sigma^k)$ 的多项式分析，从而将随机变量的方差控制转化为确定性的代数阶数比较（"去随机化"）。
- **数学工具评价**：经典工具（U-统计量方差公式、Hoeffding分解）的巧妙组合与代数化处理。未引入复杂的经验过程或随机矩阵局部特征值定律，而是通过纯粹的代数与矩方法绕过了特征结构的依赖。

### 五、与研究者兴趣的关联
- **连接到哪个子方向**：高维 U-统计量推断。
- **可借鉴的核心思路或技术工具**："比率一致性"作为高维相对误差控制的替代目标，比传统的绝对一致性（$\hat{\theta} - \theta \xrightarrow{p} 0$）在高维推断中更易达到且更实用；利用U-统计量方差公式将高维方差控制转化为代数阶数比较的"去随机化"技巧，可直接迁移至其他高阶U-统计量（如高维独立性检验、条件独立性检验中的核统计量）的渐近分析中。
- **值得精读的关键参考文献**：
  1. Hoeffding, W. (1948). *A class of statistics with asymptotically normal distribution.* - 理解本文证明基石与U-统计量方差分解的经典源头。
  2. Chen, S. X., & Qin, Y. L. (2010). *A two-sample test for high-dimensional data with applications to gene-set testing.* - 本文方法在迹检验问题上的直接对比与推广基础，有助于理解不变估计器的优势。

### 六、延伸思考与练习
- **假设扰动**：若修改"弱矩假设"为"仅存在有限一阶矩"（如重尾分布），结论会如何变化？技术上需要什么新工具？答：Hoeffding分解的方差公式将失效（方差可能无穷），需要引入截断技术或鲁棒U-统计量框架（如 Catoni's estimator 或 median-of-means）来控制重尾影响，比率一致性的阶数条件将发生根本改变。
- **开放问题**：如何将此类基于不变U-统计量的比率一致性框架推广至条件协方差矩阵函数的估计（如给定协变量后的高维残差排序）？
- **理解检测题**：假设你要估计 $f(\Sigma) = \text{tr}(\Sigma^2)$，请写出其旋转-平移不变的无偏U-统计量估计形式，并利用Hoeffding分解说明在何种 $p/n$ 发散速率下，该估计量丧失比率一致性？
