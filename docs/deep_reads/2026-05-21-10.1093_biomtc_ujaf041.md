# Optimal dynamic treatment regime estimation in the presence of nonadherence

**作者**: Dylan Spicker, Michael P Wallace, Grace Y Yi  
**来源**: Biometrics  
**主题**: 因果推断  
**相关性**: 8/10  
**链接**: <https://doi.org/10.1093/biomtc/ujaf041>

---

### 一、核心问题与贡献
①本文研究了纵向动态治疗策略（DTR）估计中存在非依从性（分配处理与实际接受处理不一致）时最优策略的估计问题。②核心方法是将分配机制与依从机制解耦，利用半参数理论构造包含依从概率与分配概率的双稳健估计量。③主要贡献在于证明忽略非依从性将导致次优策略，并提出了具备双稳健性与 $n^{-1/2}$-CAN 性质的新估计量，其策略价值逼近无依从性理想情形。

### 二、基础设定
- **核心概念与符号**：
  - $X_k$：第 $k$ 阶段的协变量（时变混杂）。
  - $Z_k$：第 $k$ 阶段的分配处理。
  - $A_k$：第 $k$ 阶段的实际接受处理，非依从性体现为 $A_k \neq Z_k$。
  - $Y$：最终结局。
  - $d = (d_1, d_2)$：动态治疗策略，通常定义为对实际处理 $A_k$ 的决策规则。
  - $\pi_k = P(Z_k=1|H_k)$：分配机制，$H_k$ 为历史信息。
  - $r_k = P(A_k=1|Z_k, H_k)$：依从机制。
- **关键假设**：
  - **Sequential Randomization (SRA / Ignorability)**：$Z_k \perp\!\!\!\perp (Y(\bar{z}, \bar{a}), A_k(\bar{z}_{k-1})) | H_k$。分配独立于潜在结局与潜在依从行为。与标准 DTR 相比，此处需明确分配的随机性不依赖于未测的依从倾向。
  - **Exclusion Restriction (ER)**：分配 $Z_k$ 仅通过实际接受的处理 $A_k$ 影响结局 $Y$ 及后续协变量（无安慰剂效应）。
  - **Positivity**：$0 < P(Z_k=1|H_k) < 1$ 且 $0 < P(A_k=1|Z_k, H_k) < 1$。
  - **对比已有文献**：传统 DTR 估计（如 Q-learning, G-estimation）假设 $A_k = Z_k$；本文放宽了此假设，将 $Z_k$ 视作类似工具变量（IV）的结构，处理 $A_k$ 的内生性。
- **问题背景**：现有 DTR 文献大多假设完美依从。当 $A \neq Z$ 时，若直接用 $Z$ 估计，得到的是 intention-to-treat (ITT) 策略，非最优因果策略；若直接用 $A$ 估计，因 $A$ 存在自选择（混杂）导致估计有偏。与 Robins (2004) 的最优 DTR 理论相比，本文处理了纵向 IV 结构下的策略优化；与单阶段非依从因果推断相比，本文需解决时变混杂与非依从的叠加效应。

### 三、主要定理 / 核心结果
1. **原文陈述**：若忽略非依从性（将 $Z$ 视作 $A$ 或仅分析 $Z$），所得策略 $d_{naive}$ 的期望结局严格劣于考虑非依从性的最优策略 $d_{opt}$（除非依从率为1或处理效应齐次）。所提估计量 $\hat{V}(d)$ 在依从模型 $r_k$ 或分配模型 $\pi_k$（及结局模型 $Q_k$）之一正确指定时一致，且具有 $\sqrt{n}(\hat{V}(d) - V(d)) \xrightarrow{d} N(0, \Sigma)$ 的渐近正态性。
2. **直观解释**：忽略非依从性相当于用 ITT 效应代替 CACE 效应指导临床，由于依从率在协变量子群中异质，会导致错配。双稳健性意味着只要正确刻画了依从机制或分配机制之一，就能纠正 $A$ 的选择偏差，还原真实处理效应。
3. **解决了什么技术难点**：在纵向设定下，非依从性导致反事实依从行为的识别与后续协变量的演化耦合。本文解决了如何在多阶段决策中，剥离依从机制对中间协变量 $X_k$ 演化的混淆效应，从而正确反事实地推演患者轨迹。
4. **适用条件与局限**：依赖 SRA、ER 和 Positivity。若存在非依从相关的未测混杂（即 $A_k$ 的选择受与 $Y$ 相关的未测变量影响），ER 和 SRA 破坏，双稳健性失效。单调性假设在某些设定下可能是必要的。

### 四、证明框架 / 方法设计
- **证明主干逻辑**：构造半参数影响函数 -> 验证双稳健性 -> 经验过程理论证明 CAN。
- **拆解为 3-5 个关键逻辑步骤**：
  1. **识别**：利用 g-computation 公式，将含非依从性的潜在结局期望 $E[Y(\bar{a})]$ 表达为观测数据分布的泛函，需对依从机制 $r_k$ 和分配机制 $\pi_k$ 进行积分加权。
  2. **影响函数构造**：在无参模型下求 Gateaux 导数，得到有效影响函数 (EIF)。EIF 包含结局残差项与依从/分配倾向得分项的交互。
  3. **双稳健性验证**：将估计量写为 $Z$-estimation 形式 $M_n(\theta, \eta) = 0$。证明当 nuisance parameter $\eta = (\pi, r, Q)$ 中部分误设时，极限方程 $E[M(\theta_0, \eta^*)] = 0$ 仍成立。
  4. **CAN 证明**：利用 Taylor 展开、经验过程一致收敛定理，证明余项为 $o_p(1/\sqrt{n})$。
- **最关键的技巧性引理或"跳跃点"**：在多阶段非依从设定下，Stage $k$ 的协变量 $X_k$ 依赖于历史分配 $\bar{Z}_{k-1}$ 和历史依从 $\bar{A}_{k-1}$。在构造影响函数时，需利用 sequential conditional expectation 技巧（类似 Robins 的 SNMM 构造），将 $X_k$ 的演化分解，剥离出依从机制对后续状态的混淆效应，这是从单阶段 IV 推广到多阶段的核心跳跃。
- **数学工具评价**：经典 semiparametric efficiency theory 与 Z-estimation 框架的巧妙组合。创新点不在于数学工具本身，而在于将 IV/nonadherence 结构嵌入 DTR 的 sequential decision making 框架中，推导出新的 EIF。

### 五、与研究者兴趣的关联
- **连接子方向**：Longitudinal causal inference with IV / Proximal CI in dynamic settings。
- **可借鉴的核心思路**：将非依从性视为一种特殊的纵向 IV 结构（$Z$ 为 negative control exposure / IV，$A$ 为真实暴露），在 DTR 的多阶段决策中推导双稳健 CAN 估计量的技术。特别是如何处理中间协变量 $X_k$ 受历史非依从性影响的识别问题，这与 Proximal CI 中处理时变混杂与未测变量的思路有深刻对应，可迁移至 Proximal DTR 的理论推导中。
- **值得精读的关键参考文献**：
  1. **Robins (2004) Optimal structural nested models**：DTR 与 g-estimation 的基石，理解本文如何扩展其基础。
  2. **Hernan & Robins (2006) IV methods in DTR**：对比单阶段与多阶段非依从的处理差异，理解本文的 ER 假设为何在多阶段更微妙。

### 六、延伸思考与练习
- **假设扰动**：若放宽 Exclusion Restriction (ER)，即分配 $Z_k$ 直接影响结局 $Y$（例如医生鼓励效应），结论如何变化？技术上需要引入 principal stratification 框架或进行 sensitivity analysis，此时双稳健性极可能不再成立，需要新的识别策略。
- **开放问题**：如何在非依从性存在时，进行最优 DTR 的 inference（如置信区间），而非仅点估计？高维协变量下如何通过 debiasing 保持 CAN？
- **理解检测题**：在两阶段 DTR 中，若第一阶段完全依从（$A_1=Z_1$），第二阶段存在非依从（$A_2 \neq Z_2$），请写出此时最优 DTR 值函数 $V(d_1, d_2)$ 的非参数识别公式，并指出与标准 g-formula 的区别。
