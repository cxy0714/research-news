# Average direct and indirect causal effects under interference

**作者**: Yuchen Hu, Shuangning Li, Stefan Wager  
**来源**: Biometrika  
**主题**: 因果推断  
**相关性**: 9/10  
**链接**: <https://doi.org/10.1093/biomet/asac008>

---

### 一、核心问题与贡献
①研究了在存在个体间干扰的潜在结果模型下，如何非参数地定义和识别平均间接因果效应。②核心工具是利用伯努利随机化试验中处理概率的微小变分，将间接效应定义为总体平均干预效应关于处理概率的导数与直接效应之差。③证明了直接与间接效应的分解定理（二者之和恒等于边际政策干预效应的导数），且该非参数间接效应在多种参数干扰模型下仍保持自然解释性。

### 二、基础设定
- **核心概念与符号**：
  - $Y_i(W)$：单元 $i$ 在处理分配向量 $W \in \{0,1\}^n$ 下的潜在结果。
  - Interference (干扰)：$Y_i(W)$ 依赖于他人的处理状态 $W_{-i}$。
  - $\mu(p) = E_{W \sim \text{Bernoulli}(p)}[Y_i(W)]$：总体平均潜在结果（关于处理概率 $p$ 的函数）。
  - $\tau_{DIR}(p)$：平均直接效应（自身处理状态改变的效应）。
  - $\tau_{IND}(p)$：平均间接效应（他人处理概率微小改变带来的溢出效应）。
- **关键假设**：
  - **Bernoulli randomization**：处理分配 $W_i \overset{i.i.d.}{\sim} \text{Bernoulli}(p)$。统计学含义：赋予了总体均值 $\mu(p)$ 关于 $p$ 的光滑性与可微性，使得变分分析可行。相比文献中基于两阶段随机化或固定 $p$ 对比的设定，这是实现非参数导数分解的核心引擎。
  - **Regularity of $\mu(p)$**：$\mu(p)$ 关于 $p$ 可微。统计学含义：排除了阈值效应等不连续的干扰结构，保证了间接效应作为导数的良定义。
- **问题背景**：已有干扰下的因果推断文献（如 Hudgens & Halloran 2008）定义间接效应通常需要对比两个不同处理概率 $p$ 的独立实验，这导致估计量方差大且缺乏微观机理。与 Manski (2013) 的政策干预效应相比，本文将宏观的边际效应导数精确拆解为微观的直接与间接通道，填补了单一实验设定下非参数间接效应定义的空白。

### 三、主要定理 / 核心结果
1. **原文陈述**：在 Bernoulli 试验中，令 $\tau_{DIR}(p)$ 为平均直接效应，$\tau_{IND}(p)$ 为平均间接效应，$\mu(p)$ 为总体平均潜在结果。则分解定理成立：$\tau_{DIR}(p) + \tau_{IND}(p) = \mu'(p)$。
2. **直观解释**：总体处理效应关于处理概率 $p$ 的导数（即微小增加全员处理概率带来的总体效应变化），恰好等于个体自身处理改变带来的直接效应，加上由于他人处理概率增加带来的间接溢出效应。这为间接效应提供了基于变分的操作化定义：$\tau_{IND}(p) = \mu'(p) - \tau_{DIR}(p)$。
3. **解决了什么技术难点**：在无需多实验对比的情况下，赋予了间接效应非参数的因果解释，解决了干扰下间接效应"不可识别"或"依赖强参数假设"的困境。
4. **适用条件与局限**：必须依赖 Bernoulli 随机化假设（独立性），且 $\mu(p)$ 需要可微。局限在于：对于非 Bernoulli 设计（如完全随机化或区组设计），该导数分解不能直接套用；非参数估计 $\mu'(p)$ 在有限样本下收敛速度较慢（非参数导数估计的维数灾难），需依赖局部多项式或正则化。

### 四、证明框架 / 方法设计
- **证明主干逻辑**：构造法与变分分析。
- **拆解为 3 个关键逻辑步骤**：
  1. **展开总体均值**：将 $\mu(p) = E[Y_i(W) | W \sim \text{Bernoulli}(p)]$ 展开为关于 $W$ 的条件期望形式。
  2. **导数-协方差转化**：对 $\mu(p)$ 关于 $p$ 求导，利用 Bernoulli 分布的指数族性质，将导数转化为可观测量的协方差：$\mu'(p) = \frac{1}{p(1-p)} \text{Cov}(W_i, Y_i(W))$。
  3. **效应剥离与识别**：将上述协方差分解为自身处理 $W_i$ 的效应（对应 $\tau_{DIR}(p)$）与残差项，证明残差项的期望恰好等于改变他人处理状态 $W_{-i}$ 带来的期望效应变化，即 $\tau_{IND}(p)$。
- **最关键的技巧性引理或"跳跃点"**：将 $\mu'(p)$ 转化为 $\text{Cov}(W_i, Y_i(W)) / (p(1-p))$。这一步利用了 Bernoulli 随机化的独立性与线性性质，将宏观的概率路径导数与微观的协方差（可识别的矩）联系起来，从而避开了对不可观测反事实的直接比较。
- **数学工具评价**：是经典变分法与因果推断中矩方法的巧妙组合。没有引入高深的拓扑或经验过程工具，而是通过概率分布的参数路径求导，实现了非参数定义的参数化识别。

### 五、与研究者兴趣的关联
- **连接到哪个子方向**：Causal inference under interference / Mediation analysis / Semiparametric efficiency bounds under network settings。
- **可借鉴的核心思路或技术工具**：将处理概率视为连续参数，通过求导定义因果参数（Marginal Treatment Effect 的导数），这一变分思想可迁移到其他离散处理设定下的连续化因果参数定义中；此外，将协方差与导数联系的技术，在半参数效率界计算中（求切空间上的路径导数）有深刻的对偶性，可借鉴用于推导干扰设定下的有效影响函数。
- **值得精读的关键参考文献**：
  1. Manski (2013), "Identification of treatment response with social interactions"：提供干扰下政策干预效应的理论基础，理解 $\mu(p)$ 的经济学动机。
  2. Hudgens & Halloran (2008), "Toward causal inference with interference"：经典的干扰下直接/间接效应定义（基于两阶段随机化），对比本文可看出 Bernoulli 假设带来的变分优势。

### 六、延伸思考与练习
- **假设扰动**：若将 Bernoulli randomization 改为完全随机化实验，即总处理人数 $N_1$ 固定。此时 $\mu(p)$ 的定义失去意义（不存在连续的 $p$ 路径），分解定理失效。技术上需要引入超总体框架或基于置换分布的离散差分工具来重新定义间接效应。
- **开放问题**：如何在观测性研究（存在未测量混淆）下，利用工具变量或 proximal causal inference 识别此处的间接效应？如何构造 $\mu'(p)$ 的半参数有效估计量并推导其效率界？
- **理解检测题**：假设潜在结果模型为 $Y_i(W) = \alpha + \beta W_i + \gamma \sum_{j \in \mathcal{N}_i} W_j$，其中 $\mathcal{N}_i$ 是单元 $i$ 的邻居集。在 Bernoulli(p) 随机化下，显式计算 $\mu(p)$, $\tau_{DIR}(p)$ 和 $\tau_{IND}(p)$，并验证分解定理 $\tau_{DIR}(p) + \tau_{IND}(p) = \mu'(p)$ 是否成立。
