# Time-uniform central limit theory and asymptotic confidence sequences

**作者**: Ian Waudby-Smith, David Arbour, Ritwik Sinha, Edward H. Kennedy, Aaditya Ramdas  
**来源**: Annals of Statistics  
**主题**: 非参数 / 半参数  
**相关性**: 8/10  
**链接**: <https://doi.org/10.1214/24-aos2408>

### 一、核心问题与贡献
①研究了序贯分析中如何在弱假设下构建时间均匀的置信区间（即置信序列），以支持在任意停止时间下的有效统计推断。②核心工具是基于 Strassen 型强不变原理，将样本均值过程由隐式高斯过程一致逼近，提出了渐近置信序列的一般定义与推导框架。③放弃了传统非渐近置信序列的有限样本保证，换取了类似经典 CLT 的广泛适用性与渐近时间均匀保证，并成功推导了观察性研究中平均处理效应（ATE）的渐近置信序列。

### 二、基础设定
- **核心概念与符号**：
  - $C_n$：非渐近置信序列，满足 $P(\exists n \ge 1: \mu \notin C_n) \le \alpha$。
  - $\tilde{C}_n$：渐近置信序列，满足 $\limsup_{m\to\infty} P(\exists n \ge m: \mu \notin \tilde{C}_n) \le \alpha$。
  - 强不变原理：存在概率空间重构，使得部分和过程 $S_n$ 与高斯过程（如 Wiener 过程 $W(t)$）满足 $S_n - W(n) = o(r_n)$ a.s.（$r_n$ 通常为 $\sqrt{n\log\log n}$）。
  - $\psi$：平均处理效应（ATE），$\psi = E[Y(1) - Y(0)]$。
  - $\phi(Z)$：半参数有效影响函数。

- **关键假设**：
  - **有限方差**：$E[\phi(Z)^2] < \infty$。统计学含义是允许经典 CLT 弱分布假设；相比非渐近 CS 要求的有界/次高斯假设大幅放宽。
  - **强逼近余项条件**：半参数估计量余项 $\epsilon_n = o(\sqrt{\log\log n / n})$ a.s.。统计学含义是要求 nuisance parameter 估计的几乎必然收敛速度足够快；相比固定时间下的 $o_p(n^{-1/2})$ 更为苛刻。
  - **Positivity**：倾向得分 $\pi$ 有界远离 0 和 1。因果推断标准假设。

- **问题背景**：
  传统固定样本量 CLT 无法处理"偷看数据"（peeking）导致的 I 类错误膨胀；现有非渐近 CS 依赖强分布假设（如次高斯），在复杂半参数问题（如基于影响函数的 ATE 估计）中几乎无法推导。与最相关文献的区别：Robbins 等人的 CS 是非渐近的；Lai 的序贯 CI 依赖特定参数形式且非时间均匀；本文通过强逼近实现非参数/半参数下的时间均匀渐近推断。

### 三、主要定理 / 核心结果

1. **渐近置信序列的存在性定理**
   - **原文陈述**：若估计量满足 $\hat{\theta}_n - \theta = \frac{1}{n}\sum_{i=1}^n \phi(Z_i) + \epsilon_n$，且 $\epsilon_n = o(\sqrt{\log\log n / n})$ a.s.，则可构造 $\tilde{C}_n$ 使得 $\limsup_{m\to\infty} P(\exists n \ge m: \theta \notin \tilde{C}_n) \le \alpha$。
   - **直观解释**：只要估计量能写成影响函数的均值加上一个几乎必然收敛的余项，就可以利用高斯过程的边界构造出时间均匀的置信带，且余项在渐近意义上可被边界宽度"吸收"。
   - **解决的技术难点**：将半参数估计量纳入序贯推断框架，绕开了非渐近集中不等式在半参数余项上不可推导的瓶颈。
   - **适用条件与局限**：余项 $\epsilon_n$ 的 a.s. 收敛速度要求是核心局限；对于高维或复杂 ML nuisance 估计，验证此条件极具挑战性。

2. **ATE 的渐近置信序列定理**
   - **原文陈述**：在观察性研究或随机化实验中，基于交叉拟合的影响函数估计量，构造 ATE 的渐近 CS。
   - **直观解释**：将固定时间下的 AIF 估计推广到序贯设定，允许在不惩罚"偷看数据"的前提下，持续监测因果效应。
   - **解决的技术难点**：处理半参数估计中的 nuisance 估计误差，通过余项控制保证其不破坏时间均匀覆盖概率。

### 四、证明框架 / 方法设计

- **证明主干逻辑**：构造法 + 强逼近 + 极值边界。
- **拆解为 3-5 个关键逻辑步骤**：
  1. **影响函数线性化**：将半参数估计量 $\hat{\psi}_n - \psi$ 分解为 $\frac{1}{n}\sum \phi(Z_i) + \epsilon_n$，其中 $\epsilon_n$ 包含 nuisance 估计误差与二阶项。
  2. **强不变原理应用**：对 i.i.d. 的影响函数项 $\frac{1}{n}\sum \phi(Z_i)$，应用 Strassen 型强逼近定理，构造概率空间使其被 Wiener 过程 $W(n)$ 逼近，误差为 $o(\sqrt{\log\log n / n})$ a.s.。
  3. **高斯过程边界计算**：对 $W(t)/t$ 应用时间均匀的边界（如基于 Law of the Iterated Logarithm 的边界），构造高斯过程的置信序列。
  4. **余项吸收**：证明 $\epsilon_n = o(\sqrt{\log\log n / n})$ a.s.，使得余项被高斯过程边界的宽度所主导，从而保证整体的时间均匀覆盖。
- **最关键的技巧性引理或"跳跃点"**：将经典半参数理论中的 $o_p(\cdot)$ 余项提升为 $o_{a.s.}(\cdot)$ 余项。这是连接固定时间渐近与时间均匀渐近的核心跳跃，因为时间均匀性要求在无限时间维度上成立，依概率收敛不足以控制 suprema。
- **数学工具评价**：经典工具（强不变原理、影响函数理论）的精妙组合。强逼近理论在统计推断中的复兴，填补了非渐近界与固定时间渐近界之间的空白。

### 五、与研究者兴趣的关联

- **连接到哪个子方向**：序贯因果推断 / 半参数效率理论。
- **可借鉴的核心思路或技术工具**：将半参数影响函数（IF）分解与强不变原理结合的框架，可直接迁移到其他因果参数（如 CATE, longitudinal treatment effects, mediation）的序贯估计与假设检验中；特别是将 $o_p(\cdot)$ 提升为 $o_{a.s.}(\cdot)$ 的余项分析技术，对高维/半参数 DML 的序贯化至关重要。
- **值得精读的关键参考文献**：
  1. Strassen (1964) "An invariance principle for the central limit theorem"：强不变原理的奠基之作，理解逼近误差 $o(\sqrt{n\log\log n})$ 的来源。
  2. Kennedy (2022) "Semiparametric theory and empirical processes in causal inference"：理解 ATE 影响函数余项 $o_p(\cdot)$ 的经典处理，对比本文的 a.s. 收敛要求。
  3. Howard et al. (2021) "Time-uniform Chernoff bounds"：非渐近 CS 的代表作，对比理解为何在半参数下必须走向渐近 CS。

### 六、延伸思考与练习

- **假设扰动**：若修改"影响函数余项 $\epsilon_n = o(\sqrt{\log\log n / n})$ a.s."为 $\epsilon_n = O_p(\sqrt{\log\log n / n})$（即仅依概率收敛），结论会如何变化？技术上需要什么新工具？（提示：可能需要经验过程的极值理论或弱不变原理，但时间均匀覆盖概率可能无法保证，需退而求其次寻找其他形式的边界）。
- **开放问题**：如何将此渐近 CS 框架推广到高维设定（如 $p \gg n$），其中 nuisance parameter 使用 Lasso/DNN 估计，余项的 a.s. 收敛速度如何依赖于维数 $p$ 与样本量 $n$？
- **理解检测题**：假设你正在估计一个连续治疗变量的因果效应（如剂量-响应曲线的积分），其半参数影响函数包含非参数核回归项。请论述：在序贯设定下，核回归的带宽 $h_n$ 需要满足什么条件，才能保证余项 $\epsilon_n$ 满足渐近 CS 的要求？这比固定样本量下的条件更严格还是更宽松？
