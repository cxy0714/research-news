# Information-Theoretic Thresholds for the Alignments of Partially Correlated Graphs

**作者**: Dong Huang, Xianwen Song, Pengkun Yang  
**来源**: IEEE Transactions on Information Theory  
**主题**: 高维统计 / 随机矩阵  
**相关性**: 8/10  
**链接**: <https://doi.org/10.1109/tit.2025.3619813>

---

### 一、核心问题与贡献
①研究了两个随机图中仅存在部分诱导子图相关性时，隐藏顶点对应关系的恢复（图匹配）问题。②提出了部分相关 Erdős-Rényi (ER) 模型与部分相关 Gaussian Wigner (GW) 模型，并引入 correlated functional digraphs 与低阶累积生成函数作为核心分析工具。③精确刻画了 partial 与 exact recovery 的信息论阈值与 minimax 最优速率，并发现 GW 模型中 partial 与 exact recovery 的阈值重合（与 ER 模型产生本质分野）。

### 二、基础设定
- **核心概念与符号**：
  - 部分相关 ER/GW 模型：图 $G_1, G_2$ 中仅大小为 $n_s$ 的诱导子图 $S_1, S_2$ 具有相关性，其余 $n-n_s$ 个顶点独立生成。
  - 潜变量：隐藏置换 $\pi^*$（顶点对应关系）与潜变量集合 $S$（相关子图的顶点索引）。
  - 速率参数：$s = n_s/n$（相关顶点比例），$p$（ER 模型边概率），$\sigma$（GW 模型噪声标准差）。
  - 恢复目标：Partial recovery（匹配正比例顶点），Exact recovery（完美匹配所有相关顶点）。
- **关键假设**：
  - 稀疏/稠密渐近设定：$p = \Theta(\log n / n)$ 或常数；$\sigma^2 = \Theta(\log n / n)$ 等。
  - 相关子图结构假设：$S_1, S_2$ 在 $\pi^*$ 下完美对齐，且边/矩阵元的相关性由参数 $\rho$ 或噪声 $\sigma$ 控制。
  - 统计学含义：允许图的大部分结构是"纯噪声/干扰"，仅局部存在信号，极大放宽了传统全图相关假设，更贴近真实网络（如部分重叠的社交网络）。
- **问题背景**：
  - 已有图匹配文献（如 Ding et al., 2021; Wu et al., 2022）主要研究全图相关模型，要求两图所有顶点均存在对应关系与相关性。
  - 本文针对"大部分顶点无对应关系/纯干扰"的更现实设定，解决了局部信号与全局干扰共存时的信息论极限刻画难题。

### 三、主要定理 / 核心结果

1. **定理：部分相关 ER 模型的 Partial Recovery 最优速率**
   - 原文陈述：当 $n_s p \log \frac{1}{p} > (2+\epsilon) n \log n$ 时，可正确匹配 $S$ 中 $(1-o(1))n_s$ 个顶点；当 $n_s p \log \frac{1}{p} < (2-\epsilon) n \log n$ 时，匹配正比例顶点不可能。
   - 直观解释：相关子图的信号强度（边数与对数因子之积）必须战胜全图置换空间的对数基数 $(2n \log n)$。比例 $s$ 使得阈值从全图的 $2n \log n$ 降维至 $n_s$ 量级，但搜索空间仍是全局的。
   - 解决的技术难点：在全局搜索空间中定位局部信号的 minimax 界。
   - 适用条件与局限：依赖 ER 模型的独立边假设与稀疏/常数度渐近条件；若度分布为重尾（如 Power-law），阈值形式可能改变。

2. **定理：部分相关 ER 模型的 Exact Recovery 最优速率**
   - 原文陈述：阈值为 $n_s p \log \frac{1}{p} > (4+\epsilon) n \log n$（粗略形式，常数因子随 $s$ 变化）。
   - 直观解释：Exact 要求不仅找到相关子图，还要完美无错，因此需要额外一倍的信号强度来压制局部错误概率。

3. **定理：部分相关 GW 模型的 Partial/Exact Recovery 阈值重合**
   - 原文陈述：Partial 与 Exact recovery 的最优速率阈值均为 $n_s / \sigma^2 > (4+\epsilon) n \log n$。
   - 直观解释：Gaussian 矩阵元的连续性使得一旦能匹配正比例顶点，剩余顶点的似然比已足够强，必然导致完美匹配。这与 ER 模型中离散边造成的"部分匹配容易、完美匹配难"形成鲜明对比。
   - 解决的技术难点：揭示了 RMT 中连续分布与离散图模型在 phase transition 结构上的本质
