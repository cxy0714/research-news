# Statistical inference for function-on-function linear regression

**作者**: Holger Dette, Jiajun Tang  
**来源**: Bernoulli  
**主题**: 非参数 / 半参数  
**相关性**: 8/10  
**链接**: <https://doi.org/10.3150/23-bej1598>

---

### 一、核心问题与贡献
①研究了响应变量与预测变量均为随机函数的函数对函数线性回归中的统计推断问题。②采用基于薄板样条惩罚的再生核希尔伯特空间（RKHS）中的惩罚最小二乘法估计系数函数。③证明了估计量的极小极大最优收敛速率，并推导出其Bahadur表达，从而基于Banach值随机变量的sup范数收敛理论实现了bootstrap逐点与逐区域推断。

### 二、基础设定
- **核心概念与符号**：
  - $Y(t), X(s)$：分别为取值于 $L^2[T]$ 和 $L^2[S]$ 的响应函数与预测函数。
  - $\beta(s,t)$：双变量系数函数（核函数），模型为 $Y(t) = \int_S X(s)\beta(s,t)ds + \varepsilon(t)$。
  - $\mathcal{H}$：由薄板样条惩罚定义的RKHS，$\beta$ 的函数空间。
  - $J(\beta)$：薄板样条平滑惩罚，度量 $\beta$ 的粗糙度。
  - $\hat{\beta}_\lambda$：正则化参数为 $\lambda$ 的惩罚最小二乘估计量。
  - Bahadur representation：将估计量偏差分解为确定性偏差与随机经验过程的线性主项及高阶余项。
- **关键假设**：
  - **平滑性假设**（$\beta \in \mathcal{H}$ 且属于特定Sobolev空间）：控制函数空间复杂度，决定极小极大收敛速率的下界；相比参数化或单指标模型，保留了非参数建模的灵活性。
  - **协方差算子与矩条件**（$X$ 与 $\varepsilon$ 满足亚高斯尾或特定有界矩条件）：保证经验协方差算子在RKHS中的收敛性，处理泛函数据的无限维特性；相比有限维设定，此假设是控制经验过程 sup-norm 偏差的关键。
  - **正则化参数条件**（$\lambda \asymp n^{-2m/(2m+d+2)}$ 等）：平衡偏差与方差；若 $\lambda$ 衰减过慢，偏差项将主导，无法构造有效的 Bahadur 表达。
- **问题背景**：已有函数回归文献多集中于标量对函数/函数对标量，或仅关注估计的 $L^2$ 收敛速率而缺乏严格推断工具。与 Yao et al. (2005) 等经典函数线性回归文献相比，本文聚焦于双变量系数函数的*推断*而非仅估计；与一般 RKHS 估计文献相比，本文通过 Bahadur 表达处理了 sup-norm 下的 bootstrap 收敛，克服了 $L^2$ 范数无法做逐点检验的缺陷。

### 三、主要定理 / 核心结果
1. **极小极大最优收敛速率定理**
   - 原文陈述：估计量 $\hat{\beta}$ 在 RKHS 范数（或 $L^2$ 范数）下达到极小极大下界速率 $O_P(n^{-2m/(2m+d+2)})$，其中 $m$ 为光滑度，$d$ 为预测函数的维度。
   - 直观解释：惩罚项有效控制了无限维函数空间的容量，使得估计方差不会发散，偏差与方差权衡达到理论最优。
   - 技术难点：泛函数据下积分算子的谱衰减特性与惩罚算子的交互作用，需精确刻画特征值衰减对收敛的影响。
   - 适用条件与局限：依赖于协变量过程的特定谱条件；若谱衰减极快（极光滑过程），可能需要调整速率。

2. **Bahadur 表达与 sup-norm 收敛定理**
   - 原文陈述：$\hat{\beta} - \beta_0 = \text{Bias}(\lambda) + \frac{1}{n}\sum_{i=1}^n \Psi_i + R_n$，其中余项 $R_n$ 在 sup-norm 下满足 $\|R_n\|_\infty = o_P(n^{-1/2})$。
   - 直观解释：将非参数估计量的非线性随机扰动线性化为一阶经验过程，使得余项在逐点意义下可忽略，从而将推断问题转化为对 Banach 值随机变量的分析。
   - 技术难点：在无限维空间中控制余项 $R_n$ 的 sup-norm 量级，需克服经验算子逆的不稳定性。
   - 适用条件与局限：严重依赖平滑度与正则化参数的精细匹配（欠平滑化条件），若 $\lambda$ 衰减过慢，偏差项将主导，破坏 Bahadur 表达的推断有效性。

### 四、证明框架 / 方法设计
- **证明主干逻辑**：构造法 + 经验过程 + 算子谱理论。
- **拆解为关键逻辑步骤**：
  1. **算子展开**：将惩罚最小二乘的一阶条件转化为积分算子方程，利用 RKHS 的再生性质将估计量表示为样本协方差算子与惩罚算子的函数。
  2. **偏差-方差分解**：将估计误差分解为正则化引入的确定性偏差项与由样本算子替代总体算子引入的随机波动项。
  3. **极小极大下界构造**：利用 Fano 引理构造特定的参数局部假设检验子集，证明任何估计量无法超越给定速率。
  4. **Bahadur 余项控制**：对非线性项进行 Taylor 展开，利用经验过程的 chaining 技术和算子的谱截断，控制高阶余项在 sup-norm 下的概率界。
- **最关键的技巧性引理或"跳跃点"**：将泛函空间中的估计误差映射到 Banach 空间值随机变量，利用 sup-norm 下的收敛建立 Bahadur 表达。难点在于积分算子的逆（或伪逆）在经验测度下的连续性，作者通过正则化稳定算子的逆，并精细控制正则化偏差与随机波动的阶数，使得余项严格小于 $n^{-1/2}$。
- **数学工具评价**：是经典 RKHS 框架与近代 Banach 值经验过程理论的精巧组合，特别是将 sup-norm 下的 Bahadur 表达引入函数对函数回归，具有高度的技术迁移价值。

### 五、与研究者兴趣的关联
- **连接到哪个子方向**：非参数/半参数理论中的假设检验与 Bahadur 表达。
- **可借鉴的核心思路或技术工具**：
  1. 推导非参数/半参数估计量 Bahadur 表达的技术，特别是如何通过欠平滑控制无限维余项 $R_n$ 使其不破坏 $n^{-1/2}$ 推断。
  2. 基于 Banach 值随机变量 sup-norm 收敛的 bootstrap 推断框架，可直接迁移至高维/无限维因果推断参数（如连续处理效应、泛函 IV）的联合检验。
- **值得精读的关键参考文献**：
  1. Dette, H., & Sahr, B. (2021). *On the Bahadur representation of U-statistics for dependent data*. （理解 Banach 值 U-统计量/经验过程的 sup-norm 极限理论，对高阶推断至关重要）。
  2. Yuan, M., & Cai, T. T. (2010). *A reproducing kernel Hilbert space approach to functional linear regression*. （RKHS 泛函回归极小极大理论的奠基性工作，理解算子谱与收敛速率的内在联系）。

### 六、延伸思考与练习
- **假设扰动**：若修改平滑性假设，假设 $\beta(s,t)$ 仅有有限阶导数（不属于高阶Sobolev空间），极小极大速率将退化，Bahadur表达中的余项 $R_n$ 可能不再是 $o_P(n^{-1/2})$，此时需要引入更高阶的 Bahadur 表达（对应 higher-order U-statistics 技术）或改变正则化策略（如自适应惩罚）。
- **开放问题**：如何将此框架扩展至带有内生性的函数对函数回归，即引入工具变量函数？此时算子的可逆性将面临严重挑战，可能需要结合 Proximal causal inference 的泛函扩展。
- **理解检测题**：假设你有一个半参数模型，其无穷维干扰参数也属于RKHS，请简述如何利用本文的 Bahadur 表达思路，推导该模型中目标参数估计量的有效影响函数，并说明控制余项使其不干扰一阶线性项的关键步骤。
