# Concentration Inequalities for High-Dimensional Linear Processes with Dependent Innovations

**作者**: Eduardo Fonseca Mendes, Fellipe Lopes Lima Leite  
**来源**: Statistica Sinica  
**主题**: 高维统计 / 随机矩阵  
**相关性**: 8/10  
**链接**: <https://doi.org/10.5705/ss.202023.0364>

### 一、核心问题与贡献
①研究了高维向量线性过程在创新项兼具重尾与时间依赖下的逐元素最大范数浓度不等式问题。②核心工具是结合 mixingale 不等式与 sub-Weibull 尾概率控制，构建了 $l_\infty$ 范数的 Bernstein-type 浓度界。③主要贡献是突破了传统 i.i.d. sub-Gaussian 假设，为高维 VAR(p) 稀疏估计与高维 HAC 协方差估计提供了有限样本误差界。

### 二、基础设定
- **核心概念与符号**：
  - $X_t = \sum_{j=0}^\infty \Psi_j Z_{t-j}$：$p$-维向量线性过程，$\Psi_j$ 为系数矩阵。
  - $Z_t$：创新序列。
  - Sub-Weibull 分布：尾部概率 $P(|Z| > z) \leq 2\exp(-(z/\nu)^\alpha)$，$\alpha \in (0,2]$ 时涵盖重尾，$\alpha=2$ 退化为 sub-Gaussian，$\alpha=1$ 为 sub-Exponential。
  - Mixingale：衡量时间序列依赖结构的渐近无关类，弱于强混合条件，用依赖系数 $\{\zeta_m\}$ 刻画。
  - $\Gamma(h) = E[X_t X_{t-h}^\top]$：lag-$h$ 总体自协方差矩阵；$\hat{\Gamma}(h)$ 为其样本估计。
  - $\|A\|_{\max} = \max_{i,j} |A_{i,j}|$：矩阵逐元素最大范数。

- **关键假设**：
  - **线性过程系数衰减条件**：$\sum_{j=0}^\infty \|\Psi_j\|_{\ell_1}$ 有限或多项式衰减，控制长期依赖的累积效应。
  - **Sub-Weibull 尾部**：创新项 $Z_t$ 的各分量满足参数为 $\alpha$ 的 sub-Weibull 条件。**放宽**：允许 $\alpha < 1$ 的重尾分布，突破了高维时间序列文献中常见的 sub-Gaussian/sub-Exponential 假设。
  - **Mixingale 依赖**：$\{Z_t\}$ 构成 $L_2$-mixingale 序列。**放宽**：允许创新项存在未被线性过程完全捕捉的序列依赖（如条件异方差），突破了 i.i.d. 创新或强混合假设。

- **问题背景**：现有高维时间序列浓度不等式（如 Basu & Michailidis 2015）严重依赖 i.i.d. 与 sub-Gaussian 假设，无法覆盖金融/经济数据中常见的波动聚集与重尾现象。与最相关文献的区别：①相比 Basu & Michailidis (2015) 基于 VAR 谱半径与 i.i.d. sub-Gaussian 的 $l_\infty$ 界，本文处理一般线性过程与弱依赖重尾创新；②相比 Wong et al. (2020) 对 HAC 的逐元素界，本文提供了更基础的向量和浓度不等式，并直接推导出 $\hat{\Gamma}(h)$ 的界。

### 三、主要定理 / 核心结果

**定理 1：线性过程的 $l_\infty$ 范数浓度不等式**
1. **原文陈述**：在 sub-Weibull 与 mixingale 假设下，存在常数 $C$ 使得 $P\left( \left\| \sum_{t=1}^n X_t \right\|_\infty \geq v \right) \leq C p \exp\left( -c \min\left( \frac{v^2}{n \sigma^2}, \frac{v^\alpha}{\nu^\alpha} \right) \right) + \text{mixingale tail}$，其中 $\sigma^2$ 涉及长期方差与依赖系数。
2. **直观解释**：即使创新项有重尾和时间依赖，线性过程向量和在各坐标上的最大值仍具有类似高维 sub-Weibull 的收敛行为，浓度界呈现方差项与尾部项取最小值的 Bernstein 型特征。
3. **技术难点**：解决 sub-Weibull 矩生成函数不存在（$\alpha<1$）时的尾部分离，以及 mixingale 依赖结构下鞅差不等于独立和的方差累积问题。
4. **适用条件与局限**：需 $\log p / n \to 0$ 且 $\Psi_j$ 衰减足够快；若 $\alpha$ 极小（极重尾），浓度界的指数衰减极慢，实质上可能无法支撑高维推断。

**定理 2：lag-$h$ 自协方差矩阵的逐元素最大范数浓度界**
1. **原文陈述**：$\|\hat{\Gamma}(h) - \Gamma(h)\|_{\max} = O_p\left( \sqrt{\frac{\log p}{n}} \right)$，常数项隐含依赖 sub-Weibull 参数与 mixingale 系数。
2. **直观解释**：样本自协方差矩阵的逐元素最大偏差收敛率不受重尾和弱依赖的破坏，仍保持 $\sqrt{\log p / n}$ 的极小化最优速率。
3. **技术难点**：将 $\hat{\Gamma}(h)$ 的二次型交叉项转化为线性过程内积，并利用 Cauchy-Schwarz 将其控制转化为对 $l_\infty$ 范数的控制。
4. **适用条件与局限**：要求 $E[X_t X_{t-h}^\top]$ 存在且有限，极重尾下（如 $\alpha \leq 1/2$）二阶矩可能不存在，此时定理失效。

### 四、证明框架 / 方法设计

- **证明主干逻辑**：截断 + 解耦 + mixingale 不等式 + union bound。
- **拆解关键逻辑步骤**：
  1. **截断与中心化**：对 sub-Weibull 创新项设定阈值 $\tau_n$ 进行截断，分离大值与小值，构造有界随机变量以应用矩生成函数技术。
  2. **依赖结构解耦**：利用 mixingale 性质，将截断后的时间序列和转化为鞅差序列的叠加，应用鞅的 Azuma-Hoeffding/Bernstein 型不等式。
  3. **Bernstein-type 界构造**：结合 sub-Weibull 的矩界与 mixingale 的依赖系数，构造方差项与偏差项分离的浓度界，处理截断带来的偏差项。
  4. **Union bound 与 $l_\infty$ 映射**：对 $p$ 个坐标应用 union bound，将标量浓度界扩展至 $l_\infty$ 范数，引入 $\log p$ 惩罚。
  5. **二次型控制**：将 $\hat{\Gamma}(h)$ 的逐元素偏差表示为线性过程的内积，利用步骤 1-4 的结果闭合不等式。
- **最关键的技巧性引理/跳跃点**：**Sub-Weibull 随机变量的截断阈值设定与 mixingale 鞅差逼近的误差平衡**。截断阈值 $\tau_n$ 必须随 $n$ 精心选取：若 $\tau_n$ 过大，截断尾部概率不足；若 $\tau_n$ 过小，截断后的序列失去原有的依赖结构信息。这一步的参数调优是证明的核心难点。
- **数学工具评价**：是经典工具（截断法、mixingale 不等式、Bernstein 界）在更复杂设定下的精巧组合与参数优化，非全新分析框架，但技术执行严密。

### 五、与研究者兴趣的关联
- **连接子方向**：高维时间序列的 debiased ML 推断与 HAC 协方差矩阵估计中的噪声控制。
- **可借鉴的核心思路**：Sub-Weibull + mixingale 的截断-解耦证明策略，可直接迁移到高维 IV 估计或 DML 中存在序列相关与重尾扰动时的 nuisance 估计误差分析；$\|\hat{\Gamma}(h) - \Gamma(h)\|_{\max}$ 的界可直接用于高维 HAC 协方差矩阵的谱范数控制（通过 $\|A\|_2 \leq \sqrt{p}\|A\|_{\max}$ 或 RMT 谱分析）。
- **值得精读的关键参考文献**：
  1. Basu & Michailidis (2015, Annals of Statistics)：高维 VAR 估计的经典文献，对比其 i.i.d. sub-Gaussian 假设，可深刻理解本文放宽假设的价值与技术代价。
  2. Wong et al. (2020, JASA)：高维 HAC 估计，本文的 entrywise max bound 是对该文理论基础的强化与推广，对您关注的高维推断直接相关。

### 六、延伸思考与练习
- **假设扰动**：若将 mixingale 假设放宽为长记忆过程（如 $\Psi_j$ 多项式衰减且系数不可和），结论的收敛速度会如何退化？技术上需要引入何种分数阶微积分或长记忆鞅不等式？
- **开放问题**：如何将此 $l_\infty$ 浓度界推广到矩阵谱范数 $\|\cdot\|_{op}$，以直接服务于高维 RMT 中的谱分析？是否需要额外的谱结构假设（如谱下界）？
- **理解检测题**：假设你要对高维时间序列构造一个 Debiased Lasso 估计量，其一步修正项包含 $\hat{\Gamma}(0)^{-1}$ 和样本自协方差向量。请说明本文的 Theorem 2 如何帮助你控制该修正项的 $l_\infty$ 误差，从而保证渐近正态性所需的 $\sqrt{n}$ 相合性？
