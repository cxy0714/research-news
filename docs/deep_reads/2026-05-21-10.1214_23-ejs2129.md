# Tail inference using extreme U-statistics

**作者**: Jochem Oorschot, Johan Segers, Chen Zhou  
**来源**: Electronic Journal of Statistics  
**主题**: 其他  
**相关性**: 9/10  
**链接**: <https://doi.org/10.1214/23-ejs2129>

---

### 一、核心问题与贡献
①研究了核阶数随样本量发散且仅依赖少量最高次序统计量的极端 U-统计量的渐近理论。②核心工具是基于位置-尺度不变核的 Hoeffding 方差分解与 Hájek 投影。③证明了极端 U-统计量的渐近方差虽与 Hájek 投影一致，但证明必须处理高阶项贡献；并据此构造了渐近正态的极端 Pickands U-估计量，填补了极值分析中 block maxima 与 peaks-over-threshold 方法间的空白。

### 二、基础设定
- **核心概念与符号**：
  - $U_n = \binom{n}{m_n}^{-1} \sum_{1 \le i_1 < \dots < i_{m_n} \le n} h_{m_n}(X_{i_1}, \dots, X_{i_{m_n}})$：极端 U-统计量，核阶数 $m_n \to \infty$。
  - $h_{m_n}$：核函数，仅依赖前 $k$ 个最大次序统计量（$k$ 固定）。
  - Location-scale invariant kernel：满足 $h_{m_n}(a + b x_1, \dots, a + b x_{m_n}) = h_{m_n}(x_1, \dots, x_{m_n})$ 对 $\forall a \in \mathbb{R}, b > 0$ 成立。
  - Hájek projection：$\hat{U}_n = \sum_{i=1}^n \mathrm{E}[U_n | X_i] - (n-1)\mu_n$。
  - Extreme value index $\gamma$ 与 Pickands estimator。
- **关键假设**：
  - **Domain of attraction ($F \in D(G_\gamma)$)**：分布尾部满足极值吸引场条件，是极值理论的基石假设。
  - **Kernel dependence on top order stats**：核仅依赖前 $k$ 个最大次序统计量。与经典 U-统计量（核依赖所有 $m_n$ 个变量）截然不同，这是"极端"的来源，使得 $m_n \to \infty$ 时计算可行。
  - **Location-scale invariance**：核的位置尺度不变性。与已有文献相比，这是一个强结构假设，但它是将高维极值问题降维至固定 $k$ 维空间的关键，保证了估计量在极值分布族下的不变性。
- **问题背景**：
  - 极值分析中，Block Maxima (BM, 对应 $m_n$ 中取最大，$k=1$) 和 Peaks-Over-Threshold (POT, 对应阈值超额) 是两大对立框架。极端 U-统计量介于两者之间（利用前 $k$ 个最大值，$k \ge 1$）。
  - 与经典 U-统计量文献（如 Arcones 1998 处理 $m_n \to \infty$ 但核依赖所有变量）的区别：本文核仅依赖极值，导致 Hoeffding 分解中高阶项的衰减行为发生质变。
  - 与极值文献（如 Drees 1998）的区别：将极值估计统一进 U-统计量框架，提供了更一般的渐近理论视角。

### 三、主要定理 / 核心结果
1. **原文陈述**：在 $F \in D(G_\gamma)$ 且核 $h_{m_n}$ 满足位置-尺度不变性及仅依赖前 $k$ 个最大次序统计量的条件下，$\sigma_n^{-1}(U_n - \theta_n) \stackrel{d}{\to} N(0,1)$，其中渐近方差 $\sigma_n^2$ 等于 Hájek 投影的方差。
2. **直观解释**：虽然 U-统计量是高度非线性的组合且阶数发散，但其波动主要由一阶投影（线性项）主导。然而，高阶项在方差计算中虽渐近可忽略，但在证明过程中不能直接截断，因为它们在中间步骤的协方差结构中起关键作用，不能简单抛弃。
3. **解决了什么技术难点**：解决了 $m_n \to \infty$ 时，经典 Hoeffding 分解中高阶项方差和不为零（或不可直接忽略）的问题。由于核仅依赖极值，变量间的依赖结构在尾部发生质变，传统矩方法或退化核理论失效。
4. **适用条件与局限**：位置-尺度不变性是核心条件，限制了核的选取（例如排除了直接使用样本方差作为核）；仅适用于分布尾部满足极值吸引场的情况；渐近方差的具体计算仍需依赖核的具体形式与极值指标 $\gamma$。

### 四、证明框架 / 方法设计
- **证明主干逻辑**：基于 Hoeffding 分解与极值理论的结合，通过不变性降维精细控制高阶项的协方差结构。
- **拆解为 3-5 个关键逻辑步骤**：
  1. **Hoeffding 分解与方差展开**：将 $U_n - \theta_n$ 分解为 Hájek 投影（一阶项）与退化核的高阶项之和，展开总方差为各项协方差之和。
  2. **极值次序统计量的联合渐近分布**：利用 $F \in D(G_\gamma)$，将前 $k$ 个最大次序统计量的联合分布转化为 Poisson 过程点过程的表示，处理 $m_n \to \infty$ 下的尾部相依结构。
  3. **高阶项方差的控制**：证明高阶项对总方差的贡献渐近为 0。利用位置-尺度不变性，将核函数转化为仅依赖归一化极值间距的函数，从而将 $m_n$ 维的积分问题转化为固定 $k$ 维的积分问题。
  4. **Lindeberg 条件验证**：对 Hájek 投影项验证中心极限定理的 Lindeberg 条件，结合高阶项的渐近可忽略性，完成渐近正态性的证明。
- **最关键的技巧性引理或"跳跃点"**：**利用位置-尺度不变性对高阶项的降维处理**。在经典 U-统计量中，$m_n \to \infty$ 时高阶项方差通常直接趋于 0。但在极端 U-统计量中，高阶项的方差衰减速度与一阶项方差衰减速度的比值需要极精细的界。作者利用不变性，将核中 $m_n$ 个变量的信息压缩到前 $k$ 个极值的相对间距上，使得原本随 $m_n$ 发散的求和与积分转化为固定维度的极值分布期望计算，这是整个证明的"跳跃点"。
- **数学工具评价**：是经典 Hoeffding 分解与极值理论（Poisson 过程表示）的巧妙组合。并非全新分析框架，但在处理 $m_n \to \infty$ 与极值依赖的交互作用时展现了极高的技术技巧，特别是对退化 U-统计量在极值域的降维处理。

### 五、与研究者兴趣的关联
- **连接到哪个子方向**：higher-order U-statistics 理论（特别是核阶数发散 $m_n \to \infty$ 时的渐近性质与 Hoeffding 分解高阶项的处理）。
- **可借鉴的核心思路或技术工具**：
  1. **Hájek 投影与高阶项的分离处理**：当 $m_n \to \infty$ 时，即使渐近方差等于 Hájek 投影的方差，也不能直接丢弃高阶项，必须严格证明其渐近可忽略性。这一思路对研究高维/高阶 U-统计量的推断至关重要。
  2. **利用不变性降维**：位置-尺度不变性将发散维度的核函数转化为固定维度的统计量，这一"不变性降维"思想可迁移到其他具有对称性或不变结构的高维统计问题中（如高维位置-尺度分布的半参数推断）。
- **值得精读的关键参考文献**：
  1. Arcones (1998) "Asymptotic theory for U-statistics with varying kernels"：对比经典 $m_n \to \infty$ 理论，理解极端设定下高阶项行为的差异。
  2. Drees (1998) "On smooth statistical tail functionals"：极值统计量渐近正态性的经典框架，理解极值域下经验过程的处理。

### 六、延伸思考与练习
- **假设扰动**：若取消核函数的"位置-尺度不变性"假设（例如核函数依赖于绝对尺度而非相对间距），结论会如何变化？技术上需要什么新工具？（提示：高阶项的降维失效，可能需要发展新的矩不等式或经验过程界来控制 $m_n$ 维积分，或引入更严格的尾部衰减条件）。
- **开放问题**：如何构造极端 U-统计量的自助法推断？由于 $m_n \to \infty$ 且依赖极值，传统自助法在尾部的一致性可能失效，可能需要引入 subsampling 或特定的极值 bootstrap。
- **理解检测题**：考虑一个极端 U-统计量，其核为 $h_{m_n}(x_1, \dots, x_{m_n}) = \log(X_{(m_n)} - X_{(m_n-1)})$（仅依赖前两个最大次序统计量）。请说明在证明其渐近正态性时，为什么不能直接对 Hoeffding 分解的第一项（Hájek 投影）应用中心极限定理并声称剩余项方差为 0，而必须显式计算高阶项的衰减阶数？
