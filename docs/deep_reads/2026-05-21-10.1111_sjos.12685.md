# Testing the missing at random assumption in generalized linear models in the presence of instrumental variables

**作者**: Rui Duan, C. Jason Liang, Pamela A. Shaw, Cheng Yong Tang, Yong Chen  
**来源**: Scandinavian Journal of Statistics  
**主题**: 数理统计 / 假设检验  
**相关性**: 8/10  
**链接**: <https://doi.org/10.1111/sjos.12685>

---

### 一、核心问题与贡献
①研究了在广义线性模型(GLM)下，如何利用工具变量(IV)对缺失机制是缺失完全随机(MAR)还是非随机缺失(MNAR)进行假设检验。②核心方法是构造仅在MAR下一致的估计量与在MNAR下也一致的IV估计量，并基于两者的差异度量构建Wald型检验统计量。③证明了该差异度量在MAR原假设下概率极限为零且统计量服从$\chi^2$分布，在MNAR备择假设下偏离零，从而实现了数据驱动的缺失机制选择。

### 二、基础设定
- **核心概念与符号**：
  - $Y$：带缺失的响应变量；$X$：完全观测的协变量；$R$：缺失指示变量（$R=1$观测，$R=0$缺失）。
  - $Z$：工具变量，影响$R$但不直接影响$Y$。
  - $\beta$：GLM中的回归系数真值。
  - $\hat{\beta}_{MAR}$：基于完全病例分析(CC)或逆概率加权(IPW)的估计量，仅在MAR下一致。
  - $\hat{\beta}_{IV}$：利用IV构造的估计量，在MAR和MNAR下均一致。
  - $D_n = \hat{\beta}_{MAR} - \hat{\beta}_{IV}$：差异度量。

- **关键假设**：
  1. **IV相关性**：$Z \not\perp R \mid X$（IV与缺失指示强相关，保证IV估计量的识别性与弱工具变量规避）。
  2. **IV排他性**：$Z \perp Y \mid X$（给定协变量，IV不影响响应变量，且不通过未观测变量影响$Y$，这是识别MNAR的核心）。
  3. **参数模型设定**：GLM均值结构 $E(Y|X) = \mu(X^T\beta)$ 及缺失机制模型 $P(R=1|X,Z,Y)$ 的特定参数形式。
  - *假设对比*：传统缺失数据文献直接假设MAR成立（因MAR不可检验），本文通过引入IV的排他性与相关性，将MAR从不可检验的暗箱假设转化为可检验的零假设，放宽了分析框架的盲区。

- **问题背景**：传统缺失数据分析直接假设MAR，但MAR本质不可检验（因缺失的$Y$未观测）。已有文献多通过敏感性分析处理MNAR，缺乏直接的假设检验方法。与最相关文献的区别：相比仅做敏感性分析（如Rotnitzky et al., 1998），本文提供了正式的假设检验框架；相比非参数检验，本文在GLM下利用IV的过度识别构造了具有精确渐近分布的检验统计量。

### 三、主要定理 / 核心结果
1. **原文陈述**：在$H_0: \text{MAR}$下，$\sqrt{n}(\hat{\beta}_{MAR} - \hat{\beta}_{IV}) \xrightarrow{d} N(0, \Sigma)$，检验统计量 $T_n = n D_n^T \hat{\Sigma}^{-1} D_n \xrightarrow{d} \chi^2_p$；在$H_1: \text{MNAR}$下，$D_n \xrightarrow{p} c \neq 0$，$T_n$发散。
2. **直观解释**：这是Hausman检验思想在缺失数据机制检验中的推广。在MAR下，$\hat{\beta}_{MAR}$有效（或至少一致），$\hat{\beta}_{IV}$也一致，两者差异仅反映抽样误差；在MNAR下，$\hat{\beta}_{MAR}$因缺失非随机而产生偏误，但IV排他性保证了$\hat{\beta}_{IV}$仍一致，两者差异反映了缺失机制的系统性偏差。
3. **解决了什么技术难点**：解决了在缺失数据下联合推导两个具有不同有效性与一致性属性的估计量的渐近分布，特别是处理了IV估计量在缺失机制模型可能误判时的稳健性边界。
4. **适用条件与局限**：必须存在有效且强相关的IV $Z$；GLM的参数模型设定需正确；若IV排他性违背，检验size将扭曲。假设条件较强，非参数或弱假设拓展空间大。

### 四、证明框架 / 方法设计
- **证明主干逻辑**：估计方程理论 + 联合渐近正态性 + Delta方法 / 阻塞矩阵求逆。
- **拆解为4个关键逻辑步骤**：
  1. 构造MAR估计方程与IV估计方程，证明在MAR下两者均无偏，在MNAR下仅IV方程无偏。
  2. 将两类估计方程堆叠为联合估计方程组，利用M估计量理论，推导联合估计量 $(\hat{\beta}_{MAR}, \hat{\beta}_{IV})$ 的渐近正态性。
  3. 通过Delta方法（或对阻塞矩阵分块求逆），提取差异度量 $D_n = \hat{\beta}_{MAR} - \hat{\beta}_{IV}$ 的边际渐近分布，计算其渐近方差 $\Sigma$。
  4. 构造二次型统计量 $T_n$，证明其在$H_0$下服从$\chi^2$分布，在$H_1$下非中心参数发散至无穷。
- **最关键的技巧性引理或"跳跃点"**：在MNAR机制下证明IV估计量一致性的逻辑。IV排他性假设将缺失机制模型与结果模型解耦，使得即使$Y$缺失，$P(R|X,Z)$的估计仍不受未观测$Y$的干扰，这是差异度量在$H_1$下具有非零概率极限的基石。
- **数学工具评价**：是经典M估计/估计方程渐近理论与Hausman检验思想的巧妙组合，创新点不在于纯数学工具的突破，而在于将"过度识别约束检验"的逻辑迁移到不可观测缺失机制的识别上。

### 五、与研究者兴趣的关联
- **连接到哪个子方向**：因果推断中的假设检验，具体为"利用工具变量/负控制对不可观测混淆或缺失机制的敏感性分析与假设检验"。
- **可借鉴的核心思路或技术工具**：将不可检验的因果/缺失假设（如无未观测混淆、MAR）转化为存在辅助变量（IV、Negative Control）下的过度识别约束，通过比较不同识别策略下估计量的差异构建检验统计量。这一思路可直接迁移到Proximal CI中对U可测性或无混淆性的检验。
- **值得精读的关键参考文献**：
  1. Hausman, J. A. (1978). "Specification tests in econometrics." *Econometrica*. （理解差异度量检验的源头思想与方差计算逻辑）
  2. Rotnitzky, A., et al. (1998). "Semiparametric regression for repeated outcomes with nonignorable nonresponse." *JASA*. （理解MNAR机制下IV识别与半参数理论的基础）

### 六、延伸思考与练习
- **假设扰动**：若IV排他性假设 $Z \perp Y \mid X$ 被违背（即$Z$是无效工具变量），差异度量在MAR下的概率极限是否仍为0？此时检验的size会发生何种扭曲？技术上需要引入何种稳健方差估计或弱IV推断工具？
- **开放问题**：如何在高维协变量设定下（$p \gg n$）构造MAR的检验？当IV或缺失机制模型涉及高维参数时，如何结合Debiased ML或Cross-fitting保证差异度量的渐近正态性？
- **理解检测题**：假设存在一个完美的IV $Z$，但在MNAR机制下，缺失不仅依赖于$Y$还依赖于不可观测的混淆$U$（即$R = f(X, Z, Y, U)$）。若$Z$仅影响$R$但不影响$U$，本文构造的IV估计量是否仍然一致？请用估计方程的期望条件说明理由。
