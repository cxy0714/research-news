# Misspecified Bernstein–von Mises theorem for hierarchical models

**作者**: Geerten Koers, Botond Szabó, Aad van der Vaart  
**来源**: Bernoulli  
**主题**: 非参数 / 半参数  
**相关性**: 8/10  
**链接**: <https://doi.org/10.3150/25-bej1940>

---

### 一、核心问题与贡献
①研究了在模型误设且非独立同分布的层次模型中，含无限维讨厌参数的有限维兴趣参数的 Bernstein–von Mises (BvM) 定理。②核心工具是基于误设下的 efficient influence function 结合局部渐近正态性（LAN）与经验过程理论，分析后验分布的渐近行为。③主要结论是证明了在模型误设下后验仍渐近正态，其方差由误设下的 efficient influence function 决定（而非真实模型下的），并将理论拓展至非线性积分算子与 PDE 约束逆问题，保证了误设下 Bayes 置信域的频率覆盖有效性。

### 二、基础设定
- **核心概念与符号**：
  - $P_0$：真实数据生成分布（不属于模型族）。
  - $P_{\theta,\eta}$：参数化的概率模型族，$\theta \in \Theta \subset \mathbb{R}^d$ 为兴趣参数，$\eta \in H$ 为无限维讨厌参数。
  - $\theta_*$：KL 散度意义下的伪真实参数，即 $\theta_* = \arg\min_{\theta} \inf_{\eta} P_0 \log(p_0/p_{\theta,\eta})$。
  - $\tilde{\psi}_\theta$：误设下的 efficient influence function，决定了 $\theta$ 的误设半参数效率界。
  - $\tilde{I}_{\theta_*}$：基于 $\tilde{\psi}_\theta$ 计算的有效信息算子。

- **关键假设**：
  - **Misspecification setting ($P_0 \notin \{P_{\theta,\eta}\}$)**：模型不包含真实分布。统计学含义是推断目标从真实参数转变为 KL 投影下的伪真实参数，score 函数在 $P_0$ 下期望非零。
  - **Local asymptotic normality (LAN) for misspecified models**：对数似然在伪真实参数处满足特定展开。统计学含义：保证局部模型仍具备渐近正态结构，是 BvM 的频率基石；相比正确设定模型，展开的方差项需替换为误设下的有效信息算子。
  - **Nuisance parameter estimation rate**：讨厌参数 $\eta$ 需以足够快的速率收敛（如 $o_P(n^{-1/4})$ 或特定 PDE 下的最优收敛率）。统计学含义：确保讨厌参数的估计误差不会干扰兴趣参数的 $\sqrt{n}$ 收敛速率与渐近分布；与正确设定下的半参数推断要求一致，但此处需在 KL 投影邻域内成立。
  - **Prior mass condition**：先验在 $\theta_*$ 邻域内的质量下界条件。统计学含义：确保后验分布能以足够概率集中在伪真实参数周围。

- **问题背景**：已有 BvM 定理多局限于正确指定的独立同分布模型，或无讨厌参数的纯参数误设模型。对于层次模型（非 i.i.d.）且存在无限维讨厌参数的误设情况，BvM 是否成立尚属空白。与最相关的文献区别：Kleijn & van der Vaart (2012) 处理了纯参数误设下的 BvM，但无无限维讨厌参数；Castillo & Nickl (2013) 处理了非参数/半参数下的 BvM，但假设模型正确设定。本文在两者交叉的硬核地带建立理论。

### 三、主要定理 / 核心结果

1. **原文陈述**：在层次模型下，若对数似然满足误设 LAN 条件、先验满足局部支撑条件、且讨厌参数可估，则有限维兴趣参数 $\theta$ 的后验分布满足：$\Pi(\sqrt{n}(\theta-\theta_*) \in \cdot | X^{(n)}) \rightsquigarrow N(0, \tilde{I}_{\theta_*}^{-1})$。
2. **直观解释**：即使模型完全猜错，只要数据量足够，Bayesian 后验依然会收敛到一个正态分布。但这个正态分布的方差不再是由真实模型决定的 Cramer-Rao 下界给出，而是由"在错误模型流形上能做的最好推断"（即误设 efficient influence function）决定。这保证了 Bayesian credible interval 实际上是对伪真实参数 $\theta_*$ 的有效频率置信区间。
3. **解决了什么技术难点**：解决了非 i.i.d. 层次模型中，对数似然二阶展开受讨厌参数干扰，且误设导致 score 函数均值非零，从而无法直接套用标准 BvM 证明框架的问题；特别是处理了 PDE 约束下非线性算子（如平方积分算子）带来的非正则映射问题。
4. **适用条件与局限**：必须满足误设下的 LAN 展开和讨厌参数的充分收敛速率。局限在于对非线性算子（如 PDE）的正则性要求极高，且要求模型误设程度不破坏局部渐近结构（如信息算子 $\tilde{I}_{\theta_*}$ 必须可逆）。

### 四、证明框架 / 方法设计

- **证明主干逻辑**：基于 testing approach 和 LAN 展开的构造法。
- **拆解为 3-5 个关键逻辑步骤**：
  1. **局部渐近展开**：将误设下的对数似然比在伪真实参数 $(\theta_*, \eta_*)$ 处展开，利用 LAN 条件分离出由 $\theta$ 引起的线性主项和由讨厌参数 $\eta$ 引起的余项。
  2. **余项控制**：利用讨厌参数的收敛速率和经验过程理论（如局部渐近经验过程界限），证明 $\eta$ 的估计误差对 $\theta$ 局部渐近分布的影响是 $o_P(1/\sqrt{n})$ 的。
  3. **构造局部检验**：构造指数族检验统计量，证明在 $\theta_*$ 的局部邻域外，后验概率指数级衰减。
  4. **局部渐近正态性收敛**：在局部邻域内，后验分布由 LAN 主导项决定，结合先验局部线性近似，得出后验收敛于方差为 $\tilde{I}_{\theta_*}^{-1}$ 的正态分布。
- **最关键的技巧性引理或"跳跃点"**：在非 i.i.d. 且误设的设定下，将 score 函数的非零均值（由于误设）与讨厌参数的估计误差解耦。具体而言，如何证明基于误设 influence function 构造的 one-step estimator 与后验均值具有相同的渐近行为，这是连接 Bayesian 后验与频率有效性的关键跳跃点。
- **数学工具评价**：是经典 BvM testing approach 与半参数效率理论的深度结合。创新点在于将误设下的 efficient influence function 嵌入到局部检验和 LAN 展开的框架中，处理了非 i.i.d. 层次模型带来的依赖结构。

### 五、与研究者兴趣的关联
- **连接到哪个子方向**：Semiparametric efficiency under misspecification（误设下的半参数效率理论）与 Semiparametric BvM（半参数贝叶斯渐近理论）。
- **可借鉴的核心思路或技术工具**：
  1. 误设下 efficient influence function 的构造与性质，可直接迁移到因果推断中处理 outcome model 或 propensity score misspecification 时的稳健推断与 sensitivity analysis。
  2. 非线性算子（PDE）作为 nuisance parameter 的处理方式，对研究复杂因果结构（如带潜变量的纵向因果模型或连续时间处理）中的半参数推断有启发。
- **值得精读的关键参考文献**：
  1. Kleijn & van der Vaart (2012), "The Bernstein-Von-Mises theorem under misspecification"：本文的直接理论基石，理解 misspecified BvM 的必读文献。
  2. Castillo & Nickl (2013), "Nonparametric Bernstein–von Mises theorems"：非参数/半参数 BvM 的经典文献，提供了处理无限维讨厌参数的先验与检验框架。

### 六、延伸思考与练习
- **假设扰动**：若假设讨厌参数 $\eta$ 的收敛速率慢于 $n^{-1/4}$（例如非参数收敛速率 $n^{-\beta}$ 且 $\beta < 1/4$），结论会如何变化？技术上需要引入什么新工具？（提示：可能需要 higher-order influence function 或 debiasing 技术来修正一阶偏差）。
- **开放问题**：如何将此 misspecified BvM 理论推广到高维参数设定（$\dim(\theta) \to \infty$），或者当真实分布 $P_0$ 位于模型流形的边界时？
- **理解检测题**：在正确指定的半参数模型中，efficient influence function 的方差达到了半参数有效下界；而在本文的误设模型中，由误设 efficient influence function 决定的方差 $\tilde{I}_{\theta_*}^{-1}$ 是否仍具有某种"有效性"（optimality）？请解释这种有效性的频率意义。
