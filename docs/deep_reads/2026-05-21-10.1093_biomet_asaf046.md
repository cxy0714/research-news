# Goodness-of-fit tests for linear non-Gaussian structural equation models

**作者**: D Schkoda, M Drton  
**来源**: Biometrika  
**主题**: 数理统计 / 假设检验  
**相关性**: 8/10  
**链接**: <https://doi.org/10.1093/biomet/asaf046>

---

### 一、核心问题与贡献
①研究了线性非高斯结构方程模型在有/无潜在混杂下的模型假设检验问题。②核心工具是利用线性结构方程隐含的二阶及高阶矩之间的代数关系，将其转化为对矩构造的矩阵和张量的秩约束检验，并采用不完全U-统计量与multiplier bootstrap实现推断。③主要贡献是建立了LiNGAM设定下基于矩约束的拟合优度检验框架，解决了高阶矩张量秩约束的估计与渐近零分布推导的计算与理论难题。

### 二、基础设定
- **核心概念与符号**：
  - $X = BX + \epsilon$：线性非高斯结构方程模型，$B$为严格下三角矩阵（无环性）。
  - $\mathcal{M}_r$：$r$阶矩张量，其元素为 $\mu_{i_1,\dots,i_r} = E[X_{i_1}\cdots X_{i_r}]$。
  - 秩约束：$\text{rank}(\mathcal{M}_r) \le p$（将张量适当矩阵化后，其秩不超过变量维度$p$）。
  - 不完全U-统计量：通过随机抽取指标子集计算的U-统计量，用于控制高阶计算复杂度。
- **关键假设**：
  - **线性**：变量间的因果机制为线性。统计学含义：这是矩张量出现秩亏缺的代数根源；若为非线性，秩约束不再成立。
  - **非高斯性**：误差项$\epsilon_j$非高斯。统计学含义：保证高阶（$r \ge 3$）矩张量携带非冗余信息；若为高斯，高阶累积量为零，秩约束退化为平凡条件。
  - **误差独立**：$\epsilon_j$之间相互独立。统计学含义：使得$X$的矩张量可分解为$\epsilon$的矩张量的线性组合，从而触发秩的上界约束。相比传统基于条件独立的DAG检验，该假设结合非高斯性实现了方向的可识别性。
- **问题背景**：
  现有LiNGAM的因果发现方法（如ICA-based或DirectLiNGAM）均假设模型本身正确，但缺乏对"模型是否误设（如存在非线性或未观测混杂）"的拟合优度检验。传统基于条件独立的图模型检验无法识别因果方向，而本文通过高阶矩的代数约束直接检验LiNGAM的核心假设。与Shimizu et al. (2006)的LiNGAM估计文献相比，本文解决的是上游的模型设定检验问题；与一般的DAG拟合优度检验相比，本文利用了非高斯性带来的高阶矩约束。

### 三、主要定理 / 核心结果
1. **原文陈述**：在无潜在混杂的LiNGAM下，对任意$r \ge 3$，由$r$阶矩构造的矩阵$\mathcal{M}_r^{(mat)}$的秩不超过$p$；若存在潜在混杂，该秩约束表现为更一般的代数秩亏缺。检验LiNGAM假设等价于检验$\mathcal{M}_r^{(mat)}$的所有$(p+1)\times(p+1)$子行列式为0。
2. **直观解释**：由于$X = (I-B)^{-1}\epsilon$且$\epsilon$的$p$个分量独立，$X$的$r$阶矩张量本质上是$\epsilon$的$r$阶矩张量在线性映射$(I-B)^{-1}$下的像。独立分量的矩张量具有极简的秩结构（秩为$p$），线性映射不增加秩，因此$X$的矩张量矩阵化后秩至多为$p$。若模型非线性或有未观测混杂，这种独立分解不存在，秩将大于$p$。
3. **解决了什么技术难点**：将抽象的因果模型误设问题转化为具体的代数秩约束检验，并解决了高阶矩子行列式估计的统计与计算难题。
4. **适用条件与局限**：必须依赖非高斯假设（高斯分布下检验失效）；无混杂设定下要求DAG的因果充分性；子行列式数量随$p$指数增长，当$p$较大时面临严重多重检验问题。

### 四、证明框架 / 方法设计
- **证明主干逻辑**：构造法 + U-统计量投影理论 + 经验过程。
- **拆解为 3-5 个关键逻辑步骤**：
  1. **代数表征**：从$X = A\epsilon$出发，推导$r$阶矩张量矩阵化后的列空间被$A$的列空间张成，从而证明$\text{rank}(\mathcal{M}_r^{(mat)}) \le p$。
  2. **统计量构造**：将秩约束转化为子行列式为0的约束，构造基于样本矩的U-统计量来估计这些子行列式。
  3. **计算降维**：引入不完全U-统计量，通过随机抽取组合索引计算核函数，证明在适当抽样率下其方差与完全U-统计量同阶。
  4. **渐近分布与Bootstrap**：利用Hadamard微分和Delta方法，证明子行列式估计量在零假设下渐近正态；通过multiplier bootstrap构造条件分位数，逼近退化U-统计量的复杂零分布。
- **最关键的技巧性引理或"跳跃点"**：
  将张量秩检验转化为**子行列式检验**是最大的跳跃。直接对矩阵的奇异值进行检验在零假设下会遇到奇异值非中心化参数为0的问题（导致渐近分布为多维chi分布的极小值，分位数难以估计），而子行列式为0是等价的代数条件，且作为多项式函数，其U-统计量估计量的渐近理论更成熟，配合multiplier bootstrap可完美避开奇异值分布的推导。
- **数学工具评价**：是经典工具（U-统计量投影、Delta方法、Multiplier bootstrap）的巧妙组合。将因果图模型的代数几何特征与经典的非参数检验框架桥接，极具方法论美感。

### 五、与研究者兴趣的关联
- **连接到哪个子方向**：高阶U-统计量的假设检验与因果推断（LiNGAM）的交叉；具体为"基于矩约束的因果模型拟合优度检验"。
- **可借鉴的核心思路或技术工具**：
  1. **高阶矩张量的秩约束作为模型误设表征**：这一思路可推广至其他半参数模型（如测量误差模型、潜在变量模型）的拟合优度检验，只要隐变量具有低秩结构。
  2. **Incomplete U-statistics 估计高阶多项式核**：当需要估计高阶交互矩或子行列式时，不完全U-统计量是解决计算爆炸的标准范式，可直接迁移到其他高维/高阶推断问题中。
  3. **Multiplier bootstrap 处理退化U-统计量**：在零假设下，子行列式期望为0，U-统计量是退化的，直接渐近分布推导困难，multiplier bootstrap 提供了计算上可行的推断路径。
- **值得精读的关键参考文献**：
  1. Shimizu et al. (2006) "A linear non-Gaussian acyclic model for causal discovery" - LiNGAM开山之作，理解非高斯性如何带来可识别性。
  2. Blom (1976) / Lee (1990) 关于 Incomplete U-statistics 的经典文献，理解其方差分解与最优抽样率设定。
  3. Drton & Xiao (2022) "Testing latent variable models via rank constraints" - 本文的直接前作，理解秩约束检验在图模型中的先期探索。

### 六、延伸思考与练习
- **假设扰动**：若将"误差项独立"假设放宽为"误差项存在弱依赖（如给定潜在混杂下的条件独立）"，秩约束会如何变化？技术上需要引入张量的低秩近似（如Tucker rank）而非精确秩约束，推断可能需要结合高维矩阵/张量的谱方法与随机矩阵理论。
- **开放问题**：如何在变量维度$p$随样本量$n$增长的高维设定下进行该拟合优度检验？子行列式的数量随$p$指数增长，多重检验校正将极其保守，如何构造全局的聚合统计量是一个未解决的难题。
- **理解检测题**：假设真实的模型是一个非线性非高斯SEM（例如$X_j = f_j(X_{\text{pa}(j)}) + \epsilon_j$且$f_j$非线性），使用本文基于$r$阶矩张量秩约束的检验统计量，在样本量趋于无穷时，检验的势会趋于1吗？请从矩张量的代数结构解释原因。
