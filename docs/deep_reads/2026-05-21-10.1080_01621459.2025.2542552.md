# Confidence Sets for Causal Orderings

**作者**: Y. Samuel Wang, Mladen Kolar, Mathias Drton  
**来源**: Journal of the American Statistical Association  
**主题**: 因果推断  
**相关性**: 9/10  
**链接**: <https://doi.org/10.1080/01621459.2025.2542552>

---

### 一、核心问题与贡献
①本文研究了可识别加性误差结构方程模型（SEM）下因果序的不确定性量化问题。②核心方法是基于残差 bootstrap 的拟合优度检验反转来构造数据未能拒绝的因果序置信集。③主要贡献是证明了该置信集的渐近有效性，并展示了如何从排序置信集导出祖先关系的子集/超集以及融合模型不确定性的因果效应置信区间。

### 二、基础设定
- **核心概念与符号**：
  - $X = (X_1, \dots, X_p)$：$p$-维随机向量。
  - $\pi$：因果序，为 $\{1, \dots, p\}$ 的一个排列；若 $\pi(i) < \pi(j)$，则 $X_i$ 不能是 $X_j$ 的后代。
  - 加性误差 SEM：$X_j = \sum_{k \in \text{pa}(j)} f_{jk}(X_k) + \epsilon_j$，其中 $\epsilon_j$ 相互独立。
  - $T_n(\pi)$：针对排序 $\pi$ 的拟合优度检验统计量（基于残差独立性）。
  - $\hat{C}_{1-\alpha}$：置信集，$\{\pi : T_n(\pi) \le \hat{c}_{1-\alpha}(\pi)\}$，其中 $\hat{c}_{1-\alpha}(\pi)$ 为 bootstrap 临界值。
- **关键假设**：
  - **加性误差与误差独立性**：$X_j = f_j(\text{pa}(X_j)) + \epsilon_j$，且 $\epsilon_j$ 联合独立。统计学含义：使得因果方向可识别，且残差独立性成为检验排序有效性的等价条件。相比 LiNGAM 放松了非高斯性假设，相比 PC/GES 算法放松了 faithfulness 假设（不追求唯一真值点估计）。
  - **非参数平滑/矩条件**：函数 $f_j$ 满足平滑性条件，误差 $\epsilon_j$ 具有有限矩。统计学含义：保证非参数回归/残差估计的收敛速度，以及 bootstrap 经验过程的渐近有效性。
- **问题背景**：现有因果发现方法多聚焦于点估计（单一 DAG 或马尔可夫等价类），缺乏对因果图核心特征——因果序——的有限样本不确定性量化。与 Shimizu et al. (LiNGAM, 基于非高斯性的点估计) 和 Peters et al. (基于不变性的因果效应估计) 不同，本文不追求单一的图结构点估计，而是通过 test inversion 给出未被数据拒绝的序集合，将因果发现从点估计推向集合推断。

### 三、主要定理 / 核心结果
1. **原文陈述**：在真实因果序 $\pi^0$ 下，基于残差 bootstrap 拟合优度检验反转构造的置信集 $\hat{C}_{1-\alpha}$ 满足 $\lim_{n \to \infty} P(\pi^0 \in \hat{C}_{1-\alpha}) \ge 1-\alpha$（渐近有效性）。
2. **直观解释**：通过 test inversion 构造的因果序集合，在样本量足够大时，能够以至少 $1-\alpha$ 的概率包含真实的因果序。即数据有 $1-\alpha$ 的概率不会"拒绝"真实的生成逻辑。
3. **解决了什么技术难点**：解决了在非参数/半参数 SEM 下，由于非参数拟合偏差导致的残差相依性干扰问题，以及基于残差重构的 bootstrap 经验过程能否有效逼近真实残差驱动的检验统计量分布的问题。
4. **适用条件与局限**：必须依赖加性误差假设（非加性误差下残差不独立，检验失效）；当变量维度 $p$ 较大时，排列空间 $p!$ 呈指数增长，遍历所有排序计算不可行；非参数拟合的收敛速度要求较高，可能需要 undersmoothing 以消除拟合偏差对检验水平的影响。

### 四、证明框架 / 方法设计
- **证明主干逻辑**：经典的 test inversion + bootstrap 经验过程理论。
- **拆解为 3-5 个关键逻辑步骤**：
  1. **构造检验统计量**：对给定排序 $\pi$，按序拟合加性模型得到残差 $\hat{\epsilon}$，构造基于残差独立性的拟合优度统计量 $T_n(\pi)$（如基于距离协方差或 HSIC）。
  2. **建立原假设下的渐近分布**：证明当 $\pi = \pi^0$ 时，$T_n(\pi)$ 弱收敛到某个非退化极限分布。
  3. **残差 bootstrap 的相合性**：证明基于残差重采样构造的统计量 $T_n^*(\pi)$ 的条件分布能渐近逼近 $T_n(\pi)$ 的真实分布。
  4. **覆盖率保证**：由 bootstrap 相合性，$P(T_n(\pi^0) \le \hat{c}_{1-\alpha}(\pi^0)) \to 1-\alpha$，从而 $\pi^0 \in \hat{C}_{1-\alpha}$ 的概率趋于 $1-\alpha$。
- **最关键的技巧性引理或"跳跃点"**：非参数残差 bootstrap 的有效性证明。由于残差 $\hat{\epsilon}$ 包含了非参数拟合偏差，证明基于 $\hat{\epsilon}$ 重采样的经验过程能够耦合真实误差 $\epsilon$ 驱动的经验过程，是整个证明中最具技巧性的地方，通常需要精细的矩控制和耦合技术。
- **数学工具评价**：是经典 test inversion 与经验过程理论的巧妙组合，将因果发现的结构不确定性问题转化为假设检验框架下的多重比较/集合推断问题。

### 五、与研究者兴趣的关联
- **连接到哪个子方向**：因果推断中的 identification 与假设检验交叉，特别是 causal discovery 的 test inversion 与半参数/非参数拟合优度检验。
- **可借鉴的核心思路或技术工具**：将因果结构的识别问题转化为 test inversion 构造置信集的思路，可直接迁移至 **Proximal CI** 中的 negative control 排序或 **IV** 的排他性假设检验中；残差 bootstrap 处理非参数残差独立性的技术，可用于高维/半参数模型的 **sensitivity analysis**（如检验隐变量是否完全被捕获）。
- **值得精读的关键参考文献**：
  1. Romano & Wolf (2005, *Nonparametric bootstrap for hypothesis testing*)：理解 test inversion 构造置信集的严格渐近理论基石。
  2. Shah & Peters (2020, *The Hardness of Conditional Independence Testing*)：理解条件独立性/独立性检验的困难与拟合优度检验的局限，对评估本文方法的假设依赖至关重要。

### 六、延伸思考与练习
- **假设扰动**：若将加性误差假设 $X_j = f_j(\text{pa}(X_j)) + \epsilon_j$ 放松为非加性（如乘性或一般函数 $\epsilon_j = g(X_j, \text{pa}(X_j))$），残差独立性检验失效。技术上需要引入何种条件独立性检验（如基于 RKHS 的条件独立性度量）？在非参数条件下，如何构造相应的 bootstrap 并保证其相合性？
- **开放问题**：当变量维度 $p$ 增长时，如何利用因果图的局部结构（如 V-structure 或紧凑性先验）对排列空间进行剪枝，使得高维情况下的置信集构造在计算上可行？
- **理解检测题**：假设真实模型为 $X_1 \to X_2 \to X_3$（加性误差），若给定一个错误排序 $\pi = (3, 2, 1)$，请描述拟合优度检验统计量 $T_n(\pi)$ 在大样本下为什么会趋向于无穷大（即被拒绝），并指出残差独立性检验在哪一步捕捉到了排序的错误？
