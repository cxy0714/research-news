### 一、子领域定位

- **本文属于天文学的哪一支**：Solar physics / Space weather（太阳物理 / 空间天气）。这个子领域的核心科学问题是：太阳表面的磁场活动如何通过日冕（太阳大气）向外膨胀形成太阳风，进而影响整个日地空间环境乃至地球上的航天器和电网。目前该领域处于“物理模型成熟但计算极度受限，正试图通过数据驱动模型实现业务化实时预报”的转型期。
- **本文在这个子领域里的位置**：它针对的是空间天气预报链条中的“计算瓶颈”切片——即如何跳过耗时的磁流体力学（MHD）日冕模拟，直接从太阳表面观测数据快速推算出0.1 au（约20个太阳半径，日冕外边界）处的太阳风参数，为后续的日地传播模型提供实时初始条件。

### 二、关键术语扫盲

1. **Solar wind（太阳风）**：从太阳日冕持续向外喷射的等离子体流，主要由质子和电子组成，携带磁场，是日地空间环境的主体介质。
2. **Magnetohydrodynamics / MHD（磁流体力学）**：将等离子体视为导电流体，结合流体力学和电磁学方程来描述太阳风运动的物理框架。求解MHD方程组是当前日冕物理模拟的标准范式，但计算代价极高。
3. **0.1 au（0.1天文单位）**：约1500万公里，相当于太阳半径的20倍左右。这是日冕（封闭磁场区）向太阳风（开放磁场区）过渡的典型边界，也是内日球层模型的常规内边界条件输入位置。
4. **Photospheric magnetogram（光球层磁图）**：太阳最底层大气（光球层）的磁场强度分布图，通常是在特定吸收谱线处利用塞曼效应测量得到，是我们能直接获取的最核心的太阳表面观测输入。
5. **GONG (Global Oscillation Network Group)**：一个由全球六台望远镜组成的观测网络，全天候提供太阳光球层的速度场和磁场图（即本文的输入数据源）。
6. **COCONUT model**：COolfluid COroNal UnsTructured模型的缩写，一种基于非结构化网格的先进日冕MHD模拟代码，用于从光球磁图推演日冕和太阳风状态，是本文生成训练标签的“真值”来源。
7. **Carrington rotation（卡林顿旋转）**：太阳自转的基准周期，约27.27天。由于太阳表面特征随自转演变，一个卡林顿旋转周期代表了太阳表面磁场结构经历的一个完整观测周期。
8. **Surrogate model（代理模型）**：在天文和工程中，指用数据驱动方法（如神经网络）拟合一个计算昂贵的物理模拟器的输入-输出映射，以极低成本近似替代原物理模拟器进行推理。
9. **Radial velocity / Number density / Radial magnetic field（径向速度 / 数密度 / 径向磁场）**：太阳风在0.1 au处的三个核心状态变量（动力学和磁流体参数），决定了太阳风的动量、能量和磁压力。
10. **Alfvén speed / Dynamic pressure（阿尔芬速度 / 动压）**：由上述基本变量导出的物理量。阿尔芬速度是磁流体中横波传播的特征速度（磁压与动压之比的平方根），动压是等离子体撞击物体的冲击压力。它们是评估空间天气灾害性效应的关键指标。

### 三、天文学家关心的问题

天文学家在这个子领域追问的核心是：**给定今天的太阳表面磁场观测，太阳风将以何种速度、密度和磁场强度吹向地球？** 这不仅是纯物理问题（理解日冕加热和太阳风加速机制，至今仍是未解之谜），更是极具现实紧迫性的应用问题——高速太阳风和日冕物质抛射（CME）会引发地磁暴，直接威胁卫星、宇航员和地面电网安全。

当前领域的主流分析方法是基于第一性原理的MHD数值模拟。天文学家将光球磁图作为底边界条件，求解包含热传导、辐射损失和经验加热项的MHD方程组，从太阳表面一直外推到0.1 au乃至地球轨道。这种方法的已知局限极其明显：**计算代价极其高昂**。为了得到一个稳态解，通常需要在数百个CPU核心上运行数小时（如本文的COCONUT模型至少需要1小时收敛）。这种速度完全无法满足空间天气“实时或近实时预报”的需求。此外，MHD模型中关于日冕加热的物理机制依然依赖人为的经验参数调节，模型本身在物理上就不完备。
