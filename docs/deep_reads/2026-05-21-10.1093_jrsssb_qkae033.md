# Covariate-adaptive randomization inference in matched designs

**作者**: Samuel D Pimentel, Yaxuan Huang  
**来源**: Journal of the Royal Statistical Society Series B  
**主题**: 因果推断  
**相关性**: 9/10  
**链接**: <https://doi.org/10.1093/jrsssb/qkae033>

---

### 一、核心问题与贡献
①研究了匹配观察性研究中传统Fisher随机化检验假设匹配集内处理分配为均匀排列，从而忽略了由倾向得分差异刻画的分配不均衡性问题。②核心方法是提出协变量自适应随机化推断，通过倾向得分差异调整匹配集内的排列概率，无需剔除匹配对或建模结果变量。③主要贡献是证明了该检验在常数可加效应下的大样本性质，且当倾向得分估计基于大样本时I类错误可任意接近名义水平，并将Rosenbaum敏感性分析推广至该框架。

### 二、基础设定
- **核心概念与符号**：
  - $I$：匹配集数量，$n_i$：第 $i$ 个匹配集内的个体数。
  - $Z_{ij} \in \{0,1\}$：第 $i$ 集第 $j$ 个体的处理指示变量。
  - $e_{ij}$：真实倾向得分，$\hat{e}_{ij}$：估计的倾向得分。
  - $\Delta_{ij} = e_{ij} - \bar{e}_i$：集内倾向得分差异，$\hat{\Delta}_{ij}$为其估计。
  - $\pi_{ij} \propto \exp(\alpha \hat{\Delta}_{ij})$：协变量自适应排列概率（权重）。
  - $T = \sum_{i,j} Z_{ij} (r_{ij} - \bar{r}_i)$：均值差检验统计量（基于偏离集内均值的残差）。
- **关键假设**：
  - **常数可加效应（Sharp Null with constant $\tau$）**：$H_0: Y_{ij}(1) - Y_{ij}(0) = \tau$。含义：允许在Fisher框架下计算反事实结果并固定残差；局限：排除了处理效应异质性，这是Fisher随机化检验的固有要求。
  - **大样本倾向得分估计**：倾向得分基于独立的大样本估计且相合。含义：保证 $\hat{\Delta}_{ij} \approx \Delta_{ij}$，使得自适应排列分布逼近真实条件分布；与仅要求匹配而不要求倾向得分模型正确的传统方法相比，引入了对外部样本或正确模型的依赖。
  - **有界倾向得分与有限方差**：倾向得分远离0和1，结果变量二阶矩有限。含义：保证排列权重的非退化性与统计量矩的存在性。
- **问题背景**：
  - 传统匹配后FRT视匹配集内分配为均匀排列，当匹配不完美（存在残差协变量不平衡）时，均匀排列假设失效，导致I类错误膨胀。
  - 与 Rosenbaum (2002) 的区别：Rosenbaum的敏感性分析是在均匀排列基础上通过参数 $\Gamma$ 放大几率比进行最坏情况分析，本文直接利用估计的倾向得分修改排列分布，将协变量不平衡内化到零假设分布中。
  - 与 Abadie & Imbens 的区别：后者侧重于匹配估计量的点估计与方差估计（Neyman框架），本文侧重于Fisher随机化检验的分布重构。

### 三、主要定理 / 核心结果
1. **原文陈述**：在常数可加效应 $H_0$ 下，基于协变量自适应排列分布的均值差统计量 $T$，经标准化后渐近服从标准正态分布。当倾向得分基于大样本估计时，检验的I类错误率收敛于名义水平 $\alpha$。
2. **直观解释**：当排列概率按倾向得分差异加权时，处理组更可能包含倾向得分高的个体。这改变了检验统计量的零分布方差。只要倾向得分估计准确，这种加权排列分布就能精确逼近给定协变量下的真实条件分配分布，从而校准检验。
3. **解决了什么技术难点**：解决了非均匀排列下统计量渐近分布的推导问题，以及倾向得分估计误差对排列分布分位数造成的摄动影响。
4. **适用条件与局限**：必须满足常数可加效应假设；若倾向得分与主样本同时估计，估计误差将与处理分配耦合，可能导致I类错误膨胀（必须依赖外部大样本或样本拆分）。

### 四、证明框架 / 方法设计
- **证明主干逻辑**：矩量法 + 条件概率极限定理 + 渐近展开。
- **拆解为关键逻辑步骤**：
  1. **排列分布的矩计算**：在非均匀排列概率 $\pi_{ij}$ 下，推导均值差统计量 $T$ 的一阶、二阶矩，证明其依概率收敛到确定性极限。
  2. **条件中心极限定理**：利用组合中心极限定理，证明在给定协变量条件下，标准化后的 $T$ 在自适应排列分布下渐近正态。
  3. **摄动分析**：将真实倾向得分 $e$ 与估计值 $\hat{e}$ 的差异视为摄动，证明当估计样本量 $N \to \infty$ 时，基于 $\hat{e}$ 的排列分布与基于 $e$ 的排列分布的Wasserstein距离趋于0。
  4. **I类错误收敛**：结合分位数的Bahadur表示，证明基于 $\hat{e}$ 的检验分位数收敛于基于 $e$ 的分位数，从而控制I类错误。
- **最关键的技巧性引理或"跳跃点"**：将倾向得分估计误差对离散排列分布的影响，转化为对连续渐近分布分位数的摄动分析。通过Bahadur展开，将分布的微小偏移映射为分位数的同阶偏移，从而桥接了"估计误差随 $N \to \infty$ 消失"与"I类错误渐近精确"之间的逻辑缺口。
- **数学工具评价**：经典工具的巧妙组合。将Rosenbaum的几率比参数化与条件排列检验结合，用大样本理论弥补了非均匀排列下精确检验的不可解性。

### 五、与研究者兴趣的关联
- **连接到哪个子方向**：匹配设计中的 Fisher randomization test 与 sensitivity analysis 交叉方向。
- **可借鉴的核心思路或技术工具**：将倾向得分差异内化为排列权重（类似倾向得分加权与随机化推断的融合），这为半参数框架下的条件推断提供了新视角；敏感性分析的推广方式（将 $\Gamma$ 参数与倾向得分差异权重结合）可直接迁移到其他依赖排列检验的因果推断场景（如断点回归或工具变量排列检验中的协变量调整）。
- **值得精读的关键参考文献**：
  1. Rosenbaum, P. R. (2002). *Observational Studies*.（敏感性分析与排列检验的经典基石，理解本文如何修改几率比参数化的必读）
  2. Ding, P. (2017). A Paradox from Randomization-Based Causal Inference.（理解Fisher随机化检验在异质性效应下的局限与常数效应假设的必要性，对理解本文假设的不可放宽性至关重要）

### 六、延伸思考与练习
- **假设扰动**：若放宽"常数可加效应"假设至"任意Sharp Null $H_0: Y(1)=Y(0)$"，由于无法观测到所有潜在结果，排列分布无法直接构造。此时需要什么新工具？（提示：需转向Neymanian重复抽样框架，或构造异质性稳健的检验统计量，此时自适应排列权重的方差估计将面临挑战）。
- **开放问题**：如何在主样本上直接估计倾向得分而不引入I类错误膨胀？Cross-fitting在匹配设计中会破坏匹配结构，如何设计一种保持匹配结构同时消除估计误差耦合的样本拆分策略？
- **理解检测题**：假设在一个匹配对设计中，某一对的倾向得分差异极大（例如 $\hat{e}_{i1} = 0.9, \hat{e}_{i2} = 0.1$）。在协变量自适应随机化推断下，这对个体对均值差统计量 $T$ 的方差贡献，相比于均匀排列下，是变大了还是变小了？请从排列权重的非退化性角度给出直观解释。
