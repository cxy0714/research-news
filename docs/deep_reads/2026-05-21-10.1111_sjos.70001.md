# Variable importance measures for heterogeneous treatment effects with survival outcome

**作者**: Simon Christoffer Ziersen, Torben Martinussen  
**来源**: Scandinavian Journal of Statistics  
**主题**: 因果推断  
**相关性**: 8/10  
**链接**: <https://doi.org/10.1111/sjos.70001>

---

### 一、核心问题与贡献
①研究了右删失生存结局下，如何度量协变量对条件平均处理效应（CATE）异质性的边际贡献。②核心工具是基于半参数效率理论的 efficient influence function 与 one-step 估计，并引入交叉拟合消除经验过程项。③主要贡献是将处理效应变量重要性度量（TE-VIM）扩展至基于生存函数和 RMST 的 CATE，新提出基于 CATE 最佳偏线性投影的异质性度量，并证明了所提估计量在干扰参数收敛速率不足 $n^{-1/4}$ 时仍可保持渐近线性。

### 二、基础设定
- **核心概念与符号**：
  - $T$: 生存时间，$C$: 删失时间，$\tilde{T}=T\wedge C$, $\Delta=I(T\le C)$。
  - $A$: 二值处理，$X$: 协变量，$X_{-j}$: 剔除第 $j$ 个协变量后的协变量集。
  - CATE 定义：基于生存函数 $\tau_S(t\mid x) = P(T^1>t\mid X=x) - P(T^0>t\mid X=x)$；基于 RMST $\tau_R(x) = \int_0^t P(T^1>u\mid X=x)du - \int_0^t P(T^0>u\mid X=x)du$。
  - TE-VIM：$\sigma_j^2 = \text{Var}(\tau(X)) - \text{Var}(E[\tau(X)\mid X_{-j}])$，衡量 $X_j$ 对 CATE 方差的边际贡献。
  - 最佳偏线性投影：$(\beta_0, \beta_1) = \arg\min_{b_0, b_1} E[(\tau(X) - b_0 - b_1 X_j)^2]$，将非参数 CATE 投影至可解释的线性尺度。
- **关键假设**：
  - Ignorability ($A \perp\!\!\!\perp T^a \mid X$)：无未测量混杂，因果推断标准假设。
  - Positivity ($0 < P(A=1\mid X) < 1$)：处理分配概率有界。
  - Independent censoring ($C \perp\!\!\!\perp T \mid A, X$)：条件独立删失，生存分析标准假设，限制了在非随机删失下的应用。
  - Nuisance estimator rate conditions：干扰参数（如倾向得分、条件生存函数）需满足特定收敛速率（如 $o_P(n^{-1/4})$），这是 DML/one-step 估计保证 $\sqrt{n}$-收敛的典型要求。
- **问题背景**：现有 TE-VIM 仅适用于连续/二值结局，缺乏对右删失生存数据的理论支持；单纯估计 CATE 无法量化协变量的异质性驱动因素。与 Hines et al. (2022) 相比，本文处理了删失带来的额外干扰参数（生存函数/删失分布）；与常规生存 CATE 估计相比，本文聚焦于异质性归因而非预测。

### 三、主要定理 / 核心结果
1. **定理：TE-VIM 与偏线性投影的 EIF 及渐近线性**
   - **原文陈述**：目标参数（$\sigma_j^2$ 与 $\beta$）的 one-step 估计量 $\hat{\Psi} = \Psi_n + \mathbb{P}_n \hat{\phi}$ 在交叉拟合下，若干扰参数估计满足收敛速率条件且属于 Glivenko-Cantelli 类，则 $\sqrt{n}(\hat{\Psi} - \Psi) = \frac{1}{\sqrt{n}}\sum_{i=1}^n \phi(O_i) + o_P(1)$，其中 $\phi$ 为 EIF。
   - **直观解释**：EIF 揭示了估计 $\sigma_j^2$ 时的信息边界，残差项 $(\tau(X) - E[\tau(X)\mid X_{-j}])$ 与协变量残差 $(X_j - E[X_j\mid X_{-j}])$ 的交互作用是驱动方差贡献的核心。偏线性投影则将不可直接推断的非参数量映射到具有 $\sqrt{n}$-收敛率的低维参数上。
   - **解决的技术难点**：在生存设定下，$\tau(X)$ 包含不可观测的潜变量生存函数，通过 IPCW 或嵌套逆概率映射将其转化为可观测变量的泛函，并推导出闭合形式的 EIF。
   - **适用条件与局限**：条件独立删失假设是识别的基石；若 $X_j$ 与 $X_{-j}$ 强相关，$\sigma_j^2$ 的边际贡献可能极小，导致估计量不稳定（方差膨胀）。

### 四、证明框架 / 方法设计
- **证明主干逻辑**：构造法 + Von Mises 展开 + 经验过程理论。
- **拆解为 3-5 个关键逻辑步骤**：
  1. **目标参数的切线性与 EIF 推导**：利用 Gateaux 导数计算 $\sigma_j^2$ 和偏线性投影参数在扰动路径上的导数，结合生存设定的 IPCW 结构，导出 EIF $\phi(O)$。
  2. **One-step 构造与 Von Mises 展开**：构造 $\hat{\Psi} = \Psi_n + \mathbb{P}_n \hat{\phi}$，对 $\hat{\Psi} - \Psi$ 进行展开，分离出线性主项 $(\mathbb{P}_n - P)\phi$ 和二阶余项 $R(\hat{\eta}, \eta)$。
  3. **交叉拟合控制经验过程项**：利用样本划分，证明 $(\mathbb{P}_n - P)(\hat{\phi} - \phi) = o_P(n^{-1/2})$，绕过对 Donsker 类的强依赖。
  4. **余项收敛分析**：利用 Cauchy-Schwarz 不等式和干扰参数的收敛速率条件，证明二阶余项 $R(\hat{\eta}, \eta) = o_P(n^{-1/2})$。
- **最关键的技巧性引理或"跳跃点"**：将生存函数和 RMST 的 CATE 表达为可观测数据的泛函。在独立删失下，利用 Kaplan-Meier 权重或条件生存函数的逆概率加权，将不可观测的 $E[\tau(X)\mid X_{-j}]$ 转化为包含 $E[\frac{\Delta}{G(T)} I(T>t) \mid X]$ 等可估计量的形式，这是推导生存设定下 EIF 的关键跳跃点。
- **数学工具评价**：是经典的半参数理论（Bickel et al., 1993; van der Vaart, 2000）与 DML/One-step 框架的标准化应用，创新点在于目标参数的构造与生存设定下 EIF 的具体代数推导。

### 五、与研究者兴趣的关联
- **连接到哪个子方向**：生存分析下的半参数因果推断 / CATE 异质性度量 / 半参数效率理论。
- **可借鉴的核心思路或技术工具**："Best partially linear projection" 的构造思路。在半参数模型中，当复杂非参数量（如 CATE）难以直接进行稳健推断时，将其向一个可识别的、具有良好性质的低维参数空间（偏线性投影）投影，从而获得 $\sqrt{n}$-收敛的估计量。这一思路可迁移到其他难以直接估计的泛函参数（如 longitudinal 设定下的动态处理效应异质性度量）。
- **值得精读的关键参考文献**：
  1. Hines, Diaz-Ordaz, Vansteelandt (2022a): "Variable importance measures for heterogeneous treatment effects"，本文的理论基石，必须精读以理解 TE-VIM 的原始定义与无删失设定下的 EIF 推导。
  2. Rytgaard et al. (2022/2023) 或类似关于生存分析 DML/EIF 的文章：展示了在右删失数据下如何处理 IPCW 与 EIF 的结合，对处理带删失的半参数估计有直接参考价值。

### 六、延伸思考与练习
- **假设扰动**：若将"条件独立删失"（$C \perp\!\!\!\perp T \mid A, X$）放宽为"依赖于未观测协变量的删失"（如 $C \perp\!\!\!\perp T \mid A, X, U$），结论如何变化？技术上需要引入新的识别假设（如 instrumental censoring 或 negative control），且 EIF 的推导将包含不可识别的项，可能需要转向局部敏感性分析或 bounds 推断。
- **开放问题**：当协变量维度 $p$ 随样本量 $n$ 增长时，如何构造 TE-VIM 的有效估计量？高维下的条件方差 $\text{Var}(E[\tau(X)\mid X_{-j}])$ 估计极具挑战，可能需要借鉴高维稀疏回归的 debiasing 技术。
- **理解检测题**：假设 $X_j$ 与 $X_{-j}$ 完全独立，请写出此时基于 RMST 的 TE-VIM ($\sigma_j^2$) 的 EIF 简化形式，并说明在此极端假设下，干扰参数的估计速率要求是否可以放宽？为什么？
