# Gaussian universal likelihood ratio testing

**作者**: Robin Dunn, Aaditya Ramdas, Sivaraman Balakrishnan, Larry Wasserman  
**来源**: Biometrika  
**主题**: 数理统计 / 假设检验  
**相关性**: 9/10  
**链接**: <https://doi.org/10.1093/biomet/asac064>

---

### 一、核心问题与贡献
①研究了基于样本拆分的 universal likelihood ratio test (ULRT) 在无需正则条件下的有限样本性质与渐近效率损失。②通过在经典 LRT 适用的 $d$ 维高斯均值检验设定下，对比多种 ULRT 变体（split LRT, repeated subsampling）与经典 LRT 的表现。③证明重复子抽样是最优 ULRT 变体，并精确量化了高维下 ULRT 置信集期望平方半径约为经典 LRT 的 3/2 倍，同时在非凸零假设检验中展现了超越传统方法的功效。

### 二、基础设定
- **核心概念与符号**：
  - $X_1, \dots, X_n \sim N_d(\mu, I_d)$：基础数据生成过程。
  - Split LRT：将样本分为 $D_1$ (大小 $n_1$) 和 $D_2$ (大小 $n_2$)，统计量 $\Delta_n = \prod_{i \in D_2} \frac{p_{\hat{\mu}_1}(X_i)}{p_{\hat{\mu}_0}(X_i)}$，其中 $\hat{\mu}_1$ 为 $D_1$ 上的 MLE，$\hat{\mu}_0$ 为零假设下的 MLE。
  - Repeated subsampling：进行 $B$ 次样本拆分，取统计量的最大值（或最小 p 值）以消除单次拆分的随机性。
  - 置信集反转：$C_B = \{\theta_0 : \text{未拒绝 } \theta_0\}$。
- **关键假设**：
  - **高斯等方差假设**（$N_d(\mu, I_d)$）：为对比 ULRT 与经典 LRT 提供了基准测试平台，此假设下经典 LRT 是最优的，从而能量化 ULRT 的效率折损。
  - **样本拆分独立性**：$D_1$ 与 $D_2$ 独立，这是 ULRT 保证有限样本 Type-I 错误控制的核心（无需正则条件）。相比传统 LRT 依赖的 Wilks 定理（需正则条件），该假设放宽了对参数空间和似然函数的光滑性要求。
- **问题背景**：
  - Universal inference (Wasserman et al., 2020) 提出了无需正则条件的检验框架，但其功效损失未知。本文首次在经典 LRT 适用的设定下，精确量化了这种"无代价的代价"（price of universality）。
  - 与最相关文献的区别：Wasserman et al. (2020) 仅给出了 Markov 不等式下的保守水平控制，本文则通过精确分布计算给出了 3/2 的效率界；区别于一般的高维功效分析，本文聚焦于置信集体积的期望平方半径比较。

### 三、主要定理 / 核心结果
1. **原文陈述**：在 $d$ 维高斯均值检验中，当 $B \to \infty$ 时，repeated subsampling ULRT 的置信集近似球形；在高维设定下（$d \to \infty, n \to \infty$），最优 ULRT 置信集的期望平方半径约为经典 LRT 球形置信集期望平方半径的 3/2 倍。
2. **直观解释**：3/2 倍率源于样本拆分带来的信息损失。经典 LRT 使用全部 $n$ 个样本既估计参数又计算似然，而 split LRT 用 $n_1$ 估计参数、$n_2$ 计算似然。这导致似然比统计量的方差中额外增加了由 $\hat{\mu}_1$ 估计误差引入的变异性，使得反转出的置信集半径扩大。
3. **解决了什么技术难点**：克服了 ULRT 统计量在有限样本下精确分布难以刻画的问题，通过巧妙的高斯分布分解，将 split LRT 统计量转化为非中心卡方分布的函数，从而精确计算出置信集半径的期望。
4. **适用条件与局限**：3/2 的精确倍率依赖于高斯等方差假设。若模型非高斯或协方差未知，该常数可能改变。此外，$B \to \infty$ 的计算成本在实际中可能受限。

### 四、证明框架 / 方法设计
- **证明主干逻辑**：直接分布特征化 + 高维渐近展开。
- **拆解为 3-5 个关键逻辑步骤**：
  1. **统计量分解**：利用 $D_1$ 和 $D_2$ 的独立性，将 split LRT 的对数似然比统计量分解为独立高斯向量的二次型。
  2. **分布推导**：证明对数似然比统计量在零假设和对立假设下服从特定参数的非中心卡方分布，从而得到精确的检验临界值和 p 值。
  3. **置信集半径表征**：通过反转检验，将置信集的边界表示为样本均值和样本方差的函数，推导出半径平方的解析表达式。
  4. **高维渐近分析**：在 $d/n \to \gamma$ 的设定下，对半径平方的期望进行泰勒展开，提取主导项，得出经典 LRT 与 ULRT 半径平方期望之比为 2/3（即 ULRT 是经典 LRT 的 3/2 倍）。
- **最关键的技巧性引理或"跳跃点"**：将 split LRT 的对数似然比 $\sum_{i \in D_2} (\|X_i - \hat{\mu}_0\|^2 - \|X_i - \hat{\mu}_1\|^2)$ 重构为 $\|\bar{X}_2 - \hat{\mu}_1\|^2$ 的缩放形式，并利用 $\bar{X}_2$ 与 $\hat{\mu}_1$ (即 $\bar{X}_1$) 的独立性，将其分解为独立的卡方变量。这一步将复杂的乘积似然比转化为可加性的二次型，是精确计算期望半径的关键。
- **数学工具评价**：经典高斯分布理论与二次型分布理论的巧妙组合。未引入全新的分析框架，但在 universal inference 的语境下，首次用最基础的概率论工具精确量化了效率损失，极具启发性。

### 五、与研究者兴趣的关联
- **连接到哪个子方向**：高维假设检验 / 无正则条件的半参数推断。
- **可借鉴的核心思路或技术工具**：
  - **"代价量化"范式**：在开发无需正则条件的新方法（如 proximal CI 或 DML）时，可借鉴本文思路，在经典方法最优的参数模型下精确计算新方法的效率损失常数（如 3/2），为方法使用提供明确的 trade-off 基准。
  - **Repeated subsampling 消除随机性**：在依赖样本拆分的方法（如 DML 中的 cross-fitting）中，通过多次拆分取聚合（如取统计量最大值或 p 值最小值）来收紧置信集，同时保持有限样本有效性。
- **值得精读的关键参考文献**：
  1. Wasserman, L., Ramdas, A., & Balakrishnan, S. (2020). Universal inference. *PNAS*. (Universal inference 的奠基之作，理解 ULRT 的前提)
  2. Spokoiny, V. (2017). Penalized maximum likelihood estimation and effective dimension. *Ann. Inst. H. Poincaré*. (高维似然比检验的另一种无正则条件处理方式，与 ULRT 的 3/2 代价形成对比)

### 六、延伸思考与练习
- **假设扰动**：若将假设从 $N_d(\mu, I_d)$ 改为 $N_d(\mu, \Sigma)$ 且 $\Sigma$ 未知，3/2 的常数会如何变化？估计 $\Sigma$ 引入的额外变异性是否会使效率损失倍率进一步放大？技术上需要处理 Wishart 分布与卡方分布的交互。
- **开放问题**：能否通过更复杂的样本拆分策略（如 DML 中的交叉拟合，每个样本既用于估计又用于评估），在保持有限样本有效性的同时，将 3/2 的效率损失压缩至 $1+\epsilon$ 甚至 1？
- **理解检测题**：在一维高斯均值检验 $N(\mu, 1)$ 中，取 $n_1=n_2=n/2$。请推导 split LRT 对数似然比统计量在 $\mu=\mu_0$ 下的精确分布，并计算其方差。将该方差与基于全部 $n$ 个样本的经典 LRT 对数似然比统计量的方差对比，解释 3/2 倍率的来源。
