# Robust Mean Change Point Testing in High-Dimensional Data With Heavy Tails

**作者**: Mengchu Li, Yudong Chen, Tengyao Wang, Yi Yu  
**来源**: IEEE Transactions on Information Theory  
**主题**: 数理统计 / 假设检验  
**相关性**: 8/10  
**链接**: <https://doi.org/10.1109/tit.2025.3634207>

---

### 一、核心问题与贡献
①研究了高维数据在误差分布具有指数衰减或多项式衰减重尾时的均值变点检测问题。②根据均值变化向量的 $\ell_0$-范数区分 dense 与 sparse regime，分别使用 CUSUM-type 与 median-of-means-type 统计量进行检验。③首次在变点文献中刻画了两种重尾条件下的 dense/sparse 相变边界，并在 $\alpha \ge 4$ 时达到近最优 minimax 检验速率；揭示了 $2 \le \alpha < 4$ 时不存在 sparse regime 的新现象（信息论上 sparse 与 dense 同等困难）。

### 二、基础设定
- **核心概念与符号**：
  - $p$：数据维度；$n$：时间序列长度。
  - $\theta^{(1)}, \theta^{(2)} \in \mathbb{R}^p$：变点前后的均值向量；$\Delta = \theta^{(2)} - \theta^{(1)}$，均值变化向量。
  - $s = \|\Delta\|_0$：变化向量的稀疏度。
  - $\alpha$：多项式衰减重尾分布存在的矩阶数（$\mathbb{E}|X_{ij}|^\alpha < \infty$）。
  - Dense regime 与 Sparse regime：由 $s$ 与 $p$ 的相对关系界定的相变边界，决定 minimax 检验速率的量级。
- **关键假设**：
  - **指数衰减尾**（Sub-exponential / Sub-Gaussian）：误差分布的矩生成函数存在或有界，对应经典轻尾设定。
  - **多项式衰减尾**（$\alpha \ge 4$ 或 $2 \le \alpha < 4$）：误差仅存在有限阶矩，这是本文处理重尾的核心设定。相比已有变点文献（多假设亚高斯噪声），大幅放宽了分布假设。
  - **最小间距条件**（Minimal spacing condition）：在备择假设下，要求变点间或变点与端点有足够距离。在此条件下，重尾设定亦可达到 sub-Gaussian 速率。
- **问题背景**：已有高维变点检测文献（如 Wang & Yu 等人的前期工作）主要聚焦于亚高斯误差，对重尾数据缺乏 minimax 检验速率的刻画。本文填补了重尾下 dense/sparse 相变边界的空白。最相关的前期工作包括：高维变点检测的 CUSUM 方法（Wang, Yu 等），以及高维重尾均值估计的 median-of-means (MoM) 文献（Lugosi & Mendelson 等）。

### 三、主要定理 / 核心结果

1. **指数衰减尾下的 Minimax 检验速率与相变边界**
   - **原文陈述**：在亚指数误差下，minimax testing rate 在 dense regime 为 $\sqrt{p/n}$ 量级，在 sparse regime 为 $\sqrt{s \log p / n}$ 量级，相变边界为 $s \asymp \log p$。CUSUM-type 统计量达到近最优速率。
   - **直观解释**：轻尾下，噪声集中性好。当变点涉及足够多坐标（$s > \log p$）时，信号在所有方向累积，dense 速率仅依赖维度 $p$；当变点极稀疏时，需在 $p$ 维中寻找 $s$ 个坐标，付出 $\log p$ 的搜索代价。
   - **解决的技术难点**：在高维 CUSUM 中控制亚指数随机变量的极大值（通过 union bound 与 Bernstein 不等式）。
   - **适用条件与局限**：依赖指数矩存在，无法推广至仅存在低阶矩的分布。

2. **多项式衰减尾（$\alpha \ge 4$）下的 MoM 检验与相变**
   - **原文陈述**：当存在 $\alpha \ge 4$ 阶矩时，MoM-type 统计量在 dense 与 sparse regime 均达到近最优速率，相变边界为 $s \asymp p^{2/\alpha}$。
   - **直观解释**：重尾下噪声极大值破坏经典 CUSUM。MoM 通过将数据分块取中位数，截断了极端离群值的影响。相变边界 $p^{2/\alpha}$ 反映了重尾噪声在 $p$ 维投影下的有效维度：矩阶越低，噪声越重，"看似 sparse 的变点"也会被噪声淹没，导致 sparse regime 的阈值向 $p$ 靠拢。
   - **解决的技术难点**：在高维空间中同时实现 MoM 的鲁棒
