# Exact Bias Correction for Linear Adjustment of Randomized Controlled Trials

**作者**: Haoge Chang, Joel A. Middleton, P. M. Aronow  
**来源**: Econometrica  
**主题**: 因果推断  
**相关性**: 8/10  
**链接**: <https://doi.org/10.3982/ecta20289>

---

### 一、核心问题与贡献
①研究了在随机化模型下，Freedman (2008) 指出的 RCT 线性回归调整估计量存在有限样本偏差的问题。②核心方法是基于势结果框架的严格代数推导，提取 OLS 估计量偏差的精确闭式表达式并进行直接校正。③主要结论是推导出了精确的偏差校正项，且证明校正后估计量的极限分布与未校正估计量完全一致，结合 Lin (2013) 的交互项调整，彻底解决了 Freedman 对回归调整的理论质疑。

### 二、基础设定
- **核心概念与符号**：
  - $N$: 总样本量；$N_1, N_0$: 处理组与对照组样本量；$p = N_1/N$: 处理组分配比例。
  - $D_i \in \{0,1\}$: 处理指示变量；$Y_i(1), Y_i(0)$: 潜在结果；$Y_i = D_i Y_i(1) + (1-D_i) Y_i(0)$: 观测结果。
  - $X_i$: 协变量；$\tau = N^{-1}\sum_{i=1}^N [Y_i(1) - Y_i(0)]$: 有限样本平均处理效应 (SATE)。
  - $\hat{\tau}_{OLS}$: 带协变量的 OLS 回归调整估计量；$\hat{\tau}_{BC}$: 偏差校正后估计量。
- **关键假设**：
  - **随机化模型**：$D_i$ 的分配机制已知且独立于潜在结果和协变量。统计学含义：随机性仅来源于处理分配，潜在结果 $\{Y_i(1), Y_i(0), X_i\}_{i=1}^N$ 视为固定常数，属于 Neyman 的有限样本推断框架。
  - **渐近框架设定**：$N_1/N \to p \in (0,1)$，且协变量样本协方差矩阵非奇异。与经典计量经济学假设总体线性模型不同，此假设不限制总体回归函数的函数形式，是对总体分布假设的极大放宽。
- **问题背景**：Freedman (2008) 证明在随机化模型下，OLS 回归调整不仅有限样本有偏，且其传统标准误也不一致，甚至可能比简单差分估计量渐近效率更低。Lin (2013) 通过引入协变量与处理变量的交互项解决了渐近效率与标准误的问题，但未解决 OLS 的有限样本偏差。本文与 Freedman (2008) 和 Lin (2013) 构成理论闭环：在 Freedman 原设定下给出精确偏差校正，结合 Lin 的交互项设定，彻底消除回归调整的理论缺陷。

### 三、主要定理 / 核心结果
1. **原文陈述（定理1：OLS 的精确偏差）**：在完全随机化下，$E[\hat{\tau}_{OLS} - \tau] = -\frac{1}{N-2} \left( \frac{N_0-1}{N_1} S_{Y(1)X} - \frac{N_1-1}{N_0} S_{Y(0)X} \right) S_{XX}^{-1} \bar{X}$，其中 $S_{AB}$ 为样本协方差/交叉协方差矩阵，$\bar{X}$ 为样本均值。
2. **直观解释**：偏差来源于 OLS 试图在非平衡的协变量上投影时，由于处理组和对照组的样本协方差结构及残差尺度不同，导致投影系数在有限样本下产生系统性偏移。偏差不仅取决于协变量与潜在结果的相关性，还受限于样本分配比例 $N_1/N_0$ 的非对称性。
3. **解决了什么技术难点**：克服了 OLS 估计量中分母包含随机变量（残差平方和）导致无法直接求期望的难题，通过代数恒等式将期望转化为可计算的固定势结果特征的组合。
4. **适用条件与局限**：适用于完全随机化实验。局限在于：若存在异方差或更高阶的非线性势结果映射，偏差的高阶余项虽在渐近意义上可忽略，但在极小样本下可能仍具数值影响；且该结果严格依赖于固定势结果框架。

### 四、证明框架 / 方法设计
- **证明主干逻辑**：代数恒等式展开 + 矩方法提取精确偏差 + Slutsky 定理证明渐近等价。
- **拆解为关键逻辑步骤**：
  1. **代数展开**：将 OLS 估计量 $\hat{\tau}_{OLS}$ 的代数表达式展开，分离出处理组与对照组的样本回归系数，将其表示为样本矩的函数。
  2. **条件期望提取**：利用 $D_i$ 的伯努利随机性质，对样本矩关于分配变量求条件期望，通过泰勒展开或精确代数分解，处理分母中随机变量带来的非线性偏移，得到偏差的精确闭式解。
  3. **构造校正量**：用样本矩替换偏差闭式解中的总体矩（如 $S_{Y(1)X}$ 等），构造 $\hat{\tau}_{BC} = \hat{\tau}_{OLS} - \widehat{Bias}$。
  4. **渐近分析**：证明 $\widehat{Bias} - Bias = o_p(N^{-1/2})$，从而 $\sqrt{N}(\hat{\tau}_{BC} - \tau) = \sqrt{N}(\hat{\tau}_{OLS} - \tau) + o_p(1)$，得出两者极限分布相同的结论。
- **最关键的技巧性引理或"跳跃点"**：处理 OLS 估计量中分母的随机性（即 $E[A/B] \neq E[A]/E[B]$ 的问题）。作者通过构造精确的代数恒等式，将随机分母的逆转化为常数项加上一个均值为零的随机扰动项，从而在不依赖高阶渐近展开的情况下，提取出有限样本下的精确一阶偏差。
- **数学工具评价**：经典统计代数技巧的极致运用，非全新分析框架，但在 RCT 随机化推断的代数推导上做到了严密与精确，避免了复杂的经验过程理论。

### 五、与研究者兴趣的关联
- **连接到哪个子方向**：RCT 中的回归调整与有限样本因果推断。
- **可借鉴的核心思路或技术工具**：在半参数或高维 debiased 估计中，常遇到估计量存在 $O(N^{-1})$ 偏差的问题（如 debiased Lasso 中的二阶偏差项）。本文展示了一种**不依赖高阶渐近展开，而是直接通过代数恒等式提取精确有限样本偏差**的思路。这种"精确代数提取 + plug-in 估计 + 证明 plug-in 误差为 $o_p(N^{-1/2})$"的范式，可迁移到其他具有复杂非线性结构的估计量偏差校正中，特别是在处理逆概率加权（IPW）或双重机器学习（DML）中分母随机性导致的偏差时。
- **值得精读的关键参考文献**：
  1. Freedman, D. A. (2008). On regression adjustments to experimental data. *Advances in Applied Mathematics*. (问题起源，理解偏差产生的根本机制)
  2. Lin, W. (2013). Agnostic notes on regression adjustments to experimental data: Reexamining Freedman's critique. *Annals of Applied Statistics*. (渐近最优解，与本文结合构成完整理论图景)

### 六、延伸思考与练习
- **假设扰动**：若修改"完全随机化"假设为"分层随机化"或"重随机化"，偏差的闭式解将如何变化？代数展开将面临何种新的跨层相关性或条件期望计算的挑战？
- **开放问题**：对于非线性的广义线性模型（GLM）调整（如 Logit/Probit 回归调整），是否存在类似的精确或近似有限样本偏差校正公式？还是必须依赖高阶渐近展开（如 bootstrap 或解析的 higher-order expansion）？
- **理解检测题**：假设在完全随机化实验中，处理组比例 $p=1/2$，且协变量 $X$ 对 $Y(1)$ 和 $Y(0)$ 的线性投影系数恰好相等（即 $S_{Y(1)X}S_{XX}^{-1} = S_{Y(0)X}S_{XX}^{-1}$）。请基于本文推导的偏差公式逻辑，推断此时 OLS 估计量的有限样本偏差是否为 0，并说明理由。
