# High-dimensional multivariate analysis of variance via geometric median and bootstrapping

**作者**: Guanghui Cheng, Ruitao Lin, Liuhua Peng  
**来源**: Biometrics  
**主题**: 数理统计 / 假设检验  
**相关性**: 8/10  
**链接**: <https://doi.org/10.1093/biomtc/ujae088>

---

### 一、核心问题与贡献
①研究了高维多总体下基于几何中位数的位置参数异质性检验（高维MANOVA）问题。②提出了基于组间几何中位数差异的 maximum-type 检验统计量，并利用 Gaussian approximation 与 wild bootstrap 进行分布逼近。③证明了零假设下统计量的 Gaussian approximation 极限分布及备择假设下的一致性，理论验证了 wild bootstrap 的有效性，提供了高维下抗异常值的鲁棒检验方法。

### 二、基础设定
- **核心概念与符号**：
  - $k$ 个总体，第 $l$ 个总体样本 $X_{l1}, \dots, X_{ln_l} \in \mathbb{R}^p$。
  - 几何中位数 (Geometric Median, GM)：$\mu_l = \arg\min_{\mu \in \mathbb{R}^p} E[||X_l - \mu|| - ||X_l||]$。
  - 零假设 $H_0: \mu_1 = \dots = \mu_k$ vs 备择假设 $H_1: \exists i \neq j, \mu_i \neq \mu_j$。
  - Maximum-type 统计量：$M_n = \max_{1 \le l < m \le k} n_{lm} ||\hat{\mu}_l - \hat{\mu}_m||^2$（或类似由样本量加权的范数形式）。
- **关键假设**：
  - 矩条件（如有限 4 阶或 8 阶矩）：控制高维下统计量的尾部行为，是高维高斯逼近的必要条件，相比传统均值检验的子高斯假设有所放宽。
  - 协方差矩阵的正则化条件（如特征值上下界）：防止高维空间中分布退化。
  - 维度 $p$ 与样本量 $n$ 的发散速率：允许 $p \gg n$，体现高维设定。
- **问题背景**：
  - 传统高维 MANOVA 基于均值向量，对重尾或异常值极度敏感；基于空间符号/中位数的现有方法多针对单样本或两样本问题，缺乏多总体（$k>2$）的异质性检验框架。
  - 与 Chen and Qin (2010)（基于均值的高维两样本检验）的区别：用几何中位数替代均值，牺牲极小量渐近效率换取鲁棒性；与 Chaudhuri (1996)（几何中位数的渐近理论）的区别：将低维单样本理论扩展至高维多总体的 maximum-type 检验。

### 三、主要定理 / 核心结果
1. **定理1：零假设下的 Gaussian Approximation**
   - **原文陈述**：$\sup_{x} |P(M_n \le x) - P(Z \le x)| \to 0$，其中 $Z$ 是某个高斯向量的最大值。
   - **直观解释**：高维下非独立非线性统计量（几何中位数差）的最大值分布，可由高斯向量的最大值分布一致逼近，使得理论分位数可解。
   - **技术难点**：几何中位数经验估计量 $\hat{\mu}$ 的 Bahadur 展开中，非线性余项在高维最大模下的收敛控制。
   - **适用条件与局限**：需满足矩条件与维度发散速率；若分布极度重尾（如不存在二阶矩），余项将无法控制，定理失效。

2. **定理2：备择假设下的一致性**
   - **原文陈述**：在固定或局部备择假设下，若组间差异超过最小信号强度，$P(M_n > c_\alpha) \to 1$。
   - **直观解释**：只要组间位置参数的差异足够大（不被高维噪声淹没），检验的势趋于1。
   - **适用条件与局限**：信号强度需满足 $\min ||\mu_l - \mu_m|| \gg \sqrt{\log k / n}$ 量级。

3. **定理3：Wild Bootstrap 有效性**
   - **原文陈述**：基于乘子的 Bootstrap 统计量 $M_n^*$ 的条件分布与 $M_n$ 的分布的 Kolmogorov 距离依概率趋于0。
   - **直观解释**：通过重抽样直接逼近统计量分布，绕开了高维下协方差矩阵估计及极值分布中未知常数的计算。
   - **技术难点**：在条件期望框架下验证随机权重的经验过程收敛。

### 四、证明框架 / 方法设计
- **证明主干逻辑**：Bahadur 展开 + 高维高斯逼近 + 随机化乘子 Bootstrap。
- **拆解关键逻辑步骤**：
  1. **Bahadur 展开**：将经验几何中位数差 $\hat{\mu}_l - \hat{\mu}_m$ 分解为一阶线性项（独立和的线性组合）与高阶余项之和。
  2. **余项控制**：利用经验过程或矩方法，证明余项在最大模下概率收敛到 $o_p(n^{-1/2})$，从而将非线性统计量线性化。
  3. **高维高斯逼近**：对一阶线性项应用 Chernozhukov et al. (2017) 的高维中心极限定理，将最大值分布逼近为高斯向量的最大值分布。
  4. **Bootstrap 有效性**：构造乘子 统计量，利用条件高斯逼近证明 Bootstrap 分布与原分布一致。
- **最关键的技巧性引理/跳跃点**：几何中位数的 Bahadur 展开及其余项在 $L_\infty$ 范数下的收敛率。因为几何中位数的定义涉及非光滑的 $L_1$ 范数，其梯度在 0 点不可导，需通过光滑化或精细的分部积分控制经验过程的震荡。
- **数学工具评价**：是经典高维高斯逼近工具与鲁棒 M-估计量渐近理论的巧妙组合，非光滑目标函数的展开是技术突破点。

### 五、与研究者兴趣的关联
- **连接子方向**：高维假设检验、高维统计计算。
- **可借鉴思路**：(1) Maximum-type 统计量结合 Gaussian/multiplier bootstrap 的推断框架，可直接迁移到其他高维鲁棒参数（如分位数、M-估计量）的检验；(2) 非光滑目标函数（如几何中位数）的 Bahadur 展开技术，对高维 M-估计量的渐近理论及 Debiasing 有通用性。
- **值得精读的参考文献**：
  1. Chernozhukov, V., Chetverikov, D., Kato, K., & Koike, Y. (2022+). High-dimensional central limit theorems and bootstrap. (高维高斯逼近与 bootstrap 的权威综述，理解本文核心工具必读)
  2. Chaudhuri, P. (1996). On a geometric notion of quantiles for multivariate data. (几何中位数/分位数的渐近性质奠基文献)

### 六、延伸思考与练习
- **假设扰动**：若将几何中位数替换为空间深度 或带惩罚的 M-估计量，Bahadur 展开的余项控制需要什么新的矩条件？收敛率是否会变慢？
- **开放问题**：如何将此框架扩展到半参数模型下的高维鲁棒检验（如高维协方差调整后的处理效应异质性检验）？
- **理解检测题**：假设数据分布存在重尾，使得几何中位数的 Bahadur 展开余项不再满足 $o_p(n^{-1/2})$。请构造一个具体的重尾分布（如多元柯西分布），并分析此时 maximum-type 统计量的极限分布会发生什么变化？Wild bootstrap 是否仍然有效？
