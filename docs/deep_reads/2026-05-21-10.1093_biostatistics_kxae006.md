# Mendelian randomization analysis using multiple biomarkers of an underlying common exposure

**作者**: Jin Jin, Guanghao Qi, Zhi Yu, Nilanjan Chatterjee  
**来源**: Biostatistics  
**主题**: 因果推断  
**相关性**: 8/10  
**链接**: <https://doi.org/10.1093/biostatistics/kxae006>

---

### 一、核心问题与贡献
①研究了当核心暴露不可直接观测（如系统性炎症）但存在多个受其共调节的可观测生物标志物时，如何进行孟德尔随机化（MR）因果推断。②核心方法是基于结构方程模型（SEM），利用可观测性状GWAS汇总统计量的二阶矩构造估计函数（MRLE）。③在允许遗传变异对性状存在直接效应（多效性）的设定下，MRLE严格控制了第一类错误并提升了检验效能，实证中成功检测出系统性炎症对冠心病等疾病的因果效应。

### 二、基础设定
- **核心概念与符号**：
  - $U$：潜在暴露（Latent exposure，如系统性炎症）。
  - $M_j$：第 $j$ 个可观测的生物标志物/性状（$j=1,\dots,p$）。
  - $Y$：疾病结局。
  - $Z_l$：第 $l$ 个遗传变异（IV，$l=1,\dots,L$）。
  - $\hat{\gamma}_{jl}$：$Z_l$ 对 $M_j$ 的GWAS汇总统计量（边际效应估计）。
  - $\hat{\Gamma}_l$：$Z_l$ 对 $Y$ 的GWAS汇总统计量。
  - $\alpha_l$：$Z_l \to U$ 的因果效应（IV强度）。
  - $\lambda_j$：$U \to M_j$ 的因子载荷。
  - $\beta$：$U \to Y$ 的目标因果效应。
  - $\delta_{jl}$：$Z_l \to M_j$ 的直接效应（水平多效性，Pleiotropy）。
- **关键假设**：
  - **A1 (潜变量SEM结构)**：$M_j = \lambda_j U + \sum_l \delta_{jl} Z_l + \epsilon_j$，$Y = \beta U + \epsilon_Y$，$U = \sum_l \alpha_l Z_l + \epsilon_U$。
  - **A2 (IV相关性)**：$\exists l, \alpha_l \neq 0$（遗传变异与潜在暴露相关）。
  - **A3 (IV独立性/排除限制)**：$Z_l \perp \!\!\! \perp (\epsilon_U, \epsilon_j, \epsilon_Y)$，且 $Z_l$ 对 $Y$ 无直接效应（仅通过 $U$）。
  - **A4 (多效性结构假设)**：跨性状的直接效应互不相关，即 $E[\delta_{jl}\delta_{kl}] = 0$ 对 $j \neq k$ 成立。**（与标准MR相比的关键放宽）**：允许 $\delta_{jl} \neq 0$（即允许水平多效性存在），但要求不同性状间的多效性效应正交或独立。
- **问题背景**：标准单性状MR在多效性下估计严重偏倚，且单一标志物无法代表系统性的潜变量暴露。多变量MR（MVMR）将所有标志物放入模型，面临高维共线性和弱工具变量问题。MRLE通过潜变量降维和二阶矩绕过上述瓶颈。与 MVMR 或因子分析MR的区别在于，MRLE仅需汇总统计量且不要求估计每个 $\lambda_j$ 的具体数值。

### 三、主要定理 / 核心结果
1. **识别逻辑（核心理论结果）**：
   - **原文陈述**：在A1-A4下，一阶矩 $E[\hat{\gamma}_{jl}] = \alpha_l \lambda_j + \delta_{jl}$ 被多效性污染，但跨性状的二阶矩 $E[\hat{\gamma}_{jl}\hat{\gamma}_{kl}] = \alpha_l^2 \lambda_j \lambda_k$（$j \neq k$）中多效性项期望为0。结合 $E[\hat{\Gamma}_l] = \beta \alpha_l$，可构造仅依赖二阶矩的估计函数识别 $\text{sign}(\beta)$ 并进行假设检验。
   - **直观解释**：不同生物标志物共享同一个潜变量“信号”，而它们各自的多效性“噪声”被假设为互不相关。因此，对两个标志物的GWAS统计量做叉乘（二阶矩），噪声的期望相互抵消，留下的全是潜变量的共享信号。
   - **解决的技术难点**：在存在不可观测多效性且潜变量也未知的情况下，绕开了一阶矩中 $\delta_{jl}$ 的混淆，实现了因果方向（$\text{sign}(\beta)$）和显著性的非参数识别。
   - **适用条件与局限**：必须假设 $E[\delta_{jl}\delta_{kl}] = 0$。若同一SNP通过同一生物学通路影响多个标志物（即多效性跨性状相关），该识别策略失效。此外，仅能推断 $\text{sign}(\beta)$ 和显著性，无法识别 $\beta$ 的绝对尺度（因 $\alpha_l$ 和 $\lambda_j$ 不可分）。

2. **渐近性质与检验**：
   - 基于汇总统计量的联合渐近正态性，MRLE构造的检验统计量在零假设下收敛于标准正态，第一类错误得到严格控制。

3. **实证结果**：
   - 整合5种炎症标志物（CRP, IL-6, IL-8, TNF-α, MCP-1），MRLE发现炎症增加冠心病（CAD）、结直肠癌和类风湿性关节炎风险（$p < 0.05$，方向为正），而标准单标志物MR因多效性干扰无法得出一致结论。

### 四、证明框架 / 方法设计
- **识别策略与估计量设计**：
  1. 建立潜变量SEM，推导GWAS汇总统计量的一阶矩与二阶矩的解析形式。
  2. 发现一阶矩包含不可观测的 $\delta_{jl}$，无法直接用于IV估计；转向二阶矩，利用 $E[\delta_{jl}\delta_{kl}]=0$ 消除多效性项。
  3. 构造估计函数：利用 $\text{sign}(\hat{\Gamma}_l) \cdot \hat{\gamma}_{jl}\hat{\gamma}_{kl}$ 或类似二阶矩组合，消除潜变量路径系数 $\alpha_l^2 \lambda_j \lambda_k$ 的绝对尺度，提取 $\text{sign}(\beta)$ 的符号信息。
  4. 基于多个SNP和多个性状的二阶矩组合，构造广义估计方程（GEE）或Z统计量进行假设检验。
- **核心假设的可信度分析**：
  - $E[\delta_{jl}\delta_{kl}] = 0$ 是脆弱点。若SNP通过某种未测量的中间表型同时影响 $M_j$ 和 $M_k$，则该假设违背。验证方法：可通过观察残差协方差矩阵是否对角占优来部分评估，但在汇总统计量层面难以完全验证。
- **稳健性检验策略**：
  - 留一法敏感性分析（逐个剔除SNP观察结果稳定性）。
  - Cochran's Q异质性检验（基于二阶矩的变异评估IV异质性）。
- **计算/实现细节**：
  - 仅需各性状和结局的GWAS汇总统计量协方差，无需个体级数据。计算复杂度为 $O(L p^2)$，在 $p$（标志物数）较小时计算极快。

### 五、与研究者兴趣的关联
- **连接子方向**：IV方法在潜变量/测量误差设定下的识别；流行病学应用中的因果推断。
- **可借鉴的核心思路**：
  - **二阶矩识别策略**：在一阶矩被unmeasured confounding（此处为多效性）污染时，利用跨变量二阶矩（协方差）的共享主成分结构实现识别。这一思路极具一般性，可迁移到其他存在测量误差或潜变量的IV/Proximal CI设定中（例如，若Negative Control Outcomes之间存在共享的潜变量结构，可利用其二阶矩消除独立混杂）。
  - **基于汇总统计量的半参数推断**：仅依赖矩条件构造估计函数，避免了参数分布假设，适合大规模数据。
- **值得精读的关键参考文献**：
  1. **Bowden J, et al. (2015) Mendelian randomization with invalid instruments: effect estimation and bias analysis** (MR-Egger提出)：对比理解多效性假设的演进，从一阶矩的均值独立假设到本文二阶矩的叉积独立假设。
  2. **Sanderson E, et al. (2022) Mendelian randomization** (MVMR综述)：理解多变量MR的共线性困境，从而深刻体会本文潜变量降维的必要性。

### 六、延伸思考与练习
- **假设扰动**：若多效性跨性状相关，即 $E[\delta_{jl}\delta_{kl}] = \rho_{jk} \neq 0$，二阶矩识别失效。技术上需要什么新工具？可能需要引入三阶矩，或利用外部注释数据对多效性协方差结构 $\rho_{jk}$ 进行参数化建模/约束。
- **开放问题**：如何将 MRLE 扩展到非线性结局（如 case-control 的 logistic 模型）？此时GWAS汇总统计量（log-OR）的一阶/二阶矩关系因非折叠性变得极其复杂，二阶矩的简单代数分解不再成立。
- **理解检测题**：假设有2个生物标志物 $M_1, M_2$ 和1个IV $Z$。已知 $E[\hat{\gamma}_1] = \alpha \lambda_1 + \delta_1$, $E[\hat{\gamma}_2] = \alpha \lambda_2 + \delta_2$, $E[\hat{\Gamma}] = \alpha \beta$。若 $\delta_1, \delta_2$ 均不为0且未知，请证明仅靠一阶矩无法识别 $\text{sign}(\beta)$；进一步，在 $E[\delta_1 \delta_2] = 0$ 假设下，写出利用二阶矩 $E[\hat{\gamma}_1 \hat{\gamma}_2]$ 识别 $\text{sign}(\beta)$ 的表达式。
